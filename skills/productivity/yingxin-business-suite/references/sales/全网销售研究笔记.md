## 一、八大销售心理学原理（财税版落地）

| 原理 | 落地话术 |
|------|---------|
| **锚定效应** | 先报标准价398元/月，再给创业扶持价198元/月 |
| **损失厌恶** | "因为账目不清被查，罚款几千到几万，我们一年才两千多" |
| **社会认同** | "上个月园区新注册20家，15家选了盈信" |
| **互惠原理** | "我帮您免费做一次税务健康体检，15分钟出报告" |
| **稀缺性** | "这个创业扶持价只针对本月新注册企业" |
| **承诺一致性** | "既然认可合规重要性，我们先签三个月体验？" |
| **喜好原理** | "我也是苏州本地人，服务过好几个园区的老板" |
| **权威效应** | "团队高级会计师+注册会计师+税务师三证齐全" |

## 二、B2B价格谈判八法

### 2.1 价值优先报价法
先讲价值再报价格。全套服务4800/年，今天给您3600/年——价值感拉满。

### 2.2 三明治谈判法
肯定客户(您很懂行) + 陈述价值(防护风险) + 守住底线(已是特别政策)

### 2.3 让步节奏控制法
- 第一次让步：5%（配合价值重申）
- 第二次让步：3%（配合服务缩减）
- 第三次让步：送增值服务（不降价，加感知价值）

### 2.4 拆解报价法
"一年3600，一天才10块，一杯奶茶钱换全年账务无忧"

### 2.5 竞品对比法
不贬低对手，突出差异化："他们只做基础记账，我们额外提供季度报告+风险预警"

### 2.6 假设成交法
"如果我们合作，您希望每月几号上门取票？报表通过邮箱还是微信？"

### 2.7 价格锚点法
三档报价：基础198 → 标准298(推荐) → 尊享498，让客户觉得"中间档最划算"

### 2.8 沉默成交法
报完价沉默，让客户先开口——先开口的人容易让步

## 三、砍价五种心理类型

| 类型 | 动机 | 应对 |
|------|------|------|
| 习惯性 | 不砍不舒服 | 微让利+送服务，给面子不给折扣 |
| 试探型 | 看看有没有空间 | 坚定报价+强调价值，不退让 |
| 预算不足 | 真没钱 | 提供分期/月付方案，不降价 |
| 比价型 | 拿低价来压 | 做差异化对比分析 |
| 价值怀疑 | 觉得不值 | 案例+数据+资质重塑价值 |

**万能应对流程**：认同 → 追问 → 归因 → 重塑 → 方案

## 四、砍价红线管理

- 宁可丢单不做亏本生意
- 低价客户后期服务成本更高
- "这个价格确实做不了，低于成本的服务品质我们也不敢保证"
- 有底气的态度反而增加客户信任

## 五、难缠客户8种类型

1. **犹豫不决** → 设截止时间+风险保障+算拖延成本
2. **质疑挑剔** → 提前备好资质+邀请实地考察+让老客户现身说法
3. **自以为是** → 先肯定+用专业问题引导+展示最新政策变化
4. **拖延型** → 制造紧迫感+列出不做的后果
5. **纯比价型** → 快速判断是否跟+做差异化服务对比
6. **抱怨不满型** → 先倾听不反驳+24小时内给方案
7. **决策链复杂** → 帮客户准备汇报材料+直接接触决策人
8. **朋友在做型** → 不否定朋友，强调专业分工+人情隔离

## 六、代账行业价格竞争八大策略

1. **不做价格战胜者，做价值战引领者**
2. **三级服务定价**：引流198 → 标准298(核心) → 尊享498(利润)
3. **本地化深耕**：苏州聚焦园区，上海聚焦郊区初创
4. **获客差异化**：工商代办合作+园区活动+财税社群
5. **转介绍体系**：送1个月服务费/老客户答谢会
6. **拒接垃圾客户**：极度低价才成交的、要求多付费低的
7. **会员制涨价**：先超额交付→再渐进提价→感恩价锁老客户
8. **可视化服务**：月报+季报+半年评估报告，让隐性服务变看得见

## 九、心态修炼

- 平均10次客户接触→2-3次谈判→1次成交
- 客户不是拒绝你，只是当下不合适
- 续费客户获客成本为0，转介绍还能新增
- 长期主义：第一年不赚钱，留存3-5年就是优质资产
