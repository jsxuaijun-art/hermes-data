# 股东不交社保的合法方案与谈单指引

> 来源：盈信财税 · 股东社保合规实务指南
> 适用场景：谈单中客户要求"不给股东交社保"、老客户咨询、续费时社保合规升级
> 核心前提：**股东干活不发工资不合逻辑，发了工资不缴社保不合法规。**

---

## 一、为什么股东社保是个普遍痛点

中小微企业老板最常见的几个认知误区：

| 误区 | 事实 | 风险 |
|------|------|------|
| "我是老板，不用给自己交社保" | 法律上股东≠免交社保 | 全员社保是法定要求 |
| "股东不领工资就不用交" | 没工资=没劳动关系，但若存在事实劳动关系仍需缴纳 | 稽查认定事实劳动关系→补缴+滞纳金 |
| "公司亏损就不用交" | 社保是强制缴纳，与公司盈亏无关 | 欠缴可被强制执行 |
| "我先不交，以后再说" | 系统自动比对个税申报与社保 | 比对不符直接推送补缴通知 |

> **谈单话术：** 「王总，现在税务和社保系统已经打通了——你发工资申报个税的人和交社保的人对不对得上，系统自动对比。与其等着被追缴，不如选个合规的方案。」

---

## 二、四大方案对比

### 方案一：劳务报酬（股东当顾问，不当员工）

**操作要点：**
- 股东不在公司担任总经理、财务负责人等高管职务（**不签劳动合同**）
- 通过股东会决议，明确股东以顾问/监事身份领取报酬
- 每月开票或申报个税（劳务报酬）
- 个税成本：800以内免，4000以内扣除800后×20%，4000以上扣除20%后按档计税

| 月金额 | 个税计算 | 实缴个税 | 社保成本 |
|--------|----------|----------|----------|
| 3,000 | (3,000-800)×20% | 440元 | 0元 |
| 5,000 | 5,000×(1-20%)×20% | 800元 | 0元 |
| 10,000 | 10,000×(1-20%)×20% | 1,600元 | 0元 |

**风险提示：**
- 劳务报酬比工资个税重（没有专项扣除、专项附加扣除）
- ⚠️ 如果股东是**法定代表人+日常在公司坐班**，税务局可能直接认定为事实劳动关系，要求补社保

### 方案二：分红代替工资（不交社保）

**操作方法：**
```
不发工资
  ↓
公司利润 → 企业所得税（25%或优惠税率）→ 税后利润 → 分红给股东
  ↓
个税20%，不交社保
```

**适用条件：**
- 公司有实际利润
- 股东不介意个税20%（比工资最高45%低，但工资有免征额和扣除项）

**⚠️ 问题：**
- 公司亏损或没有利润，此路不通
- 分红不能替代"工资成本"在企业所得税前扣除（分红是税后分配）
- 税负对比：综合税负约40%（企税25%+分红20%），远高于工资方案

### 方案三：最低基数缴社保（折中方案）⭐ 推荐

很多老板说"不愿意缴社保"，实际是不愿意按**实发工资全额**缴。谈到这一步时可以：

**协商路径：**
```
① 按当地最低工资签劳动合同
② 社保按最低基数缴（不是按实发工资）
③ 差额部分走报销/补贴/年终奖
```

**成本测算（苏州2025年参考）：**

| 项目 | 金额 |
|------|------|
| 最低工资标准 | ≈ 2,490元/月 |
| 社保最低缴费基数 | ≈ 4,494元/月 |
| 公司部分（约24.9%） | ≈ 1,119元/月 |
| 个人部分（约10.5%） | ≈ 472元/月 |
| 合计 | ≈ 1,591元/月/人 |

**关键话术：**
> 「王总，社保按最低基数走，三个人加起来一个月也就四五千块钱。比起被查到**补缴+滞纳金+罚款**，这点钱是买个'合规'。你说是吧？」

### 方案四：硬刚（不推荐）

有些客户油盐不进，这时候需要把账算给他看：

**被查到的后果（按3年追溯）：**

| 项目 | 金额 |
|------|------|
| 补社保公司部分（3人×36月） | ≈ 162,000元 |
| 滞纳金（日万分之五累计） | ≈ 40,000~60,000元 |
| 罚款（0.5~3倍） | ≈ 80,000~480,000元 |
| **合计** | **≈ 30万~70万** |

## 三、分场景推荐方案

| 客户类型 | 推荐方案 | 理由 |
|----------|----------|------|
| 初创公司，没钱 | 方案一（劳务报酬/董事费） | 最低成本，暂时合规 |
| 有利润，老板怕麻烦 | ③ 最**低基数社保** | 花小钱买平安 |
| 亏损或没收入 | ① 劳务报酬或③ 最低工资+社保 | 成本可控 |
| 老板就是不愿意 | ① 董事费/劳务报酬 | 扛住，不签劳动合同 |
| 正规发展要融资 | ③ 全额社保 | 尽调过不了关 |

## 四、谈单话术

### 初次提及社保话题

> 「老板，问你一个实在的问题——你公司现在股东们交社保吗？大多数公司都是老板说'不交'，会计就不管了。但金税四期上线后，**社保系统和税务系统已经打通了**，你的个税申报人数和社保缴纳人数不一样的，系统自动标红推送。」

### 老板坚持不交

> 「理解，我见的老板90%第一反应都是不想交。但我们可以不按工资全额交——**按最低基数，一个月一个人也就一千多**。比起被查到补三年的钱，这点钱是买个安心。而且这部分是公司成本，可以抵企业所得税。」

### 老板说"我会计说可以不用交"

> 「很多代账会计为了省事或者留住客户，会说'没事不用交'。但你要知道——**税务稽查抓的是企业法人，不是代账会计。** 罚单下来是写你名字，不是会计的名字。」

### 老板说"那给我方案，我看看"

> 「好，根据你的情况，我推荐走**最低基数方案**——你正常发工资，社保按最低基数交，不多花什么钱。如果是股东不参与经营，可以走**劳务报酬**，不签劳动合同，但需要走正规的报税通道。你看哪个方向你更倾向？」

## 五、实操要点

1. **劳动合同必须签**：即使按最低基数，也要有劳动合同，否则连最低基数都不用交——但也没有劳动关系，股东不能以员工身份领取工资
2. **个税必须申报**：不交社保不等于个税可以0申报，发工资就要报个税
3. **不要挂靠**：虚列员工名单、虚假挂靠缴纳社保属于骗保行为
4. **关注地方政策**：苏州目前社保基数逐年上调，幅度约5~8%/年，长期方案需动态调整
5. **城乡居民社保兜底**：股东如果不交职工社保，至少要有城乡居民社保（≈几百元/年），否则没有基本医疗保障
