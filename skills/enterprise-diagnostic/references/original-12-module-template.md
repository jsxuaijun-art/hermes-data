## 二、12模块输出框架

### 模块一：企业总体诊断

输出企业综合评分（100分制）：

| 评分项目 | 说明 |
|---------|------|
| 商业模式 | |
| 股权架构 | |
| 财税合规 | |
| 风险控制 | |
| 企业成长性 | |
| 融资能力 | |

然后给出总体评价。

### 模块二：行业深度分析

分析以下维度：

1. 行业现状
2. 行业未来3-5年发展趋势
3. 行业主要盈利模式
4. 行业内常见经营模式
5. 行业平均利润率
6. 行业内卷程度
7. 行业进入门槛
8. 行业未来机会
9. 行业未来风险

**要求：** 内容专业、真实、具体。避免空泛表达。

### 模块三：行业内幕与常见陷阱

分析以下维度：

1. 行业内常见违规行为
2. 行业内常见税务问题
3. 行业内常见法律风险
4. 老板最容易踩的坑
5. 税务机关重点关注事项
6. 真实经营中最容易出问题的环节

**要求：** 内容专业、真实、具体。避免空泛表达。

### 模块四：商业模式分析

输出以下内容：

- 当前盈利方式
- 利润来源
- 风险来源
- 可增加的利润来源
- 可提升的业务模式
- 高利润业务建议

### 模块五：股权架构分析

判断当前股权架构是否存在以下风险：

- 控制权风险
- 僵局风险
- 退出风险
- 股东纠纷风险
- 融资障碍风险

输出：

- 现状分析
- 风险分析
- 优化建议
- 推荐股权架构方案
- 推荐持股比例
- 是否建议持股平台
- 是否建议有限合伙企业
- 是否建议设立控股公司

### 模块六：税务风险诊断

检查以下风险：

- 增值税风险
- 企业所得税风险
- 个人所得税风险
- 社保风险
- 发票风险
- 资金流风险
- 私户收款风险
- 成本票不足风险
- 两套账风险
- 金税监管下重点风险

输出：

- 风险等级：★★★★★
- 风险原因：
- 优化建议：

### 模块七：企业架构与风险隔离设计

判断是否需要：

- 母公司 / 子公司 / 分公司 / 控股公司
- 风险隔离 / 资产隔离 / 业务隔离

输出：推荐架构方案。

### 模块八：融资与发展路径分析

分析企业现阶段适合的融资方式：

- 银行贷款
- 科创贷
- 知识产权贷款
- 股权融资
- 战略投资

并分析未来融资障碍。

### 模块九：政策红利分析

结合行业分析可申请的资质和补贴：

- 高新技术企业
- 专精特新
- 创新型中小企业
- 科技型中小企业
- 补贴 / 税收优惠 / 扶持政策

### 模块十：未来三年经营风险预测

预测未来三年最可能出现的风险：

- 股权风险
- 财税风险
- 法律风险
- 劳动用工风险
- 资金风险
- 市场风险

按风险等级分类：高 / 中 / 低

### 模块十一：老板最应该优先解决的3件事

按重要程度排序，每件事写明原因：

1. 第一优先事项
2. 第二优先事项
3. 第三优先事项

### 模块十二：顾问结论（麦肯锡/德勤风格）

输出以下内容：

- 当前企业最大的机会
- 当前企业最大的风险
- 当前企业最值得优化的地方
- 如果由专业顾问介入，预计可改善的方向

**要求：** 专业、具体、具有商业洞察力。不要空话套话。不要简单重复用户输入内容。尽可能结合行业特点进行深度分析。

### 模块十三（附加）：成交引导

1. 本次诊断发现的重点问题（列出3-5项）
2. 建议优先开展的专项诊断：
   - 股权架构专项诊断
   - 财税合规专项诊断
   - 企业顶层架构设计
   - 企业税务优化方案设计
   - 企业风险隔离设计
