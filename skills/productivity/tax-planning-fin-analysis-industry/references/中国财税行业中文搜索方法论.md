# 中国财税行业中文搜索方法论

## 核心发现：Bing 中文搜索的"代理"算法缺陷

| 搜索引擎 | 搜索 "代理记账机构等级认定" | 结论 |
|----------|---------------------------|------|
| **Bing** (cn.bing.com) | 返回 VPN/代理服务器/翻墙内容 | ❌ 无法使用 |
| **Sogou** (sogou.com) | 返回 CACFO、gov.cn、maigoo 等政府/协会内容 | ✅ 首选 |
| **360 搜索** | 返回部分可用结果 | 🟡 备选 |

**根因：** Bing 的算法在处理中文多义词"代理"时，优先匹配了"proxy"语义而非"agent"语义（"代理记账"=bookkeeping agency，而非 proxy）。这是 Bing 的算法级问题，不是环境/网络问题。

## 财税类中文搜索优先级推荐

```
搜狗(首选) → 360搜索(备选) → 百度(第三方备选) → Bing(不可用)
```

## 搜索策略（按场景）

### 场景A：找政策原文
```
搜狗搜索关键词 = 政策文号 OR "政策名称全称" site:chinatax.gov.cn
例："中总秘〔2017〕60号"
```

### 场景B：找行业数据/认证名单
```
搜狗搜索关键词 = "行业/认证名称" + "名单" + "年份"
例："代理记账机构等级认定" "2024" "名单"
```

### 场景C：找评分标准/细则
```
搜狗搜索关键词 = "行业" + "评分" + "标准"
备选：相关网站的子域名搜索，如 dljz.cacfo.com 下的PDF文件
```

### 场景D：交叉验证
```
同一关键词用不同搜索引擎搜，对比结果来源
发现不一致时，以政府官方网站(如 gov.cn/cacfo.com)为准
```

## 已验证搜索技巧

1. **中总协文件获取**：CACFO 网站 URL 有 /details/{id}.html 模式，可遍历相近ID找相关文件
2. **PDF 评分表**：评分细则通常以 PDF 形式挂在 dljz.cacfo.com 下，URL 模式 /d/file/content/{year}/{hex}.pdf
3. **搜狗摘要有用**：Sogou 搜索结果页摘要中含关键信息（如评分满分100分、90分+为5A级），可以不用点开原文
4. **Bing不可用时不要重试**：同一关键词换 URL 形式(https/http/www/cn)不会改变算法匹配，直接切 Sogou
