# 账号蒸馏：RIA-TV 五步萃取法

## 来源
博入蒸馏skill（2026.6.21 江姐桌面文档 `/mnt/d/360MoveData/Users/Admin/Desktop/博入蒸馏skill.docx`）

## 核心公式

```
博主蒸馏 = 人30% + AI 70%
```

| 角色 | 占比 | 职责 |
|:--|:--:|:--|
| **人工（导演）** | 30% | 选对博主 → 选方向 → 定判断 |
| **AI（剪辑师）** | 70% | 读取 → 归纳 → 写skill |

## 人工负责的 30%（三件事）

1. **选对博主** — 判断谁值得花时间研究
2. **选方向** — 要他的哪一块（人设/金句/结构/思想）
3. **定判断** — AI给的东西对不对

## AI 负责的 70%（三件事）

1. **读取** — 扒最精华的笔记/帖子（30-50条起步）
2. **归纳** — 归纳博主套路、思想、方法论
3. **写** — 输出可反复调用的 skill

---

## RIA-TV 五步萃取法

```
Read → Identify → Archive → Template → Verify
读出来 → 看得懂 → 装起来 → 过工遍 → 验一次
```

### 第一步：读取（Read）

| 字段 | 必要性 |
|:--|:--|
| 标题 | 必须有 |
| 正文 | 必须有 |
| 点赞数 | 区分爆款 vs 水文 |
| 收藏数 | 判断内容深度价值 |
| 评论数 | 识别读者真正共鸣的点 |

- 最少 **30条** 才能分析出稳定模式
- 一般抓 **50条** 起步
- 来源：抖音/视频号/小红书/公众号等

### 第二步：识别（Identify）—— 最关键的萃取步骤

三层模式，**必须自上而下，不能跳、不能倒着来**：

```
认知层 → 策略层 → 内容层
（相信什么）（怎么打）（具体写什么）
```

| 层级 | 定义 | 产出 | 示例（小微学财经） |
|:--|:--|:--:|:--|
| **认知层** | 他相信什么 | 3-5条核心信念 | "财经不是分析是转化" |
| **策略层** | 具体怎么打 | 2-3条运营准则 | 宏观叙事→个人影响 |
| **内容层** | 具体写什么 | 3-5条标题公式 | "特朗普×房贷和存款" |

**验证方法**：认知→策略→内容是否逻辑一致。如果认知说A但内容写B，说明没拆对。

### 第三步：归档（Archive）

把三层内容装进一份结构化档案：

```
## 身份描述
一句话说清楚他是谁

## 核心信念（认知层）
1. ...
2. ...
3. ...

## 运营准则（策略层）
1. ...
2. ...

## 标题公式（内容层）
1. ...
2. ...
3. ...
```

### 第四步：模板化（Template）—— 最重要的一步

**提炼翻译原则**：把他的话翻译成通用话。

目的：AI下次用这个skill写任何话题时，都知道套什么框架。这叫**建模**。

到达这一步，你已不在复制他的句子，而是在用他的**思考结构**。

### 第五步：验证（Verify）—— 最后一道闸门

**必做，不能跳过。**

做法：
1. 拿他没写过的话题
2. AI按skill生成一次
3. 像不像？
   - **像** → skill成了，上线
   - **不像** → 扔掉重新来

跳过这步的后果：下次用他来写新话题，照样翻车，你根本不知道错在哪一层。

## 最终产出

| 交付物 | 给谁看 | 用途 |
|:--|:--|:--|
| **三层顺序报告** | 你自己 | 为什么会爆、他的套路，一次性看明白 |
| **可触发 skill** | AI 用 | 下次直接调用，不喂任何东西 |

> 报告只是副产品，skill 才是真正的资产。

## 两条心法

1. **你手上有哪件事值得用这种方式去对待？**
   - 这件事你每周都在做吗？
   - 每次都是从零开始吗？
   - 两个都√ → 应该造资产，应该捏skill

2. **不用学技术，先拿一张纸**
   - 把你这周重复做了3次以上的事情写下来
   - 写完之后你可能非常害怕——里面藏着你那些还没造出来的资产

## 实际执行中的陷阱（2026.6.21 实战记录）

### 抖音反爬限制（自动获取 data 不可行）

| 尝试方式 | 结果 |
|:--|:--:|
| 浏览器直接访问主页 | 视频列表区域显示「服务异常」 |
| 抖音搜索页面 | 触发 CAPTCHA 验证码 |
| API 端点（aweme/v1/） | 返回 blocked |
| 百度/Bing 搜索 | 触发 CAPTCHA/Cloudflare |
| 第三方 SDK 库 | 同样被反爬拦截 |

**结论**：在当前环境（WSL + 无登录cookie + 无住宅代理）下，抖音**不可自动化采集数据**。

### 可用的替代方案

| 方案 | 操作方式 | 成本 |
|:--|:--|:--:|
| **人工喂数据** ★推荐 | 江姐在抖音App按「最多点赞」排序，截图/复制30条爆款发我 | 免费，15分钟 |
| 手机端抓包 | 安装Stream App，拦截抖音真实API请求，导出json | 需配合操作 |
| 已登录Chrome cookie | 从江姐电脑Chrome导出cookie注入WSL浏览器 | 需配合操作 |
| 第三方数据平台 | 蝉妈妈/飞瓜/新抖 等付费平台查看公开数据 | 付费 |
