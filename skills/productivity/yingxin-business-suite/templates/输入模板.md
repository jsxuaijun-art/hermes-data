# 税务筹划 + 财务分析 + 行业调研 — 输入模板

当你需要向我咨询时，可以按以下格式填写输入信息：

## 输入模板

```
## 🔍 业务背景
（描述你的业务场景、公司情况、遇到的问题）

## 🎯 需求类型
- [ ] 税务筹划
- [ ] 财务分析（触发词：上传财务报表 / 财务报表）
- [ ] 行业调研
- [ ] 综合（多个职能配合）

## 📊 数据提供
（见下方"财务数据格式指引"）

## ❓ 具体问题
（列出你最想解决的问题点）
```

## 财务数据格式指引

当触发财务分析模式时，建议按以下字段提供数据（至少提供**利润表**部分即可做初步分析）：

### 利润表（最少也得有这个）
| 字段 | 单位 | 说明 |
|------|------|------|
| 营业收入 | 万元 | 当期总收入 |
| 营业成本 | 万元 | 直接成本 |
| 销售费用 | 万元 | 销售相关的费用 |
| 管理费用 | 万元 | 管理相关的费用 |
| 财务费用 | 万元 | 利息、手续费等 |
| 净利润 | 万元 | 税后利润 |

### 资产负债表
| 字段 | 单位 | 说明 |
|------|------|------|
| 货币资金 | 万元 | 现金+银行存款 |
| 应收账款 | 万元 | 客户欠款 |
| 存货 | 万元 | 库存商品/在制品 |
| 固定资产 | 万元 | 设备、房产等 |
| 短期借款 | 万元 | 1年内到期的借款 |
| 应付账款 | 万元 | 欠供应商的款 |
| 长期借款 | 万元 | 超过1年的借款 |
| 所有者权益 | 万元 | 资产-负债后的净资产 |

### 现金流量表（有的话更好）
| 字段 | 单位 |
|------|------|
| 经营活动现金流净额 | 万元 |
| 投资活动现金流净额 | 万元 |
| 筹资活动现金流净额 | 万元 |

## 数据提供方式

| 方式 | 说明 | 适用场景 |
|------|------|----------|
| **贴文字** | 把数据直接打字发过来 | 手边有数据，不想开文件 |
| **文件路径** | 告诉我 Excel/Word 的桌面路径 | 文件在电脑上，懒得打字 |
| **口头描述** | 说几个大概数字 | 记不清具体，但想先判断一下 |

## 触发说明

| 你说 | 我会 |
|------|------|
| "上传财务报表" 或 "财务报表" | 先激活财务分析模式，引导你提供数据 → 搜索指标标准值 → 分析 |
| 其他任何问题 | 默认调用税务筹划/行业调研/综合判断 |
