# 财税服务价格商谈谈判技能包

## 核心理念

> **服务费 ≈ 买断一定的工作时间。**
>
> 你支付的价格，对应的是会计/团队在这个月里为你花的时间。
> 上帝对每个人唯一公平的就是24小时——用来工作的时间就那么多。
> **做了这个，就不能做那个。**

这不仅是"怼低价同行"的杀手锏，更是贯穿整个价格谈判的逻辑锚点：

```
┌─────────────────────────────────────────────────┐
│  服务费 = 买断工作时间                          │
│         ↓                                        │
│  时间总量固定（每个人每天就那么多工作时间）       │
│         ↓                                        │
│  时间花在开票上 → 做账时间就会少                   │
│  时间花在专业上 → 做账质量才能高                   │
│         ↓                                        │
│  低价 = 花时间做事务性工作（开票/跑腿）            │
│  中高价 = 花时间做专业工作（做账/分析/风控）       │
│         ↓                                        │
│  先比专业资质 → 再比价格 → 才是科学理智的比价      │
└─────────────────────────────────────────────────┘
```

## 适用场景

| 场景 | 触发条件 | 谈判目标 |
|------|---------|---------|
| 初次报价 | 客户说"多少钱一年" | 建立价值认知，避免直接比价 |
| 嫌贵/比价 | 客户说"XX家便宜一半" | 用"时间买断逻辑"化解价格异议 |
| 要求免费服务 | 客户说"能不能免费代开发票" | 明确定价边界，拒绝不合理要求 |
| 续费谈判 | 客户说"明年能不能便宜" | 把"续费"重新定义为"继续买安心" |
| 同行挖墙脚 | 客户说"XX家给我更低价" | 用专业资质对比锁定价值 |

## 全套话术

### ① 怼低价同行杀手锏（核心话术）

客户说"XX家公司比你便宜，还免费开发票"时：

> 王总，想一个问题：您的服务费其实是买断了我们一定的时间，您支付的这个价格就对应着我的工作时间。
>
> 肯定会有同行说：'我们是跟别人不同，帮你免费开票的'——但是您要知道：**开发票这件事情是没有技术性的，只是耗时间而已，而且是做在明面上的，让你有感觉的，让大家开心的事情。** 但是他把分配给你的时间全部用在开发票上面了，那您说他分配给您用来认真做账的时间会有多少？
>
> **做账这件事情才是真正体现专业性的，而且还是看不见的无形服务，做多做少花多少时间根本不知道。** 现在税务大数据监管这么严格，风险也就来了。
>
> 上帝对每一个人唯一公平的就是给了24小时，每个人用来工作的时间就那么多，大家都是靠出卖时间来赚钱的。这个承诺免费开票的同行，他的时间也是这样的——**做了这个就不能做那个。**
>
> **先比资质、专业再比价格是高认知的做法：** 同样的价格您要比质量得到性价比。在相同的专业资质情况下，才去对比价格，这才是科学理智的、认知高的做法。

### ② 先比专业再比价格（比价反制）

客户拿低价同行来比：

> 同样的价格您要比质量。在相同的专业资质情况下，您才去对比价格，这才是科学理智的。
>
> 如果您只看价格高低，我还是建议您多比较比较。**先比专业，再比价格。** 我们的高级会计师资质摆在这——您先看看同行有没有？再看我们公司是TSC五级涉税专业服务机构（438.11分，全省前列）。既没有高级会计师也没有TSC五级？那就不用比价格了。

### ③ 价值量化话术（把"贵"变成"省回来的钱"）

客户说"一年X万太贵"：

> 我给您算一笔账：一个月服务费不到X千块，合每天不到X块钱。
>
> 这个价格买的是什么？不是一个会计在做账，是一个高级会计师团队在帮您看住风险。**只要我们从40多个风险点里帮您拦下一个——不管是稽查罚款、还是政策没用到多缴的税——省回来的钱就已经超过一年的服务费了。**
>
> 您不是花钱，您是花小钱买一份保障。**一年的服务费，换一个稽查无忧的放心。**

### ④ 拒绝免费代开发票（定价边界）

客户要求免费代开发票：

> 我们不开发票的。原因简单——
>
> 一个人的时间是有限的，有同行提供免费代开发票，那是人家的服务模式。我把时间花在开发票上，做账的时间就会少。
>
> 我们的精力放在把账做清楚、申报做仔细上。发票这块，您方如有需要，建议另外找能代开的同行或者自己去税务局申请代开。

### ⑤ 续费价格异议（年费续约）

客户续费时说"能不能便宜"：

> 王总，说实话一年X元的服务费，在财税行业是良心价了。
>
> 还是那个逻辑：**服务费买的是我们的工作时间。** 这一年我们花了多少时间在您的账上——12个月记账零差错、X次风险预警、X次政策提醒、问题响应平均X小时内——这些时间加起来乘以我们的专业成本，远远超过您付的服务费。
>
> 如果您觉得服务有哪里不到位，您直说，我改进。如果是纯粹想便宜点——我再多说一句：生意越不好的时候，账越不能乱，越经不起罚款。要不这样，我给您争取一个老客户早鸟价，X月X号前签的话优惠X元，算我们的心意。

## 谈判五步流程

### 第1步：听清异议

客户说"贵/便宜/再看看"，不要急着反驳。先确认对方说的是：
- **价格敏感型**：真觉得贵，预算不够
- **比价型**：拿同行报价压价
- **价值不认型**：没看懂你的服务值什么

### 第2步：认同+转化

```
"我理解你的感受——"
"不过我想让你看一个逻辑："
```

### 第3步：抛出"时间买断逻辑"

用核心理念话术，把谈判从"比价格"拉到"比时间分配、比专业价值"。

### 第4步：用专业资质收尾

高级会计师 + TSC五级 + 438.11分 → 用硬资质锁定"不是所有同行都一样的专业"。

### 第5步：给台阶下

```
"建议您先比专业，再比价格。等您比完了，随时找我。"
```
不催单、不逼单、给对方体面的决策空间。
