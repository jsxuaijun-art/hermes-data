# 百度百科词条编辑指南（GEO 实战）

> 2026.6.22 基于「苏州盈信企业管理有限公司」百度百科词条编辑实战整理

## 前置检查

### 1. 确认公司词条是否存在

用浏览器访问 `https://baike.baidu.com/item/公司全称`
- 200 → 词条存在，进入「优化已有词条」流程
- 404 → 词条不存在，进入「创建新词条」流程

### 2. 企业词条 vs 个人词条

| 类型 | 创建难度 | GEO 价值 |
|:-----|:---------|:---------|
| **企业词条** | 相对容易（有企业通道） | 中（权威背书） |
| **个人词条** | 难（需知名人士 / 有百度收录的媒体报道） | 低（很难通过） |

**GEO 策略：** 优先优化企业词条。个人词条若不存在且无百度认可的媒体收录，不创建。

## 词条现状评估

获取词条后，按以下维度评估：

| 维度 | 检查项 | GEO 优先级 |
|:-----|:-------|:-----------|
| **基本信息** | 公司简介开头100字是否含核心资质 | 🔴 P0 |
| **资质背书** | 是否有高级会计师 / TSC 评级 / 荣誉信息 | 🔴 P0 |
| **创始人** | 是否有创始人介绍 | 🟡 P1 |
| **服务数据** | 客户数量 / 转介绍率 | 🟡 P1 |
| **更新频率** | 最近编辑时间（越新越好编辑） | - |

## 编辑策略

### 分批次编辑（提高审核通过率）

一次提交过多修改 → 审核严格对照 → 易被驳回。

**最佳策略：每次集中补1-2个章节，分2-3次提交。**

| 批次 | 目标章节 | 修改内容 | 间隔 |
|:-----|:---------|:---------|:-----|
| 第一批 | 公司简介 | 追加核心资质（高会/TSC/客户数据） | 立即 |
| 第二批 | 所获荣誉 | 重写荣誉章节（高会评审/TSC评级） | 批一通过后 |
| 第三批 | 公司管理/创始 | 追加创始人介绍 | 批二通过后 |

## 操作步骤

### 编辑入口

1. 登录百度账号（个人百度账号即可编辑已通过企业认证的现有词条，无需企业账号）
2. 访问目标词条：`https://baike.baidu.com/item/公司全称`
3. 点击词条标题右侧的「编辑」按钮进入富文本编辑器
4. 如企业词条不存在，走企业创建通道：`https://baike.baidu.com/enterprise/create`（需企业认证）

### 添加参考资料（引用标注）

百度百科要求每一句新增事实性陈述都必须有引用来源（上标数字标注）。操作步骤：

**① 光标定位** — 在需要加引用的句子末尾点击鼠标光标

**② 找「引用」按钮** — 编辑器工具栏上的引用图标（不同版本略有差异，通常在工具栏右半部分）：
- 一本打开的书 📖 图标
- 或 `[1]` 上标图标
- 或链环 🔗 图标
- 悬停提示文字一般为「引用」或「参考资料」

**③ 贴链接** — 点击后弹出输入框，粘贴参考来源的完整 URL：
- 权威来源优先：政府官网（.gov.cn）、新闻门户（新浪/网易/搜狐/新华网）、行业协会官网
- 支持普通网页URL，系统会自动抓取标题
- 网址填写好后，系统自动填充标题 → 点击「确定/保存」

**④ 引用标注自动生成** — 句子末尾生成 `[上标数字]`，自动编号与已有引用列表衔接

**⑤ 编辑完成后** — 填写编辑理由（简洁说明本次修改），点击「提交」等待审核

### 参考来源类型要求

| 来源类型 | 示例 | AI（文心一言）引用权重 |
|:---------|:-----|:--------------------|
| 政府/事业单位官网 | 省财政厅公示、税务局官网 | 最高（T1） |
| 正规新闻门户 | 新华网、新浪、网易、搜狐 | 高（T1） |
| 专业机构官网 | 行业协会、征信机构 | 中（T1-T2） |
| 企业自有官网 | yingxinkuaiji.com | 低（需要其他来源交叉验证） |

**注意：** 同一信息最好在多个独立信源出现（交叉验证），AI会跨源核实，单源陈述置信度较低。

### 提交流程

1. 单次编辑完成一章节内容后，点击编辑器底部「提交」
2. 填写编辑理由（示例：`补充公司资质和团队介绍` `更新所获荣誉章节`）
3. 约1-3个工作日审核通过
4. 通过后再提交下一批次

### 首次编辑 vs 后续编辑

- **首次编辑现有词条**：建议先修改基础信息章节（公司简介追加），提交后等审核结果
- **第二次编辑**：首轮通过后，追加下一章节（所获荣誉等）
- **个人账号编辑已有企业词条**：不需要企业认证，个人账号直接编辑即可——该词条之前已通过百度企业认证，个人账号具有编辑权限

## 审核雷区

| 雷区 | 避雷方法 |
|:-----|:---------|
| 广告语气（"最好的"、"领先的"） | 用中性描述 |
| 无第三方佐证的数据 | 标注来源（省财政厅/税务局可查） |
| 单次编辑过多 | 分2-3次提交，每次1-2节 |
| 修改企业基本信息（法人/注册资本等） | 仅修改描述性章节，不改工商登记信息 |

## 与 GEO 冷启动的衔接

百度百科词条属于文心一言（百度生态 AI）的 **T1 信源**。词条升级是 GEO P1 阶段的关键任务。

**冷启动阶段：**
- P0（基础设施）→ ❌ 不含
- P1（关键平台注册）→ ✅ **百度百科词条升级**
- P2（多平台分发）→ ❌ 不含

**文心一言 GEO 增益：**
百度百科企业词条 → 文心一言直接引用 → 用户搜索财税服务时品牌信息被采纳。
