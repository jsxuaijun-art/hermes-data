# 苏州盈信·初次洽谈全流程 SOP 与话术

> **适用场景：** 新客户从电话接触到签约成交的全流程
> **版本：** v1.0 | 2026.6.6
> **融合素材：** 李薇薇8步法 + S1-S5体系 + 话术精炼库 + 高级会计师锚点

---

## 一、核心武器：高级会计师锚点

**全流程中必须出现的 3 个锚点：**
1. 介绍公司时 → 「我们老板 2018 年就评上高级会计师了，全江苏八九百人里唯一一家代账公司通过的」
2. 聊专业时 → 「不是挂靠大公司的，是以我们小代账公司的名义，自己凭本事评的，含金量比挂靠高多了」
3. 逼单时 → 「您让同行去找一个高会带队的代账公司比比看」

## 二、第1阶段：电话获客（3-5分钟）

### 2.1 目标
- ✅ 客户接了电话没挂
- ✅ 拿到一个约见面/视频的理由
- ✅ 建立「这人有点东西」的第一印象

### 2.2 话术模板

**模板A：打新公司（刚注册/刚拿执照）**

> 「喂，（客户称呼）对吧？我是苏州盈信的（你的名字）。您公司刚下来，通知您一声——执照下来30天内要做税务登记，超期有罚款。您现在税务这块有人弄了没有？」

**关键：** 用政策信息开场，不是推销。客户回答后 ↓

> 「那我跟您说一下流程：税务登记→票种核定→三方协议→每月申报。如果没人弄，这些全部要您自己跑。您现在全职还是兼职在做？」

**模板B：打老公司（挖同行墙角）**

> 「（客户称呼），我是苏州盈信的（你的名字）。今天打电话没别的事，就想问一句——您现在的会计，是什么样的资质？助理会计师还是中级？」

停顿等回答 ↓

> 「您知道吗，现在代理记账行业有一个很稀缺的资质——高级会计师。全江苏省八九百人里，唯一以代账公司名义评上高会的，就在我们公司。做账报税这种事情，初级200一个月也能做，高会带队做也是这个价——一样的钱为啥不找专业对口的？」

**模板C：转介绍/被动咨询**

> 「（客户称呼），感谢您给我们这个机会。我先花30秒介绍一下我们是谁——苏州盈信，做了16年财税，老板是高级会计师，2018年全省唯一以代账公司评上的高会。您公司什么情况，我需要了解几个问题，帮您判断匹配不匹配。」

### 2.3 电话必问6题（需求挖掘）

1. 「您公司什么行业？开了多久了？」
2. 「现在账谁在做？（自己/兼职/代账/全职会计）」
3. 「每个月开票量大吗？一般纳税人还是小规模？」
4. 「最近有没有遇到过税务问题？稽查、预警、补税之类的？」
5. 「您现在每个月付多少服务费？」
6. 「您公司对财税最看重什么？省心？合规？省钱？还是能帮您筹划？」

### 2.4 电话目标转化

| 情况 | 目标动作 |
|------|---------|
| 客户有兴趣 | → 「我整理一份您行业的对比方案，明天上午还是下午方便，我到您公司坐15分钟？」 |
| 客户犹豫 | → 「那我先把我们公司的资料发您微信，您看看。不管做不做，有问题随时问我。」 |
| 客户直接拒绝 | → 「没关系。那我加您个微信，以后税务方面的问题您随时问，免费解答。」 |

## 三、第2阶段：面谈/视频诊断（30-45分钟）

### 3.1 开场（3分钟）· 破冰+亮身份

> 「（称呼），感谢您抽时间。我先简单介绍一下我自己——盈信财税，（你的名字）。我们公司做了16年，我们老板37岁就评上了高级会计师——不是挂靠大公司的，是以代账公司的名义自己评的。2018年全江苏八九百人，就她一个代账公司出来的高会。」

**停顿，等对方反应——** 一般会追问「那挺厉害的，高会难考吗？」

> 「难。会计从初级到高会，正常十几年。而且高会不是考过就行的，要评审——把你这十几年的业绩、案例、资质全部摆给专家团打分。大部分通过的都是国企财务总监、大学教授、上市公司CFO。代理记账公司能过的，全国都没几个。」

### 3.2 企业诊断（15分钟）· SPIN提问法

**S — 背景问题**
- 「公司目前主要做什么业务？」
- 「团队多少人？有专职财务吗？」
- 「现在用什么财务软件？还是手工做账？」

**P — 难点问题**
- 「现在的账务有没有让你操心的点——比如报税逾期、发票出问题、有些数据对不上？」
- 「有没有收到过税务预警？或者被专管员找过？」
- 「每个月报表出来你看得懂吗？有没有分析过成本、利润、税负率？」

**I — 暗示问题（放大后果🔥）**
- 「如果漏报一个税种，金税系统自动预警，轻则补税罚款，重则进异常名录，经营受影响——这一点您了解吗？」
- 「报表看不懂，等于您做了半年一年的生意，到底赚没赚钱、哪儿赚钱、哪儿漏钱——完全不清楚。」
- 「现在金税四期上线以后，数据都是自动比对的。发票、社保、银行流水都在一张网上，以前能混过去的现在混不过去了。」

**N — 需求-效益问题（让客户自己说价值）**
- 「如果有一套专业团队帮您盯紧税务、出月报分析、提前预警风险——每个月多花两三百块钱，值得吗？」
- 「您最希望解决哪一块？」

### 3.3 方案呈现（10分钟）

**FABG法则：**

| 层级 | 内容 | 话术 |
|------|------|------|
| F（Feature） | 我们做账+报税+合规+预警+分析 | — |
| A（Advantage） | 双会计交叉审核，高会带队把关 | 「别人家一个会计管70家，我们一个会计只管20家，每份账还有高会二审」 |
| B（Benefit） | 你不操心、不被罚、看得懂 | 「合作之后您就一件事——专心搞业务。税务那边我们全包，您不用想」 |
| G（Grabber） | 高级会计师全流程把关 | 「全省代账公司里，有高会带队把关的，你数不出5家」 |

### 3.4 报价（5分钟）

**三档报价法（具体数字根据实际定价调）：**

> 「我们按照您的需求，有三种方案：
> 
> **基础方案**——每月××元。标准代账+报税，适合业务简单的客户。
> 
> **标准方案**——每月××元。基础服务+合规审核+季度报表分析+风险预警。大多数客户选这个。
> 
> **高端方案**——每月××元。标准全包+高级会计师一对一托管+税务筹划建议+不定期上门。适合业务复杂、对财税重视的客户。
> 
> 您先看看哪个方案听着比较顺，咱们细聊。」

### 3.5 常见异议现场处理

| 异议 | 话术 |
|------|------|
| 「太贵了」 | 「您说得对，这不是一笔小钱。但我换个角度帮您算——一年×××元，平均一天一杯奶茶钱。请一个全职会计至少五六万一年，还不管合规。万一出一次风险，补税罚款都超过这个数了。」 |
| 「同行比你们便宜」 | 「我一个同行朋友，他那儿只要×××。但我有两个问题想问您——第一，他是不是高会带队？第二，出问题他赔不赔？我们做错账，全额赔付。一分钱一分货。」 |
| 「我先签一个月试试」 | 「我们所有合作都是一年起签。不是因为我们死板——而是财务服务是按年度做规划的。一个月刚把您的账理顺，刚进入状态，时间就到了。您先签一年，有任何不满意——全额退款。」 |
| 「我和股东商量一下」 | 「应该的。要不我今天先把方案整理一下发给您，您转给股东看看。或者您安排个时间，我当面给股东也讲一遍，有什么问题现场解决。」 |
| 「现在零申报，用不着」 | 「零申报虽然简单，但每个季度还是要报。年底还有汇算清缴。如果哪次忘了——罚款500起步。而且金税系统不认『没业务』，它只认你有没有按时申报。」 |

## 四、第3阶段：跟进与成交

### 4.1 跟进节奏表

| 时间节点 | 动作 | 话术/内容 |
|---------|------|----------|
| 面谈后2小时 | 微信发送对方名片+整理后的需求确认 | 「xx总，今天聊得很愉快。我整理了一下您的需求：①行业×× ②规模×× ③重点关注×× ④推荐方案××。您看看对不对？」 |
| 次日（第1天） | 电话回访+发补充资料 | 「xx总，昨天聊完之后我又查了一下您行业的最新政策，发现××。我发您看看。您跟股东聊得怎么样了？」 |
| 第3天 | 政策节点提醒 | 「xx总，提醒您一下——本月的申报截止日期是××号，记得提醒您现在的会计。另外我这边最近出了个新优惠……」 |
| 第7天 | 案例驱动跟进 | 「xx总，最近签了一个和您同行业的客户。给您看看他之前遇到的问题和我们的解决方案——跟您情况很像。」 |
| 第14天 | 最后真诚跟进 | 「xx总，我不打扰您了。不管您选不选我们，之前发您的政策信息您留着看就行。以后有任何财税问题，随时问我，免费解答。」 |

### 4.2 逼单7法（按场景选用）

| 方法 | 适用场景 | 话术 |
|------|---------|------|
| 二选一法 | 客户犹豫选哪个方案 | 「您觉得是基础方案够用，还是标准方案更放心？」 |
| 稀缺性法 | 客户一直拖着 | 「我们每个会计服务名额有限，目前您能对上的会计手上只剩最后一个名额了。」 |
| 从众法 | 客户不相信小代账公司 | 「我们合作了上千家客户，光您这个行业就一百多家。」 |
| 案例法 | 客户担心风险/质量 | 「上个月有个做贸易的客户，之前那家代账给他少报了十几万收入，金税预警了，他转过来我们帮他补申报、写说明，一周解决。」 |
| 损失厌恶法 | 客户觉得不急 | 「这个价格是我们季度优惠的价格，下周一就要调回原价了。」 |
| 高会锚定法 | 客户还在比价 | 「您可以去问任何一家同行——问他有没有高会带队，问他的会计是什么资质。一比就知道了。」 |
| 假定成交法 | 客户已经聊得差不多了 | 「那我先帮您把资料建好，明天安排会计跟您对接。」 |

### 4.3 成交后的动作

> 「xx总，合作愉快。明天我会安排您的专属会计加您微信，先做一次全面的账务体检。之后每个月的申报情况、每季度的报表分析，我都会同步发给您。有任何问题，随时找我和会计。」

**成交后24小时内必做：**
1. 拉专属服务群（客户+你+会计+主管）
2. 发《服务流程说明》
3. 做首次账务交接/资料收集清单
4. 发朋友圈（脱敏版）：「欢迎xx总加入盈信大家庭」

## 六、避坑指南

| 常见错误 | 正确做法 |
|---------|---------|
| 电话里报全价 | 电话只透露大致价格范围，详细报价见面/视频聊 |
| 过早让步降价 | 先价值重塑再谈价格，不要一上来就送 |
| 客户说贵就解释为什么贵 | 不要解释——反问「跟什么比觉得贵？」 |
| 比价格的时候只说价格 | 对比专业、资质、服务配置 |
| 客户说再考虑就真等 | 定好下次联系时间，不给客户晾着的机会 |
| 面谈只聊产品不聊痛点 | 80%时间挖痛点，20%时间给方案 |
| 忘了亮高会身份 | 开场必须亮，中间必须回扣，逼单必须再用 |
