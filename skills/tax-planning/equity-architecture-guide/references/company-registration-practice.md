# 公司注册实务模块

> 来源：《一本书讲透股权》第六章~第七章精华
> 用于 equity-architecture-guide 的"地基层"——任何股权架构方案都必须先想清楚的注册细节

---

## 一、注册什么类型的企业

### 常见主体类型对比

| 类型 | 责任形式 | 纳税方式 | 适用场景 |
|------|---------|---------|---------|
| 有限责任公司 | 注册资本内有限责任 | 企业所得税+分红个税 | 最普遍，防火墙基础 |
| 股份有限公司 | 注册资本内有限责任 | 同上 | 有上市规划时 |
| 有限合伙企业 | GP无限+LP有限 | 穿透纳税（先分后税） | 员工持股平台、基金 |
| 个人独资企业 | 无限责任 | 经营所得个税（5-35%） | 小规模个体经营 |
| 个体工商户 | 无限责任 | 同上 | 小微经营 |

**防火墙架构推荐：** 控股公司 + 运营子公司 → 全部选**有限责任公司**

### 关于一人有限公司

| 事项 | 说明 |
|------|------|
| 定义 | 一个自然人股东/一个法人股东 |
| 风险 ⚠️ | 举证倒置——股东需自证财产独立（公司法第23条第3款） |
| 夫妻公司 ⚠️ | 最高院(2019)民终1364号 → 实质一人公司，举证责任同 |
| 对策 | 引入第三方持股≥12%，或改为至少2名自然人股东 |

## 二、起什么样的企业名称

### 行政区划规则

```
行政区划 + 字号 + 行业 + 组织形式
  ↑         ↑      ↑      ↑
 苏州      盈信   财税   有限公司
```

### 字号选择要点

| 要点 | 说明 |
|------|------|
| 避免近似 | 行业内已有知名品牌，可能被诉侵权 |
| 避免负面谐音 | 苏州话/普通话双重检查 |
| 名称预核准 | 当地市场监管局网站/窗口查询 |
| 字号保护 | 控股公司和子公司字号统一，增强品牌一致性 |

## 三、注册资本背后的讲究

### 注册资本的核心逻辑

```text
注册资本 ≠ 公司价值
注册资本 = 股东承诺承担的最⾼责任上限
```

### 认缴制 vs 实缴制

| 对比维度 | 认缴制（主流） | 实缴制 |
|---------|-------------|-------|
| 适用范围 | 绝大多数公司 | 特殊行业（银行、保险、劳务派遣等27类） |
| 出资时间 | 章程约定（建议≤5年） | 设立时实缴 |
| 未实缴风险 | 破产时未缴部分需补足 | 无 |
| 影响 | 信用指标之一，非核心 | 实缴可增强信任 |

### 注册资本写多少合适？

| 场景 | 建议 | 原因 |
|------|------|------|
| 普通有限公司 | 10万~100万 | 够用即可，写高不等于实力强 |
| 控股公司 | 100万~500万 | 作为控股主体，写太低没有控股感 |
| 运营公司 | 按实际需要 | 客户/供应商/银行有最低门槛时相应调整 |
| 有招投标需求 | 按招标门槛 | 很多招标要求注册资本≥X万 |
| 有行业资质要求 | 按资质要求 | 如建筑类资质有注册资本门槛 |

### 实缴注意事项

| 事项 | 说明 |
|------|------|
| 出资方式 | 货币、实物、知识产权、土地使用权、股权、债权等 |
| 非货币出资 | 必须评估作价+财产权转移 |
| 抽逃出资 | 刑法第159条——可构成抽逃出资罪 |
| 认缴≠不缴 | 公司破产/清算时，认缴部分加速到期，需补足 |
| 印花税 | 实缴后按万分之二点五缴纳资金账簿印花税 |

## 四、法定代表人的选择

### 法定代表人法律地位

| 事项 | 说明 |
|------|------|
| 定义 | 依法代表公司行使职权的负责人（公司法第10条） |
| 人选范围 | 执行事务的董事或经理 |
| 责任 | 代表公司签合同、诉讼、对外活动 |
| 风险 ⚠️ | 公司违法时法定代表人可被列入失信名单、限制高消费 |

### 选谁当？

| 人选 | 推荐度 | 说明 |
|------|-------|------|
| 实际控制人自己 | ⭐⭐⭐ | 最安全，责权对等 |
| 配偶 | ⭐⭐ | 前提：控制权清晰 |
| 核心高管 | ⭐ | 有"挂名"风险，务必签免责协议+购买董责险 |
| 挂名（员工/亲属） | ❌ | 高风险！公司出事法定代表人首当其冲 |

> **防火墙场景建议：** 控股公司的法定代表人由实际控制人担任，运营子公司的法定代表人可由核心高管担任（前提：签署免责协议+建立制度隔离）

## 五、印章的风险与管控

### 公司印章有哪些

```
公章（公司行政公章）——效力最高
  ├─ 财务专用章——银行/税务
  ├─ 发票专用章——开票使用
  ├─ 合同专用章——合同签约
  └─ 法定代表人名章——银行/特定文件
```

### 印章效力层级

| 印章类型 | 对外效力 | 对应场景 |
|---------|---------|---------|
| 公章 | 最高——代表公司意志 | 合同/决议/申请 |
| 合同专用章 | 等于公章 | 合同专用 |
| 财务专用章 | 财务专用 | 银行/税务 |
| 法定代表人签名 | 等于公章（法律上） | 合同/文件 |

### 印章管理核心制度

| 管控环节 | 措施 |
|---------|------|
| 保管 | 专人保管+保险柜存放+用印登记簿 |
| 使用 | 用印审批单（用途/份数/经办人/批准人） |
| 外带 | 双人同行+限时归还 |
| 电子印章 | 使用合规电子印章平台，保留使用日志 |
| 遗失 | 立即报警+登报声明+补办 |

### 防火墙场景的印章隔离

```
控股公司
  ├─ 公章（实际控制人管）
  ├─ 财务章（独立账户+独立出纳）
  └─ 法人章（实际控制人或其指定人）

子公司A
  ├─ 公章（高管管，控股公司备案）
  └─ 财务章（子公司财务管，独立核算）

⚠️ 关键：控股公司和子公司的印章分开保管，不能混用
```
