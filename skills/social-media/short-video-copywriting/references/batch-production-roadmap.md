# 短视频文案批量产出路线图

> 从"偶尔出一篇"到"每周稳定2-3条成品"的完整路径。
> 与 SKILL.md 中的单篇创作方法论互补——那边是战术，这边是战略。

## 一、定义：什么叫"批量产出"？

不是日更10条。标志：

```
① 选题不用想——有稳定的素材来源管道
② 结构不用编——每类文案有成熟的公式/模板
③ 修改不费劲——修改量从80%降到20%
④ 产出可预期——每周稳定出2-3条成品文案
```

## 二、总时间线：7-10 周

```
阶段一（管线搭建）  → 第1-3周
阶段二（数据打磨）  → 第4-7周
阶段三（流程固化）  → 第8-10周
          ↓
   🎉 批量产出阶段达成
```

## 三、阶段一：管线搭建（第1-3周）

**目标**：三条线各建好"原料→加工→成品"管道，闭着眼也能产出一条及格线以上的文案。

### 步骤

| # | 动作 | 投入 | 产出 |
|---|---|---|---|
| 1.1 | 定线—三条线各选一个主打套路固化 | 半天 | 3个套路模板（含钩子/结构/话术） |
| 1.2 | 建素材库—按"高级会计师名单"模式再挖2-3个独家素材 | 1天 | 3套"一素材多文案"弹药包 |
| 1.3 | 定发布节奏—固定每周几发、什么类型、几点发 | 1小时 | 排期表 |

### 三条线的固化方向

| 路线 | 母题建议 | 动结构 | 适合参考素材 |
|---|---|---|---|
| **范话题** | 面子/认知税/人情账/社会悖论 | 立论点→挖根因→定调收网 | 韩红走面儿已验证 |
| **行业自然流** | 高级会计师/代账16年/评审内幕 | 素材五步法 | 高级会计师名单素材 |
| **纯广告** | 五句法 | 直击痛点→解释+钩子→秀价值→打消顾虑→引导行动 | short-video-ad.md |

## 四、阶段二：数据打磨（第4-7周）

**目标**：用真实数据反向修正每条线，找到各自的"最优参数区间"。

### 步骤

| # | 动作 | 说明 | 投入 |
|---|---|---|---|
| 2.1 | 发8-12条 | 三条线各3-4条，记录数据 | 正常拍摄发布 |
| 2.2 | 复盘指标 | 重点看：收藏率 > 复访率 > 5秒完播 > 互动率 | 每条后30分钟 |
| 2.3 | A/B微调 | 同套路换开头/换结尾/换配乐对比 | 随时 |

### 各类型优化重心

| 类型 | 最优比指标 | 优化方向 |
|---|---|---|
| 范话题 | 5秒完播 > 评论率 > 收藏率 | 钩子强度 + 互动引导（拉评论） |
| 行业自然流 | 收藏率 > 复访率 > 5秒完播 | 留存曲线平滑 + 长期价值感（拉收藏） |
| 纯广告 | 5秒完播 > 互动率（私信/转化） | 筛选精度——对的人留下，错的人秒划 |

### 二八法则

80%精力砸在 **VPS（内容价值）和 ENG（互动设计）** 上（详见SKILL.md中抖音算法公式）。
后三个及格线（RET ≥ 18%, TAG ≥ 65%, ACCT ≥ 70%）过了就别再纠结。

### 常见问题诊断

| 现象 | 根因 | 对策 |
|---|---|---|
| 收藏率低 | 内容"用完即走"，缺持久价值感 | 增加干货密度 / 总结性金句 |
| 复访率低 | 没有记忆点，看完记不住你是谁 | 强化个人标签/人设锚点 |
| 5秒完播低 | 钩子不对或封面不行 | 换开头公式 / 测封面点击率 |
| 评论少 | 没有互动钩子 | 结尾加提问 / 留争议点 |
| 断崖式跳出 | 前期太夸张，后期内容跟不上 | 平缓留存曲线，逐层递进 |

## 五、阶段三：流程固化（第8-10周）

**目标**：形成可重复的 SOP，"低脑力消耗"产出模式。

### 标准周节奏

```
周一（素材日）   1h   翻素材库 → 选2-3个选题
周一（框架日）   1h   同一天把3条的框架定了（钩子+结构+收尾）
周二-周四        30min/条   每天填充一条
周五-周六        发布日   按排期表发
周日（复盘日）   30min   复盘一周数据，记入素材库
```

### 固化后的产出预期

```
阶段一末期：出1条文案要改2-3轮，耗时4-6小时\n阶段二末期：出1条改1轮，耗时2-3小时\n阶段三末期：出1条微调即可，耗时1-1.5小时\n批量产出状态：每周稳定2-3条（上限5条/批），总投入约5-8小时/周
```

## 六、先决条件 & 调优建议

### 三条线同时跑还是集中一条？

**建议：头一个月只做一条线**，跑通再扩。

| 你的情况 | 优先起的线 |
|---|---|
| 手上素材最成熟的 | 行业自然流（高级会计师名单已验证产出4套） |
| 想快速涨粉的 | 范话题（爆发力最强） |
| 想稳定获客的 | 纯广告（筛选精准，转化直接） |

### 何时可以进入下一阶段？

| 阶段 | 合格标志 | 不合格→继续在阶段内 |
|---|---|---|
| 一→二 | 三条线各有一套可复用的模板结构 | 还在每次从零想结构 |
| 二→三 | 每条线至少发了3条数据，知道哪些指标低 | 数据不足，无法判断方向 |
| 三→🎉 | 每周稳定产出2-3条，修改量＜20% | 每周仍要花大量时间改稿 |
