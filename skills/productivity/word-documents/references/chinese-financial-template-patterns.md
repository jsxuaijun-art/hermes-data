# 中文财税模板文档结构模式

从「期初余额调整表模板」提取的可复用模式，适用于税务合规报告、纳税调整表、会计政策说明书、账务清理方案等财税类文档。

---

## 一、文档骨架

```
1. 标题 + 适用场景说明
   - 大标题（黑体，18pt/22pt，居中）
   - 小标题或副标题
   - 编制依据（引用具体准则/法规条文）
   - 编制人/日期填空

2. 核心思路示意
   - 用箭头流程图展示逻辑（不是纯文字描述）
   - 格式：左对齐 + 仿宋 + 缩进，每行一个步骤

3. 主表（资产负债表/调整总表）
   - 全科目展开（资产→负债→权益）
   - 每列有计算公式标注（如 ④=②+③）
   - 底部有勾稽复核公式

4. 分项明细表（每组一个）
   - 科目名做小标题
   - 附具体计算表（固定资产折旧明细、摊销明细等）
   - 附计算公式（符号化，不写长句）
   - ⚠️ 风险提示用 emoji 或加粗标出

5. 配套凭证分录
   - 借/贷标准格式
   - 一个场景一个分录块
   - 注解说明什么情况下用

6. 操作建议
   - 步骤化，每条一句话
   - 前置条件（先找什么）
   - 注意事项（留痕、存档）
```

---

## 二、排版规范

| 元素 | 字体 | 字号（pt） | 加粗 | 对齐 |
|------|------|-----------|------|------|
| 大标题 | 黑体 | 22 | 是 | 居中 |
| 节标题 | 黑体 | 14 | 是 | 左对齐 |
| 小标题（科目/备注名） | 楷体 | 12 | 是 | 左对齐 |
| 正文 | 仿宋 | 12 | 否 | 两端对齐 |
| 表格表头 | 黑体 | 9 | 是 | 居中（深蓝底白字） |
| 表格正文 | 仿宋 | 9 | 否 | 居中 |
| 尾注 | 仿宋 | 9 | 否 | 居中（灰色）|

字号换算（给生成代码用）：
```
pt → w:sz 值（半磅）：
22pt → 44（大标题）
14pt → 28（节标题）
12pt → 24（正文/小标题）
9pt  → 18（表格文字）
```

## 三、表格模式

所有财税模板表格遵循统一结构：

```
1. 全边框（上下左右 + 内纵横线）
2. 表头行：深蓝底（#1A3C6E / D9E2F3）/ 白字 / 加粗
3. 数据行：偶数行浅灰蓝（#F0F4FA）交替
4. 首列：加粗（科目/序号列）
5. 末列/末行：汇总公式（加粗或底色区分）
6. 列宽：第一列文字列宽大（1000-1300dxa），其余均匀分布
7. 单元格垂直居中
```

---

## 四、凭证分录格式

```
借：会计科目                                XXXXX元
    贷：会计科目                                   XXXXX元
```

- "借"后空两格（两个全角空格）
- "贷"前空两格
- 金额右对齐
- 一个完整分录用一个空段落隔开
- 同一个月/同一类调整的分录连续排列

---

## 五、场景适配

| 文档类型 | 主表 | 核心调整项 |
|----------|------|-----------|
| 期初余额调整 | 资产负债表 | 折旧补提、摊销补提、往来清理 |
| 税务自查报告 | 税金差异汇总表 | 增值税/所得税差异逐项说明 |
| 纳税调整 | 纳税调整项目明细表 | 调增/调减项（超支、不征税收入等） |
| 会计分录调整 | 科目余额调节表 | 差额分析、冲销、重分类 |
| 资产清查报告 | 资产盘点明细表 | 盘盈/盘亏、毁损、报废 |

---

## 六、风险提示格式

使用 ⚠️ 前缀 + 加粗正文，放在对应章节底部：

```
⚠️ 存货如果存放6年，实物大概率已不存在或报废。以实地盘点为准，差额计入营业外支出。
```

字体：仿宋 12pt，红色（#CC0000）或纯黑加粗，建议红字更醒目。

---

## 七、生成优先级

当需要同时生成 .md 和 .docx 时：

1. 先用 Markdown 写内容（保证逻辑完整）
2. 转 .docx 时应用以上排版规范
3. 保证 .docx 版本字体内嵌（仿宋/黑体/楷体在 Windows 上均标准预装）
