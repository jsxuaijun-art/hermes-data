# 合规账业务流程与风险识别方法论

> 来源：盈信财税 · 合规账业务开展流程及话术 V3.0
> 适用：内部培训、新客交付SOP、团队协作
> 配套：流程话术、风险指标清单、报价策略

---

## 一、合规账业务五步曲

```
客户邀约 → 谈单 → 签约 → 交付 → 复盘跟进
   ↓        ↓       ↓       ↓        ↓
  电话邀约  挖掘痛点  签合同  历史清理   月度分析
  存量转化  方案报价  收定金  天周月报   季度复盘
```

## 二、第一步：客户邀约

### 1. 老客邀约话术

> 「您好*总，我是盈信财税顾问 xxx。金税四期上线后税务监管越来越严格，为了帮助您应对稽查、摆脱节税难，我们诚意邀请了金四专家，帮您梳理业务、降低税务风险。您看这周什么时候有空，我们一起坐下来交流一下贵公司的业务情况。」

### 2. 续费客户邀约话术

> 「您好*总，咱们合同还有**个月到期。金税四期上线了，税务监管越来越严，我们推出了代账升级服务，主要是帮您更好地规避税务风险。您看这周哪天方便，给您讲讲金四对中小企业的影响，帮您把把关。」

### 邀约要点

| 要点 | 操作 |
|------|------|
| 紧迫感 | 把「税务合规」陈述得很紧急、很重要，激发客户兴趣 |
| 扣话术 | 每句话末尾形成问句，延长通话时间 |
| 定时间 | 详细到几月几日几点，不模棱两可 |
| 显重视 | "我们公司成立财税合规部，我第一个想到您"——体现粘性 |

## 三、第二步：谈单（完整流程）

### 3.1 挖掘痛点

**两个维度：**
- **对代账公司自己**：低价竞争、同行切户、续费难、金四自动算税
- **对服务企业**：金四从粗放向精细、以票治税向以数治税、业务真实性监管

**税务重点稽查7个方面（钉入客户认知）：**

| # | 稽查重点 |
|---|----------|
| 1 | 高净值/高收入人群、股权转让股东税务监管 |
| 2 | 严查虚开发票（含自然人代开） |
| 3 | 部分重点行业行业式稽查 |
| 4 | 公转私、个人卡收款不入账、隐匿收入 |
| 5 | 税收洼地、核定征收、委托代征 |
| 6 | 税筹平台（灵活用工平台、私车公用平台等） |
| 7 | 定期选案稽查——重点稽查对象名录库+异常稽查对象名录库 |

### 3.2 开场引入

**首次面谈话术（金四环境切入）：**
> 「过去金税三期实现了国地税数据合并，主要是对税务系统业务流程的全监控。金税四期——数电票、数字信息账户、确认制申报、税银互联——不仅仅是税务，还会纳入'非税'业务，搭建了各部委+人民银行+银行之间的信息共享和核查通道。您企业的手机号码、纳税状态、登记信息、银行账户、社保、医保、上下游信息……全方位立体采集监控。」

**客户不重视 → 心态引导：**
> 「局端数字账户上线后，加入了'五表两票'比对。过去主要看发票，现在用五表两票的信息比对推导风险隐患。不论您是有意还是无意，您企业的数据局端其实都已经知道了。税局会将归集数据和您申报的数据比对——目前在广州、上海、成都已经推出确认制申报。」

**切入客户业务（让客户多说）：**
引导客户讲业务：销售与收款→采购与付款→存货及服务→长期资产→资金通道。
然后讲具体风险成因：收入/成本/利润表异常、资产负债表异常。
大类风险：**虚开、买票、虚增成本、隐瞒收入、公私不分、资金转移**

> 「在财务报表上，这些风险至少有这些呈现：6大往来数据异常、存货余额高、收入成本不匹配、费用构成不合理。做账如果只停留在发票层面，很多风险是看不出来的。」

⚠️ **关键：不要把全部解决方案都告诉客户，客户知道"有专业的人能帮他"就够了，保留神秘感。**

### 3.3 深挖需求——16大风险指标

| # | 风险指标 | 核心原因 | 关键话术方向 |
|---|----------|----------|------------|
| 1 | **应收账款余额过高（超收入60%）** | 开票金额>收款，现金票据不全、开票私户收款 | "金四模型下应收账款占比超过60%系统报红" |
| 2 | **预收账款余额过大（超收入20%）** | 收款金额>开票，公户收款未开票 | "可能被判定隐瞒收入" |
| 3 | **长期0申报（连续36个月）** | 无经营/体外循环/公户未开票 | "网格员会打电话核实有无经营" |
| 4 | **其他应付款余额过大** | 股东垫支/隐匿收入借款经营/虚开资金回流 | "税局重点稽查指标——可能被认定隐匿收入" |
| 5 | **进项发票超3个月未抵扣（滞留票）** | 有销售未开票/取得虚开发票 | "不认证也要认证，做转不抵扣处理" |
| 6 | **存货余额占收入比例过高（超50%）** | 有销售未开票/滞销 | "未入账收入导致库存只能一直挂着" |
| 7 | **有销售无采购** | 进项票缺失 | "无法抵扣增值税，也无法税前扣除" |
| 8 | **长期未销售（超6个月无出库）** | 进销不匹配/编码错误/报废未报备 | "税务根据采购发票时间认定已入库" |
| 9 | **取得大额发票未入账** | 未指定数电票入账用途标签 | "可能认定虚假交易、虚开" |
| 10 | **应付账款余额过大（超收入60%）** | 付款<采购发票，付款凭证不全 | "金四模型报红，需自证合理性" |
| 11 | **列支成本费用无发票（暂估挂账）** | 无法取得成本票/因税点不开票 | "暂估不及时冲销=虚列成本=利润虚高" |
| 12 | **无采购发票（暂估）** | 存货凭证不全/上游不能开票 | "不能抵扣进项，不能税前扣除" |
| 13 | **毛利率低（有费用无收入）** | 成本凭证不全/费用结构不合理 | "长期只有费用没有收入，不符合业务逻辑" |
| 14 | **成本大于收入（成本倒挂）** | 有销售未开票/虚开发票 | "税局重点检查对象" |
| 15 | **顾问咨询费过大/费用异常/无车却有大量车费** | 虚开发票/费用管理不善 | "金四不仅关注金额绝对值，还关注费用构成" |
| 16 | **工资金额异常（长期3500/5000临界点）** | 虚假工资 | "已列为中风险疑点数据" |

### 3.4 产品介绍与报价

**产品介绍话术：**
> 「xx总，刚刚给您介绍了金税四期核心内容，并将企业财务报表涉税风险做了解读。为了降低税务稽查风险，我们推出合规账服务——参照税局稽查内容，建立'企业数字账户'，规范交易结构、经营收支预测、发票管理、对账管理、财务核算、税务风险反馈、财务报告分析。」

> 「我们会帮您做历史账务数据分析，整理风险内容，出具调整方案，按周服务、每月/季度生成'财务服务报告'。报告中详细说明：销售与收款、采购与付款、存货与成本、资产折旧、经营收支、费用风险、人工薪酬、纳税情况核心数据及风险。专家会帮您做报告解读，给出合理化建议，在双方共同努力下把税务风险降低至可接受水平。」

**报价参考（原资料标准价）：**

| 客户类型 | 报价区间 | 成交价区间 |
|----------|----------|-----------|
| 小规模纳税人 | 12,800~15,000/年 | 9,800~12,000/年 |
| 一般纳税人 | 15,800~20,000/年 | 15,000~19,800/年 |

**报价参照维度：**
- 税款节约金额
- 坏账处理价值、库存差异
- 特殊行业合规需求
- 风险点排查数量（税负率/毛利率/进销不符/公转私/往来挂账）
- 票据及凭证数量
- 客户付款能力
- 需建立的台账数量（往来/存货/滞销/费用）

**存量客户折算：**
> 小规模剩余10个月，每月200元→折算2000元，实付10000元
> 一般人剩余10个月，每月500元→折算5000元，实付13000元

### 3.5 异议处理（6大常见）

| 异议 | 话术要点 |
|------|----------|
| **担心数据安全** | 每客户授权独立账号+1v1管理会计+保密协议 |
| **项目成熟度** | 全国已开展，北京上海广州重庆大连天津等一线城市均有覆盖 |
| **价格高** | 和稽查罚单比——"一旦税局处罚金额远超服务费" |
| **服务质量** | 团队服务（规划师+主管会计+专家老师），15日内出历史清理方案 |
| **对比同行** | 盈信优势：专家团队+智能系统+财务服务报告+多年合作信任 |
| **具体怎么落地** | "每周服务，发票/银行/合同三流合一，往来/库存/税负指标合理化" |

## 四、第三步：签约

### 签约前准备

```
□ 合同一式2份
□ 升级产品报价服务清单
□ 交付服务周期说明
□ 收款账户信息
```

### 签约注意事项

1. 客户打款后签合同；特殊情况可先签后付
2. 产品名称与清单一致
3. 存量客户剩余月份折算抵扣，合同写明
4. 客户资金不足→先付定金保成交

## 五、第四步：交付流程

### 5.1 初始化阶段（签约后7~15天）

```
第1~7天：现场初始化沟通，客户信息采集
第8~15天：历史账务数据核对，出具合规方案初稿
第15天后：方案讲解+双方确认+进入常规交付
```

### 5.2 常规交付节奏

| 频率 | 交付内容 |
|------|----------|
| **周度** | 发票/银行/合同数据核对，三流一致性检查 |
| **月度** | 财务服务报告（经营数据+风险指标+合规建议） |
| **季度** | 深度分析+风险排查+方案调整 |
| **年度** | 服务回顾+来年规划+续约沟通 |

### 5.3 财务服务报告核心内容

1. **销售与收款分析** → 应收账款周转率、回款周期
2. **采购与付款分析** → 应付账款周转率、发票取得情况
3. **存货与服务成本** → 库存周转、BOM配比
4. **资产与折旧摊销** → 资产利用率
5. **经营收支与费用** → 费用结构、异常变动预警
6. **人工薪酬分析** → 个税申报合理性、社保基数匹配度
7. **纳税情况** → 税负率分析、优惠享受情况

## 六、第五步：复盘与跟进

### 6.1 客户分析（谈后复盘）

| 维度 | 方法 |
|------|------|
| 痛点挖掘 | 是否挖出客户痛点？配合是否到位？ |
| 价格博弈 | 客户付费能力？底价是否给到位？ |
| 合作意图 | 真实想合作还是套方案？ |
| 客户类型 | 粗放型→讲风险；精细型→出具体方案 |

### 6.2 跟进节奏

```
次日跟进（价格未成交）
  → 老板直接打电话，让价逼单
  → "我也不赚您钱，给您一个本钱XXX元"

次日跟进（需和股东商议）
  → "您和股东商议怎么样了？我想这月就开始给您服务"
  → 没谈好就预约股东见面时间

次日跟进（非决策人）
  → 预约和老板见面时间
```

## 七、团队协作模式

### 角色分工

| 角色 | 职责 |
|------|------|
| **财税规划师** | 专业方案输出、风险解读、报告讲解 |
| **代账会计** | 客户信任基础、日常配合、案例代入 |
| **主管会计** | 账务合规处理 |
| **专家老师** | 疑难方案支持（注册税务师/会计师） |

### 配合技巧

> 规划师讲专业时客户产生防范情绪 → 代账会计适时加入话题讨论 → 把客户从"被教育"拉入"一起探讨"的状态 → 模拟案例情景 → 提升信任感。

## 八、报价模型（3种模式）

### 模式A：标准报价（新客）
```
小规模 → 12,800 ~ 15,000/年
一般人 → 15,800 ~ 20,000/年
```

### 模式B：存量升级（原代账客户）
```
剩余月份 × 原价折算 → 补齐差价升级
```

### 模式C：方案溢价（高复杂度客户）
```
根据风险点数量/行业复杂度/票据量 → 上浮20~30%
```
