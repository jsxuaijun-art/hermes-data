# Chinese Tax Calculation Examples & High-Tech Enterprise Risk Analysis

## 个体户A — 查账征收·小规模纳税人

**基本数据:**
- 开票收入: 1,600,000 元
- 有票据成本: 80,000 元 (可税前扣除)
- 无票暂估成本: 1,520,000 元 (无合法凭证, 不得扣除)
- 业主减除费用: 60,000 元/年

**如实申报计算:**
```
应纳税所得额 = 1,600,000 - 80,000 - 60,000 = 1,460,000 元
应纳税额     = 1,460,000 × 35% - 65,500 = 445,500 元
```

**虚列152万暂估的后果:**
- 少缴税款: 445,500 元 (占应纳税额30.5%)
- 行政罚款: 222,750 ~ 2,227,500 元 (税款50%~5倍)
- 刑事: 《刑法》§201 数额巨大 → 3~7年有期徒刑

**增值税 (小规模1%优惠):**
```
不含税收入 = 1,600,000 ÷ 1.01 = 1,584,158 元
增值税     = 1,584,158 × 1% = 15,842 元
附加税费   ≈ 15,842 × 12% × 50% = 951 元 (减半征收)
```

## 个体户B — 查账征收·一般纳税人

**基本数据:**
- 开票收入: 1,800,000 元
- 进项成本票: 600,000 元 (有合法凭证)
- 虚列暂估支出: 600,000 元 (无合法凭证)
- 业主减除费用: 60,000 元/年

**如实申报计算:**
```
应纳税所得额 = 1,800,000 - 600,000 - 60,000 = 1,140,000 元
应纳税额     = 1,140,000 × 35% - 65,500 = 333,500 元
```

**虚列60万的后果:**
- 少缴税款: 210,000 元 (占应纳税额17.8%)
- 刑事: 《刑法》§201 数额较大 → 3年以下有期徒刑

**增值税 (一般纳税人13%):**
```
不含税收入   = 1,800,000 ÷ 1.13 = 1,592,920 元
销项税额     = 1,592,920 × 13% = 207,080 元
进项税额     = 600,000 ÷ 1.13 × 13% = 69,027 元
应交增值税   = 207,080 - 69,027 = 138,053 元
```

## 公司 — 一般纳税人 (合并方案)

**基本数据:**
- 合并收入: 3,400,000 元 (1,600,000 + 1,800,000)
- 原材料成本: 2,040,000 元 (60%)
- 人工成本: 1,020,000 元 (30%)
- 总成本: 3,060,000 元
- 利润总额: 340,000 元

**企业所得税 (小型微利企业优惠):**
```
条件: 应纳税所得额 340,000 < 3,000,000 → 符合
应纳税额 = 340,000 × 25% × 20% = 17,000 元 (实际税负5%)
税后利润 = 340,000 - 17,000 = 323,000 元
```

**增值税 (一般纳税人13%):**
```
不含税收入   = 3,400,000 ÷ 1.13 = 3,008,850 元
销项税额     = 3,008,850 × 13% = 391,150 元
进项税额     = 2,040,000 ÷ 1.13 × 13% = 234,690 元
应交增值税   = 391,150 - 234,690 = 156,460 元
```

**股东分红个税 (全额分红):**
```
分红个税     = 323,000 × 20% = 64,600 元
股东实得     = 323,000 - 64,600 = 258,400 元
综合税负率   = (17,000 + 64,600) ÷ 340,000 = 24.0%
```

## 高新技术企业从偷税个体户取得专票的风险

### 风险链条

个体户偷税被查 → 金税四期穿透 → 高新企业被列入"受票方异常清单"

### 风险1: 虚开增值税专用发票罪 (§205)

| 虚开税额 | 刑期 |
|---------|------|
| ≥10万 | ≤3年 |
| ≥50万 (数额较大) | 3~10年 |
| ≥250万 (数额巨大) | 10年~无期徒刑 |

**追责人员**: 法定代表人、财务负责人、采购负责人、税务会计

**"善意取得"不免责**: 即使不知情, 进项税额仍不得抵扣 (国税发〔2000〕187号)

### 风险2: 高新资格取消 + 10年追缴

| 追缴项目 | 以年利润500万为例, 10年 |
|---------|:---------------------:|
| 企税税率差 (25%-15%) | 500万 |
| 研发费用加计扣除 | 500万 |
| 地方财政返还 | 200万 |
| **合计** | **1,200万+** |

依据: 《高新技术企业认定管理办法》(国科发火〔2016〕32号) §19; 国家税务总局公告2017年第24号

### 风险3: 34部门联合惩戒

- D级纳税人 (发票受限, 重点监控)
- 限制乘飞机高铁、限制出境
- 禁止政府采购/招投标
- 上市/融资通道彻底堵死
- 银行抽贷断贷

### 风险4: 总损失估算 (最严重情形)

| 损失项 | 金额 |
|-------|:----:|
| 补缴增值税 | 300~800万 |
| 增值税罚款 | 150~4,000万 |
| 增值税滞纳金 | 100~500万 |
| 补缴企业所得税 | 500~2,000万 |
| 追缴高新优惠 | 500~1,500万 |
| 追缴研发加计扣除 | 300~1,000万 |
| **合计** | **1,900万~1亿+** |

### 高风险信号自查

1. 供应商为个体户且开票量远超其经营规模
2. 报价显著低于市场均价 (>20%)
3. 货款支付至个人账户
4. 供应商无实际经营场所 (虚拟地址)
5. 供应商纳税信用D级或非正常户
6. 发票品名与供应商经营范围不符
7. 合同、发票、资金、物流"四流"不一致
8. 供应商已注销或失联
