## 三、方案分析：江姐方案验证 ✅+⚠️

### 方案结构

```
南通咖萌（南通）
  └── 苏州卡尔蒙（苏州·全资子公司）    ← 独立法人
        ├── 分公司①（苏州门店1）        ← 非独立核算
        ├── 分公司②（苏州门店2）        ← 非独立核算
        └── 分公司③（苏州新门店）       ← 非独立核算
```

### 核心主张验证

**① "同地级市分公司可以非独立核算" → ✅ 正确**
- 国税2012年57号公告：跨地区汇总纳税办法主要针对跨省分支机构
- 同一地级市内的分支机构，可申请非独立核算，由总公司统一申报
- 苏州卡尔蒙和它的3个分公司同在苏州，满足条件

**② "分公司之间盈亏互抵" → ✅ 正确**
- 非独立核算的分公司，收入利润汇总到总公司（苏州卡尔蒙）
- 苏州卡尔蒙统一核算，分公司间的亏损和盈利自动互抵
- 这是非独立核算的核心优势

**③ "汇总到总公司可以抵消亏损" → ⚠️ 需要分清"总公司"指哪个**

```
层级1：南通咖萌（母公司）←→ 苏州卡尔蒙（子公司）
       → 母子公司独立纳税，不能合并，不能盈亏互抵 ❌
       → 依据：《企业所得税法》第52条

层级2：苏州卡尔蒙（总公司）←→ 3个分公司
       → 总分机构统一核算，可以盈亏互抵 ✅
```

**结论**：江姐的判断**大方向对**，但表述需要澄清——"汇总到总公司"如果指**苏州卡尔蒙**，正确；如果指**南通咖萌**，不成立。

## 四、方案验证：两个选项的完整对比

### 方案A（江姐方案）：设子公司→下设分公司 ✅推荐

| 维度 | 分析 | 评级 |
|------|------|:----:|
| 同城分公司盈亏互抵 | 苏州卡尔蒙3个分公司同在苏州，可互抵 | ✅ |
| 母子公司关系 | 南通咖萌↔苏州卡尔蒙，独立纳税，不能互抵 | ⚠️ 注意 |
| 管理复杂度 | 多一层法人，多一套账+一套申报 | ⬇️ 减分 |
| 风险隔离 | 子公司独立法人，南通咖萌不直接承担苏州门店风险 | ✅ 加分 |
| 未来利润回流 | 苏州卡尔蒙盈利→分红给南通咖萌（居民企业分红免税） | ✅ |
| 成本归属 | 劳务派遣+房租发票开给南通咖萌→需建立内部结算 | ⚠️ 待办 |
| 个体户注销 | 注销前需处理私户差额补税，否则有风险 | ⚠️ 待办 |

### 方案B（替代方案）：南通咖萌直接在苏州设分公司

| 维度 | 分析 | 评级 |
|------|------|:----:|
| 跨市汇总纳税 | 南通→苏州属省内跨市，需按江苏省规定处理 | ⚠️ 待确认 |
| 三因素分配 | 省内跨市通常需就地预缴（按三因素） | ⬇️ 减分 |
| 互抵范围 | 可直接与南通咖萌总部的利润/亏损互抵 | ✅ 加分 |
| 管理复杂度 | 少一层法人，简单 | ✅ |
| 风险承担 | 南通咖萌直接承担苏州门店全部风险 | ⬇️ 减分 |
| 增值税申报 | 需苏州属地申报 | ⚠️ 同方案A |

### 最终建议：采用方案A（江姐方案），但需纠正+补丁

纠正：明确"汇总到总公司"是苏州卡尔蒙，不是南通咖萌
补丁：①建立南通咖萌→苏州卡尔蒙的内部结算机制 ②个体户注销前补税 ③劳务派遣+房租发票抬头平移至苏州卡尔蒙

## 五、个体户注销的税务风险

### 核心问题
爽煌的2个个体户核定28,000/月，实际5~6万/月
差额（2~3万/月）通过私户收款，未申报

### 注销时的风险
1. 税务局注销时通常会核查近3年申报数据的完整性
2. 如果发现银行流水（私户）与申报收入不符
3. 可能：补税+滞纳金+罚款

### 建议处理方式
① 注销前主动补申报差额部分（按实际收款补增值税+经营所得个税）
② 补税后取得完税证明，再申请注销
③ 确保私户差额补缴后再注销，避免历史问题遗留

## 六、成本归属问题

### 现状
- 劳务派遣公司开发票→给南通咖萌
- 房租开发票→给南通咖萌

### 方案A下的处理
苏州门店是苏州卡尔蒙的分公司，成本应归属苏州卡尔蒙
→ 南通咖萌代付 → 通过内部往来结算 → 苏州卡尔蒙确认成本

### 优化建议
① 劳务派遣合同重新签：甲方改为苏州卡尔蒙
② 房租合同重新签：承租方改为苏州卡尔蒙
③ 过渡期：南通咖萌代付→按月内部结算
