## 二、涉税专业服务管理办法（58号令）核心要点

### 2.1 适用范围

代理记账机构被明确列为涉税专业服务机构（第2条），与税务师事务所、会计师事务所并列。

### 2.2 服务分类

| 类别 | 服务项目 | 监管要求 |
|------|---------|---------|
| 一般服务 | ①纳税申报代办 ②一般税务咨询 ⑦其他事项代办 ⑧其他税务代理 | 门槛相对低 |
| **特定服务** | ③专业税务顾问 ④税务合规计划 ⑤涉税鉴证 ⑥纳税情况审查 | **报告要素信息，监管更严** |

> 高端会计咨询 -> ③④类特定涉税服务

### 2.3 关键义务

| 义务 | 时限 | 处罚后果 |
|------|------|---------|
| 首次提供服务前报送基本信息 | 首次服务前 | 第31条：提示->重点监管->失信主体 |
| 申领信用码（机构+人员） | 取得资格后 | 无信用码影响业务承接 |
| 报送业务委托协议 | ①②⑦⑧：服务前；③④⑤⑥：30日内 | 每份未报送扣2分，最高扣40分 |
| 特定服务报业务要素信息 | 完成服务后 | 逾期扣10-20分，未报送扣25-50分 |

### 2.4 处罚体系

**第31条（较轻违规，10类）**：
- 首次：提示/约谈 -> 扣分
- 较重/逾期不改：重点监管对象 -> 降级 -> 向委托人风险提示
- 严重：涉税服务失信主体公告 -> 共赴现场办理

**第32条（严重违规，9类 — 高压线）**：
- 存在：重点监管 -> 扣分 -> 向委托人风险提示
- 较重：**失信主体**公告 -> 共赴现场办理
- 严重：**严重失信主体** -> 联合惩戒 -> 吊销资质

> 关键红线：协助偷逃税、出具虚假意见、恶意低价竞争、歪曲解读政策、向税务机关人员输送利益

### 2.5 罚款标准

| 违规类型 | 机构最高 | 人员最高 |
|---------|---------|---------|
| 第31条（较重） | 1,000元 | 500元 |
| 第31条（严重） | 2,000元 | 1,000元 |
| 造成少缴税款（人员未罚时） | — | 5,000元 |
| 第32条②-⑨（未按其他法律处罚时） | 10,000元 | 5,000元 |

## 三、TSC信用评价体系（2026年第1号公告）

### 3.1 积分总表

| 一级指标 | 分值 | 说明 |
|---------|------|------|
| 01 上一评价周期信用情况 | 90分 | 含涉税信用40分+公共信用50分 |
| 02 委托人纳税缴费信用 | 30分 | 客户纳税信用直接影响你的分 |
| 03 纳税人和税务机关评价 | 50分 | 客户评价20分+税务机关评价30分 |
| 04 实名办税 | 90分 | 机构实名5+人员实名45+协议40 |
| 05 业务结构 | 10分 | 涉税收入占比 |
| 06 服务质量 | 120分 | 报告备查20+执业规范100 |
| 07 业务信息质量 | 50分 | 行政规定遵守30+单独报送20 |
| 08 行业自律 | 15分 | 行业协会会员 |
| 09 党的建设 | 20分 | 党员5+党组织15 |
| 10 人员信用 | 10分 | 人员积分变动 |
| 11 业务培训 | 15分 | 培训人数10+测试通过率5 |
| **总分** | **500分** | |

### 3.2 等级标准

| 等级 | 分数 | 激励/约束措施 |
|------|------|-------------|
| **TSC5级** | >=400分 | 绿色通道；发票按更高一级管理；批量申报便利化；政府购买优先 |
| TSC4级 | 300-399分 | 正常管理+政策辅导 |
| TSC3级 | 200-299分 | 正常管理 |
| TSC2级 | 100-199分 | 分类管理+重点关注+日常检查重点 |
| TSC1级 | <100分 | 向委托人风险提示；双方面签；**停止在线服务** |

### 3.3 不得评TSC5的情形
- 上期纳税缴费信用D级
- 报送基本信息不满3年
- 上期内连续3个月或累计6个月中止服务

### 3.4 直接判TSC1级的情形（触犯第32条红线）

第32条以下情形，信用积分直接扣至0分，直接判为TSC1级：
1. 违反税法造成委托人未缴少缴税款或骗取税收优惠（060202）
2. 出具虚假意见（060203）
3. 不正当手段承揽业务（恶意低价等）（060204）
4. 歪曲解读税收政策（060205）
5. 唆使诱导帮助他人涉税违法违规（060206）
6. 利用服务之便谋取不正当利益（060207）
7. 以税务机关名义敲诈纳税人（060208）
8. 向税务机关人员输送利益（060209）
9. 其他违反税收及相关法律法规行为（060210，扣100-300分）

### 3.5 评分操作指南

**高性价比加分项**（容易拿的分）：
1. 人员全部领取信用码（040201，30分）：确保所有在岗人员都申领信用码
2. 专业人员资格证书加分（040202，10分）：每有1名持证人员（税务师/注会/律师/中级会计师）加2.5分
3. 培训加分（1101，10分）：全员参加税务机关培训，年累计>=12小时
4. 行业自律（0801，15分）：加入税务师协会或省级认可的涉税行业协会
5. 党建加分（0902，15分）：设立党组织或纳入区域性党组织管理

**最容易踩坑的扣分项**：
1. 协议未按时报送（0403，每份扣2分，最高扣40分）
2. 代理纳税申报错误（060201，每次扣10分）
3. 特定服务业务信息逾期/未报送（070101，最高扣50分）
4. 报告和文书未留存备查（0601，每份扣1分）

### 3.6 信用修复

| 主体类型 | 最短公示期 | 可申请修复条件 |
|---------|-----------|--------------|
| 涉税服务失信主体 | 3个月 | 义务履行完毕+信用承诺书 |
| 涉税服务严重失信主体 | 1年 | 义务履行完毕+信用承诺书 |

修复流程：申请 -> 3个工作日受理 -> 7个工作日修复结果（最长+7日核查）

## 五、代理记账基础工作规范（财会〔2023〕27号）

### 5.1 全流程八步法

```
业务承接 -> 工作计划 -> 资料交接 -> 会计核算 -> 质量控制 -> 人员管理 -> 档案管理 -> 附则
```

### 5.2 业务承接要求

委托合同必须包含9项内容（第8条）：
1. 委托业务范围及其他预期目标
2. 会计资料传递程序和签收手续
3. 双方对会计资料真实性/完整性/合法性的责任
4. 收费
5. 有效期间
6. 签约时间
7. 违约责任
8. 解决争议方法
9. 双方认为应约定的其他事项

### 5.3 质量控制岗位设置（第25条）

必须设置3个岗位，且**项目负责人员和质量控制人员不得为同一人**：
- 项目负责人员 — 具体负责代理记账业务
- 质量控制人员 — 审查复核
- 业务负责人 — 主管代理记账业务

### 5.4 复核制度（第27条）

- 至少执行**一级复核程序**
- 重大项目至少执行**二级复核程序**
- 定期抽查未经二级复核的业务

### 5.5 档案管理

- 会计档案查阅/复制/借出需授权审批
- 会计年度终了后向委托人移交会计档案
- 受托继续保管的按《会计档案管理办法》执行

## 六、苏州工业园区执业规范指南要点

### 6.1 会计核算制度选择（第8条）

| 委托单位类型 | 适用制度 |
|------------|---------|
| 个体工商户（注册资金>=20万或销售额达标） | 复式账 |
| 个体工商户（注册资金10-20万或销售额达标） | 简易账 |
| 小微企业（按行业标准） | 《小企业会计准则》 |
| 村集体经济组织 | 《村集体经济组织会计制度》 |
| 上市公司/大型集团 | 《企业会计准则》 |
| 其他 | 《企业会计制度》 |

### 6.2 三级复核制度

| 审核层级 | 审核内容 | 留痕方式 |
|---------|---------|---------|
| 记账会计 | 科目准确性、附件完整性 | 《日常记账审核表》 |
| 主管会计 | 税务合规性、政策适用性 | 《重点科目复核记录》 |
| 质控部 | 随机抽查（每月>=30%） | 《质量抽查报告》 |

### 6.3 电子档案管理

满足以下条件可**不输出纸面资料**：
- 重复发生的日常业务
- 系统自动生成
- 可及时查询和输出
- 有防篡改机制
- 有电子备份制度
- 有完善索引体系

### 6.4 从业人员培训要求

- 每年不低于24小时继续教育（财政部门认可机构）
- 每季度业务考试，及格线80分
- 培训考试成绩记入个人档案，作为转正/升降级依据

### 6.5 信息化系统要求

代理记账机构开展财税代理业务应通过信息化系统完成。**会计核算、纳税申报必须实现信息化，不得手工处理。**

### 9.3 全流程工作指引

**接新客户时的标准操作流程**：

```
第1步：客户初步接触
  -> 填写《客户资质审查表》   (审查客户是否合规)
  -> 填写《风险评估表C-02》   (评估业务风险等级)
  -> 签署《保密协议》         (明确保密义务)

第2步：正式签约
  -> 填写《业务承接与保持评价表》  (确认承接决策)
  -> 填写《客户信息登记表C-01》   (收集完整客户信息)
  -> 签署《代理记账业务委托合同》  (含9项必备条款)
  -> 填写《代理记账服务清单勾选表》 (明确服务范围)
  -> 编制《代理记账业务工作计划》   (安排时间线)

第3步：初次资料交接
  -> 填写《资料交接手册》         (核对旧账/资料)
  -> 填写《客户资料完整性确认书》  (确认资料齐全)
  -> 检查:银行对账、账表核对、纳税申报核对

第4步：日常运营（每月）
  -> 填写《原始凭证交接表O-01》
  -> 填写《日常记账审核表》
  -> 执行《三表比对记录》
  -> 填写《税务申报差异说明表》

第5步：季度工作
  -> 填写《重点科目复核记录》
  -> 填写《特殊业务处理台账》
  -> 填写《财税健康诊断报告》
  -> 填写《监管检查预审表Q-02》

第6步：客户服务
  -> 填写《投诉处理记录》

第7步：业务终止/年度终了
  -> 填写《交接清单》
  -> 填写《会计档案移交清册》

持续：
  -> 每月填写《月度自查表Q-01》
  -> 记录《培训考核记录》
  -> 维护《岗位权限表》
```
