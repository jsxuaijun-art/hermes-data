## 一、砍价/比价应对（从洽谈沟通技术提取）

### 客户说"最低价多少" + 客户说"太贵了"

```
🔹 核心逻辑：不反驳，接纳情绪，然后转视角
🔹 三连击法：
  ① 接纳："您嫌贵很正常，所有客户在了解产品之前都嫌贵"
  ② 转视角："但很多客户知道我们贵一点还选我们"
  ③ 给理由："不是钱多，是在低价和品质之间做出了正确的选择"
```

### 客户说"我再对比对比"

```
🔹 核心逻辑：不催不慌，给客户对比的武器
🔹 话术骨架：
"对比是应该的。但发给您的资料可以让您选择时少走弯路。
建议看三点：①对方资质 ②对方合作年限 ③对方做的凭证样本"
```

### 客户说"别家才XX元" / "朋友介绍更便宜的"

```
🔹 核心逻辑：换位思考 + 价值理论 + 挖低价底牌
🔹 话术骨架：
"请问价格比较要建立在相同条件下。
出来做生意总是要赚钱的，不是做慈善。
低价必有低价的门路——无非偷工减料、用新手。
想想自己的行业，价值理论是亘古不变的道理。"
```

## 二、低价竞争整体应对（从《面对低价营销》提取）

### 16个核心论点武器库

```
1. 客户流失：很多客户是真心做事业的，刚开始"麻烦"，后来归于平静
    → 关键在于引导，这是个技巧
   
2. 行业本质：本行业零散型，没公司有决定价格权
    → 新入行低价抢客户 → 降低品质 → 事故击垮
    → 低价竞争带来洗牌，大浪淘沙
   
3. 比价类比："老婆总是别人好，孩子总是自己好"
    → 别人家的会计总是更便宜，但换了就知道
   
4. 换位思考：对方不做慈善，低价必有门路
    → 不是研发新技术降本，是偷工减料用人头凑
   
5. 低价高质悖论："低价高质必不长久，或者有坑"
    → "不是不报，时候未到"
   
6. 三年考卷：不需要时看不出来，遇稽查必有高下
    → 问问低价公司：什么资质？经历过什么稽查案例？合作最久的客户多久？
   
7. 菜市场类比：时间长的菜才低价卖，快过期食品打折
    → 好货不便宜，便宜没好货
   
8. 护栏理论：走在桥上觉得安全因为有护栏
    → 你觉得不太需要，但护栏（专业代账）没了你马上觉得危险
   
9. 成本决定论：同样的价格比拼更多服务 = 同样的问题
    → 低价招揽来，必在看不见的地方节约
   
10. 会计隐形价值：表面都申报了，数据是否有逻辑性？
    → 一个不合理会计分录可能招致重大税务风险
    → 金税系统大数据功能会检索逻辑性
   
11. 从业者素质灵魂拷问：
    → 有没有高级会计师？几个中级？几个研究生？
    → 很多非财务人员混进代账行业
    → 报表勾稽关系都没有，资产负债表硬凑平
   
12. 代理记账许可证只是底线：
    → 要看代账业务负责人什么资质、几年从业、处理过什么案例
   
13. 选择低价之后（血泪案例）：
    → 转出去，连年报都能忘记，漏申报某个税种
    → 囫囵吞枣看似完成申报，实则隐患已生
   
14. 省200块 vs 花几万补：
    → "最终省下了两千，最后要花几万弥补"
   
15. 成本比较：
    → 正版软件按户收20元/月使用费
    → 低价公司用盗版 → 数据丢失 → 损失大了
    → 遇税务查账连账簿都没有 → 事情大了
   
16. 眼界格局：
    → 为了省几百一千转低价，只看脚下
    → "创业不易，爱惜自己"
```

## 三、报价+涨价沟通（从价格沟通案例提取）

### 给大亮的报价信（电商客户）—— 核心逻辑

```
要点1：工作量增加 → 价格需要调整
  不要只说涨了，要说"为什么涨了"——加了什么工作
  "梳理了，之后将完成以下工作量：
  ①前期方案设计、政策宣导
  ②..."（列具体工作项）
  
要点2：对比专职会计成本
  "服务费相对招聘专职会计成本还是低的"
  "专职会计月薪六七千不一定熟悉电商税务稽查经验"
  
要点3：同行比较要分清楚
  "有些同行承诺做很多——开发票、安排银行、社保登记"
  "但这些都不是核心问题。账务的合规、税务的安全才是公司的命门"
  "先比较从业经验、人员资质再比价格"
  
要点4：分级解释价格差异
  基础代账300元→收到什么票做什么票→年底提醒缺票
  一人负责六七十家到一百家→每家时间有限→精细度有限
  合规代账1200元→会计负责二十家→精细化处理→时间更多
  
要点5：可感知的比喻
  "就像开车，平直公路新手和老司机都一样开"
  "但遇到突发情况（税务专项稽查）时，差别很大"
  
要点6：说事实不辩论
  "我们的资质和经验现在说再多也无感"
  "只能明年出现事情时才能体现出来"
```

### 中宏价格调整（收入挂钩定价）—— 核心逻辑

```
要点1：用客户自己的数据说话
  "您公司1-6月营业收入12,130,381.96，全年可能超2400万"
  "对比：其他客户营业收入5549045（全年约1100万）"
  "她转给我们的服务费如下："
  → 用事实和数据做对比，不靠嘴说
  
要点2：类比法
  "好像房子面积越大交的物业费越多"
  "收入越高→工作量越大→风险越高→关注度越高→时间越多"
  
要点3：行业协会做参考
  "会计服务行业有协会，对价格也有标准"
  "一般纳税人起步500，并且随凭证数量递增"
  
要点4：合同白纸黑字
  "我们在协议里也写明了服务费按收入调整"
  "微信也有过沟通记录"
  → 涨价不是临时起意，是早有约定
  
要点5：给台阶下
  "前面的月份的价格也这样了，就当帮忙节约"
  → 不是全要，是部分调整，给客户面子
```

## 四、顾华业务员个体户沟通（新手业务员）—— 核心逻辑

```
要点1：客户砍价是人性，不是针对你
  "200元报出去一定也会再砍价——这是人性"
  "我不想落入低价竞争的恶性循环"
  "行业协会价格300元/月，我们报200已经低了"
  
要点2：成本真实存在，流程不能省
  "只要是扎实完整的记账，必要流程不能少"
  "不会因为开票少就减少流程"
  "二手房销售：100万的房子和1000万的房子流程一样"
  
要点3：要注意和个体户沟通需要更多时间
  "从核定转查账征收，很多事不知道"
  "观念和认识都要改变——没开过有限公司不懂"
  "话还不能说得过于直白→前期巨大时间成本"
```

## 五、代帐销售话建议（菜鸟 vs 销冠对比）

| 场景 | 菜鸟怎么说 | 销冠怎么说 |
|---|---|---|
| 客户问最低价 | "我给你的已经是最低价了" | "市场竞争这么激烈，我给您少一百也不会少一分。我的目的就是留下您" |
| 客户说太贵 | "我们的价格真的不贵呀" | "您嫌贵是正常的，所有客户在了解产品之前都嫌贵" |
| 客户说要对比 | "我给你发点资料你先了解" | "买不买都没关系，但我发给您的资料能让你选择时少走弯路" |
| 客户嫌价格高 | "一分价钱一分货" | "价格确实高了一点，但很多客户明知贵一点还选择我们，不是因为他们钱多" |
| 问考虑好了吗 | "王姐考虑好了吗？" | "王姐，您觉得这个方案适合您还是更中意另外一个？"（二选一） |

**一句话总结**：菜鸟解释价格，销冠翻译价值。菜鸟被动回答，销冠主动引导。
