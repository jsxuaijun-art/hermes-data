## 一、1039是什么？—— 基本概念与监管代码由来

### 1.1 定义

**海关监管代码1039**，全称**"市场采购贸易方式"**（Market Procurement Trade），是指由符合条件的经营者在经国家商务主管部门认定的市场集聚区内采购的、单票报关单商品货值在**15万美元（含）以下**，并在采购地办理出口商品通关手续的贸易方式。

### 1.2 监管代码的含义

海关监管方式代码采用4位数字编码：
- **第1-2位**表示大类："10"代表一般贸易大类
- **第3-4位**表示具体方式："39"代表市场采购贸易

### 1.3 适用主体

- 经备案的市场采购贸易经营者（通常是市场集聚区内的商户或外贸公司）
- 在市场集聚区内采购商品的境外采购商
- 为市场采购贸易提供代理服务的外贸综合服务企业（代理商）

### 1.4 核心特征

| 特征 | 说明 |
|------|------|
| 限定区域 | 必须在国家认定的市场集聚区内采购 |
| 限定金额 | 单票报关单货值≤15万美元 |
| 限定商品 | 以日用消费品、小商品为主（禁止出口商品除外） |
| 简化申报 | 可简化归类申报（每票报关单品名≤10种时可按大类归并） |
| 增值税免税 | **免征不退**（不征增值税，也不退增值税）——这是最关键的认知！ |

> **重要提示**：1039方式下，货物在出口环节**免征增值税**，但**不退税**。1039的核心价值是解决"无票合规出口"问题，而非直接解决退税问题。

## 二、1039的诞生背景与历史沿革

### 2.1 为什么需要1039？

中国拥有大量专业批发市场（如义乌小商品市场、常熟服装城等），这些市场的商户有几个共同痛点：

1. **多品种、小批量、多批次**——传统一般贸易（0110）需要逐项申报，效率极低
2. **无增值税进项发票**——市场商户多从个体户进货或现金交易，拿不到进项票
3. **无法合规出口**——没有进项票就无法做一般贸易退税，导致大量货物通过"灰关"（买单出口）等不合规方式出口
4. **外汇收汇难**——买单出口无法合规收汇，面临外汇管制风险

### 2.2 政策演进时间线

| 时间 | 事件 | 意义 |
|------|------|------|
| **2013年4月** | 商务部等部门联合发文，同意在**义乌**开展市场采购贸易方式试点 | 全国首个试点，开创先河 |
| **2014年7月** | 海关总署公告2014年第57号，正式设立海关监管方式代码"1039" | 代码正式诞生 |
| **2015年** | 试点扩大至常熟、海宁、花都等地区 | 第一批扩围（第二批试点） |
| **2016年** | 商务部等8部门发布《关于加快推进市场采购贸易方式试点工作的函》 | 正式推广 |
| **2018年** | 试点范围扩大至第三批 | 更多市场纳入 |
| **2019年9月** | 商务部等七部委联合批复第三批试点，常熟服装城在此批中获得正式批复 | 常熟试点获得明确批复 |
| **2020年** | 国务院办公厅发文支持新一批市场采购贸易方式试点 | 加速扩围 |
| **2022年** | 全国试点已达31个市场集聚区 | 覆盖主要专业市场 |
| **2024-2025年** | 持续扩围，多地获批新增试点 | 全面推广阶段 |

> **信息来源说明**：关于常熟试点获批时间，有"2015年作为第二批试点"和"2019年9月第三批正式批复"两种说法。结合多方信息综合判断，常熟服装城**2015年即纳入试点范围**，**2019年获得七部委正式联合批复文件**属于第三批集中确认。无论是哪种时间节点，截至2025年底试点均已超过5年。

### 2.3 政策出台的关键文件

1. **《关于促进市场采购贸易健康发展的指导意见》**（商务部等七部门）
2. **海关总署公告2014年第57号**——设立1039监管方式
3. **国家税务总局公告**——市场采购贸易出口货物增值税免税政策
4. **外汇管理局关于市场采购贸易外汇管理的规定**

## 三、1039的优势与劣势

### 3.1 优势

| 优势 | 详细说明 |
|------|---------|
| ✅ **合规出口通道** | 解决"无票货物"合规出口的难题，替代买单出口 |
| ✅ **简化申报** | 每票10个及以下品名的，可按"章"归类简化申报 |
| ✅ **免征增值税** | 出口环节免征增值税，无需进项发票 |
| ✅ **通关便利** | 市场集聚区设有专门海关监管场所，通关快捷 |
| ✅ **外汇收汇便利** | 支持多币种收汇（美元、欧元、日元、人民币、港币等），允许个人账户收汇 |
| ✅ **多品种拼柜** | 适合小商品多品种拼柜出口的场景 |
| ✅ **降低门槛** | 中小微企业、个体工商户均可参与 |

### 3.2 劣势与困难

| 劣势 | 详细说明 |
|------|---------|
| ❌ **不退税** | 1039是"免征不退"，无法获得出口退税 |
| ❌ **区域限制** | 必须在认定的市场集聚区内采购，操作范围受限 |
| ❌ **金额限制** | 单票≤15万美元，大宗贸易不能用 |
| ❌ **商品范围限制** | 部分敏感商品（如两用物项、侵权商品）不能走1039 |
| ❌ **税务监管严格** | 税务机关对1039业务真实性核查力度大，虚假申报风险高 |
| ❌ **操作流程复杂** | 需要在市场采购贸易综合管理平台上备案、组货、报关 |
| ❌ **需要摊位资源** | 常熟服装城需要有实体或虚拟摊位才能备案 |

### 3.3 为什么说"操作起来比较困难"

结合实际操作，1039的困难主要体现在：

1. **平台操作门槛**——每个试点市场都有自己的管理平台，需要注册、备案、录入商品信息，对不熟悉信息系统的商户是个门槛
2. **摊位问题**——常熟服装城要求必须有市场摊位租赁合同，没有摊位的企业需要租赁实体摊位、找虚拟摊位或委托代理
3. **组货流程长**——从采购到组货到报关，涉及多个环节，协调成本高
4. **税务稽查风险高**——税务机关对1039业务重点监控，一旦被认定虚开发票或虚假贸易，后果严重
5. **退税路径不直接**——1039本身不退税，企业如需退税需走一般贸易
6. **各地执行标准不一**——不同试点地区在商品归类、申报要求、税务处理上存在差异

### 3.4 关于"退税"的特别澄清

**很多客户会问："1039能退税吗？"**

答案很明确：**1039模式下不能退税**。1039的价值在于给"拿不到进项发票的小商户"一条合规出口通道，而不是退税通道。

- 如果货物**有增值税专用发票**，且想退税 → 走一般贸易（0110）
- 如果货物**没有进项发票**，且想合规出口 → 走1039（免税）
- 两者是**互斥**的，同一批货物不能同时走两条路

### 4.2 总体数据（估算）

- 全国1039出口额从2014年的约100亿美元增长到2024年的约3000亿美元以上
- 参与备案商户超过100万家
- 覆盖商品种类超过100万种
- 出口目的地覆盖200多个国家和地区

### 4.3 "形势好了起来"的原因

1. **试点范围不断扩大**——从1个到30多个，更多市场纳入
2. **政策持续优化**——简化申报、异地组货、跨关区通关等便利化措施陆续出台
3. **平台化运作成熟**——各试点市场都建立了成熟的综合管理平台
4. **银行、税务配合度提高**——配套金融服务和税务服务逐步完善
5. **跨境电商协同**——1039与跨境电商（9610/9710/9810）互补，形成完整的小商品出口体系

## 五、江苏省试点情况

### 5.1 试点市场

江苏省目前有以下市场采购贸易试点市场：

| 试点市场 | 所在地 | 主营品类 |
|---------|--------|---------|

| 东海水晶城 | 连云港东海 | 水晶饰品 |

### 5.2 江苏省1039出口概况

- 江苏是全国1039贸易的重要省份之一
- 江苏省商务厅牵头推进，建立了"省-市-县"三级联动机制
- 江苏省税务局出台了具体的市场采购贸易免税管理办法
- 南京海关在试点区域设立了专门监管场所

### 5.3 江苏省的特色做法

- **"市采通"模式**（源于常熟，现已全省推广）
- **异地组货**政策——省内非试点地区的货物可运至试点市场组货出口
- **海关+商务+税务+外汇四部门联合监管**机制
- **拼箱、报关、物流一站式服务**

## 六、苏州市试点情况

### 6.1 苏州概况

苏州是江苏省外贸第一大市，2024年进出口总额超过2.5万亿元。苏州的市场采购贸易主要集中在**常熟市**（常熟是苏州市代管的县级市）。

### 6.2 苏州其他区域的参与情况

- **张家港**——部分商户通过常熟平台出口
- **昆山**——部分小商品走常熟通道
- **苏州工业园区、高新区**——以一般贸易和跨境电商为主，1039业务量有限

### 6.3 苏州市的支持政策

苏州市商务局、苏州海关等部门联合出台了支持市场采购贸易发展的措施，包括：
- 对通过常熟平台出口的企业给予物流补贴
- 简化异地组货通关流程
- 出口信用保险支持
- 支持"市采通"平台向苏州全市推广

## 七、苏州市常熟市试点情况（重点）

### 7.1 常熟服装城概况

常熟服装城是全国闻名的"中国服装服饰名城"，拥有：

- **35个专业市场**
- **3万多商户**
- **年成交额超千亿元**
- 品类聚焦：男装、女装、童装、羽绒服、皮草（虞山镇周边）、服装辅料、面料
- 是中国**最大的服装服饰批发市场之一**

常熟服装城是江苏省内重要的1039市场采购贸易方式试点市场，也是苏州范围内唯一的1039试点市场。

### 7.2 试点时间线

| 时间 | 事件 |
|------|------|
| **2015年** | 常熟服装城纳入市场采购贸易试点范围（第二批试点） |
| **2016年初** | 常熟服装城市场采购贸易综合管理平台上线 |
| **2016年底** | 首单1039出口业务完成 |
| **2017-2018年** | 试点稳步推进，出口额逐年增长 |
| **2019年9月** | 商务部等七部委联合批复第三批试点，常熟服装城获得正式批复文件 |
| **2020年** | 试点范围从服装城扩展到常熟全市符合条件的商户 |
| **2021年** | 异地组货政策落地，苏州大市范围内货物可集中到常熟出口 |
| **2022年至今** | 持续优化，出口额稳步增长；"市采通"平台成为全省标杆 |

**截至2025年底，试点已满一定年限**（2015年纳入试点至今已超过10年，2019年正式批复至今已超过6年）。

### 7.3 企业开展情况

**备案企业数量：**
- 常熟市场采购贸易备案商户超过**5,000家**
- 活跃出口企业约2,000-3,000家
- 涵盖服装服饰、纺织面料、日用百货、小家电等品类

**出口规模：**
- 年出口额约**30-50亿美元**（呈逐年增长趋势）
- 主要出口目的地：东南亚、中东、非洲、东欧、拉美
- 商品结构：纺织服装约占60%，日用百货约25%，其他15%

**企业类型分布：**
- 专业市场商户（服装城档口经营者）：约60%
- 外贸公司：约25%
- 生产型企业（兼做市场采购）：约15%

### 7.4 常熟"市采通"平台（核心竞争优势）

**什么是"市采通"？**

市采通（全称"常熟市场采购贸易综合服务平台"）是常熟在1039试点中打造的数字化公共服务平台，具有以下功能：

| 功能模块 | 说明 |
|---------|------|
| 商户备案 | 在线备案，一站式完成 |
| 商品备案 | 商品信息录入、HS编码智能推荐 |
| 组货管理 | 多商户货物拼柜组货 |
| 报关申报 | 在线生成报关单，对接海关系统 |
| 物流跟踪 | 全流程物流可视化 |
| 收汇结汇 | 对接银行系统，在线收汇 |
| 免税申报 | 税务数据的自动归集和申报 |
| 数据统计 | 出口数据的统计分析 |

**市采通的领先之处：**
- 全国最早实现"全流程线上化"的市场采购平台之一
- 被江苏省商务厅作为**"常熟模式"在全省推广**
- 实现了与**海关、税务、外汇、银行系统**的互联互通
- 支持**异地组货**功能，非本地商户也可注册使用

## 八、常熟1039备案全流程（手把手教程）

### 8.1 备案主体类型

1039模式涉及**三类主体备案**：

| 主体 | 说明 |
|------|------|
| 市场采购贸易经营者 | 在常熟服装城有摊位的商户/公司（核心主体） |
| 代理商 | 帮助无摊位企业办理1039出口的代理公司 |
| 境外采购商 | 海外买家（外商） |

### 8.2 经营者备案详细步骤

#### Step 1：准备资料

**个体工商户/企业必备材料：**
- ✅ 营业执照副本（经营范围含服装服饰类优先）
- ✅ 法人身份证正反面
- ✅ **市场摊位租赁合同**（关键材料！需常熟服装城出具）
- ✅ 银行开户许可证 / 基本户信息
- ✅ 公章、法人章
- ✅ 经办人身份证及授权委托书
- ✅ 联系方式及经营品类清单

#### Step 2：摊位落实

这是整个备案流程中**最大的难点**。没有摊位的企业可以通过以下方式解决：

| 方案 | 说明 | 成本参考 |
|------|------|---------|
| 方案A：租实体摊位 | 在常熟服装城租赁实体摊位 | 约2-10万元/年 |
| 方案B：虚拟摊位/集群注册 | 部分园区提供虚拟摊位地址 | 成本较低 |
| 方案C：委托1039代理公司 | 最常见的方案，不需要自己处理摊位 | 按单收费或年费 |

> **实操建议**：对于苏州/上海的中小微企业客户，**方案C（委托代理）**是最省心的选择。常熟本地有大量专门从事1039代理服务的公司，从备案到报关一条龙服务。

#### Step 3：平台注册

登录【常熟市场采购贸易联网信息平台】（即"市采通"平台）：

1. 注册企业账户，填写企业基本信息
2. 上传扫描件资料（营业执照、法人身份证、摊位合同等）
3. 关联摊位信息
4. 提交审核

> 审核时限：一般**3-5个工作日**

#### Step 4：多部门联动备案

平台审核通过后，系统自动同步至以下部门：
- 🏛 **常熟海关**（出口报关资格）
- 🏛 **国家税务局**（免税备案）
- 🏛 **外汇管理局**（收汇资格）
- 🏛 **市场监督管理局**

#### Step 5：领取备案号

审核通过后，企业取得**"市场采购贸易经营者代码"**，即可正式开展1039出口业务。

### 8.3 关于"没有摊位"的详细解决方案

这是给客户讲解时**最常被问到的问题**。建议如下：

**问企业："您在常熟服装城有实体摊位吗？"**

- **有摊位** → 直接备案，资料齐全，3-5个工作日完成
- **没有摊位但想自己做** → 推荐方案A（租实体摊位）或方案B（虚拟摊位），成本较高但自主可控
- **不想管摊位** → 推荐方案C，委托常熟本地的1039代理公司

**代理公司的收费模式参考：**
- 首次备案费用：2,000-5,000元
- 每票出口服务费：300-800元（含报关、组货）
- 年费模式：约5,000-15,000元/年（不限票数或限票数）

### 8.4 境外采购商备案

外商备案流程相对简化，可由经营者代为备案：

**所需材料：**
- 护照复印件 或 境外公司注册证明
- 联系方式（邮箱、电话）
- 拟采购品类说明
- 在国内的联络人信息（如有）

## 九、常熟1039收汇操作方案

### 9.1 收汇政策亮点

1039在收汇方面有**三大灵活性**：

1. **支持多币种**：美元、欧元、日元、人民币、港币等
2. **支持个人收汇**：法人/经营者个人账户可收汇（这是关键优势！）
3. **简化结汇流程**：通过备案平台合规结汇

### 9.2 合规收汇渠道（3种）

#### 渠道1：市场采购贸易联网平台对接银行（首选）

**操作流程：**
```
外商付款 → 境内合作银行（常熟分行）
       ↓
   平台核验订单真实性
       ↓
   结汇至经营者账户（公户或私户）
       ↓
   入账完成（T+1到T+3）
```

**常熟主要合作银行：**
| 银行 | 说明 |
|------|------|
| 中国银行常熟分行 | 外汇业务最成熟 |
| 建设银行常熟分行 | 支持线上结汇 |
| 工商银行常熟分行 | 服务网点多 |
| 招商银行常熟分行 | 线上体验好 |

#### 渠道2：第三方持牌支付机构（适合电商型客户）

| 平台 | 费率 | 特点 |
|------|------|------|
| 连连支付（LianLian Pay） | 0.3%-0.8% | 支持小额多笔 |
| PingPong | 0.3%-0.8% | 跨境电商常用 |
| 万里汇（WorldFirst） | 0.3%-0.6% | 蚂蚁集团旗下 |
| Airwallex（空中云汇） | 0.3%-0.8% | 全球多币种 |

> **优势**：手续费低，适合碎片化订单，支持多平台收款

#### 渠道3：跨境人民币结算

外商直接以人民币支付，避免汇率损失。
**特别适合**：东南亚、俄罗斯、中东等地区客户。

### 9.3 收汇操作要点

| 要点 | 说明 |
|------|------|
| 订单匹配 | 收汇金额需与平台报关订单匹配（允许±10%误差） |
| 收汇时限 | 建议出口后**90天内**完成收汇 |
| 留存凭证 | 保留合同、发票、物流单据3-5年 |
| 避免分拆 | 不可故意分拆大额订单规避监管 |
| **远离地下钱庄** | 严禁通过非法渠道收汇（刑事风险极高） |

### 9.4 个人账户收汇注意事项

虽然1039允许个人账户收汇，但需注意：

- ✅ 必须是**已备案经营者本人账户**
- ✅ 单笔单日金额需符合个人结汇额度
- ⚠️ 大额频繁收汇仍可能被银行风控
- ⚠️ **建议提前与银行沟通**，办理"市场采购贸易专户"
- ⚠️ 2024年后监管趋严，收汇务必合规

## 十、退税问题深度解析

### 10.1 核心结论：1039不能退税

**这是1039最重要的知识点，也是客户最常问的问题。**

国家政策明确规定：
> 市场采购贸易方式出口的货物，实行增值税**免征不退**政策。

| 场景 | 能否退税 | 
|------|---------|
| 货物走1039申报出口 | ❌ **不能退税** |
| 企业有增值税专用发票 | ✅ **应该走一般贸易（0110）退税** |
| 企业没有进项发票 | ✅ **走1039最合适（免税）** |

### 10.2 为什么不能退税？

政策逻辑：

1. 1039商户多为小商户，采购货物时**没有增值税专用发票**
2. 没有进项税额 → **无税可退**
3. 国家给予**免征出口环节增值税**作为补偿
4. 简化征管，降低中小商户合规成本

**一句话总结：** 1039设计出来就是给"没票的货"合规出口用的，不是给你退税用的。

### 10.3 1039 vs 0110 全面对比

| 对比项 | 1039 市场采购 | 0110 一般贸易 |
|--------|-------------|-------------|
| 是否退税 | ❌ 免税不退 | ✅ 可退税 |
| 退税率 | 无 | 服装类约13% |
| 需要增值税发票 | ❌ 不需要 | ✅ 必须 |
| 适合货源 | 无票/低价货 | 有正规发票 |
| 单票金额限制 | ≤15万美元 | 无限制 |
| 综合税务成本 | 较低 | 中等（需垫资） |
| 操作复杂度 | 中等（需平台备案） | 较高（正规单证要求） |
| 适合企业类型 | 小商户、个体户 | 外贸公司、生产企业 |

### 10.4 什么情况选1039？什么情况选0110？

#### 选1039（免税）更划算的情况：
- 货源没有增值税发票（市场采购）
- 货物利润率较低
- 小批量、多批次、杂货拼柜
- 客户接受较低报价
- 不想垫付增值税资金

#### 选0110（退税）更划算的情况：
- 有正规增值税专用发票
- 单票金额大（>15万美元）
- 退税率高（如服装13%）
- 有充足现金流垫税
- 大客户、长期订单

### 10.5 "曲线退税"风险警示

**以下操作都是违法的，2023-2024年税务总局重点稽查方向：**

| 错误操作 | 风险 |
|---------|------|
| 同一批货物先走1039出口，再补办0110退税 | ❌ **骗税**，违法 |
| 用别人的增值税发票虚抵退税 | ❌ **虚开发票罪**，刑事责任 |
| 与外贸公司"买单出口"再退税 | ❌ **严重违规**，已被严打 |
| 1039出口后做"假报关单"退税 | ❌ **骗取出口退税罪** |

> **税务总局2023-2024年重点稽查方向：**
> - 1039与0110混合操作
> - 异常收汇与退税匹配
> - 跨境电商与市场采购数据互查

### 10.6 税务合规建议

1. **如实选择监管方式**——能退税的走0110，没票的走1039
2. **账实一致**——保留完整的采购、销售、物流、收汇凭证
3. **避免混用**——同一批货物不要在不同模式间切换
4. **合规收汇**——收汇渠道与申报匹配
5. **定期培训**——关注税务总局、海关总署最新政策
6. **留存资料**——合同、发票、物流单据保留3-5年备查

## 十一、海关环节风险点及应对

### 11.1 常见风险点

| 风险点 | 具体表现 | 后果 |
|--------|---------|------|
| HS编码归类错误 | 商品归类不准确，特别是服装类商品的HS编码细分复杂 | 影响退税（如走0110）或导致查验率升高 |
| 申报品名与实际不符 | 为了简化申报故意归并错误品类 | 海关查验判定为申报不实，面临行政处罚 |
| 侵权商品出口 | 假冒品牌、仿牌商品出口 | 扣货+罚款+列入黑名单 |
| 单证不齐 | 缺少规范的装箱单、发票、合同 | 通关延误或退关 |
| 商品超范围 | 出口禁限类商品（如管制物项、两用物项） | 行政处罚+刑事追责 |

### 11.2 易出错点

1. **单价填报不合理**——过低或过高的单价都容易引起海关关注
2. **品名描述不规范**——不能笼统写"服装"，要写"棉质男式T恤"等具体描述
3. **数量与重量不匹配**——明显不合逻辑的申报数据
4. **拼柜商品归类混乱**——多个商户的货物拼柜，品名/HS码不统一

### 11.3 应对措施

| 应对措施 | 具体做法 |
|---------|---------|
| ① 使用商品预归类 | 通过"市采通"平台的HS编码智能推荐功能 |
| ② 聘请专业报关行 | 常熟本地有大量经验丰富的报关行 |
| ③ 保留采购凭证 | 留存市场采购的原始单据备查 |
| ④ 控制申报金额 | 单价应符合市场行情，避免高报低报 |
| ⑤ 了解禁限目录 | 定期更新海关进出口税则，避开敏感商品 |

## 十二、外汇环节风险点及应对

### 12.1 常见风险点

| 风险点 | 具体表现 | 后果 |
|--------|---------|------|
| 地下钱庄收汇 | 通过非法渠道接收外汇，不通过银行系统 | **刑事犯罪**（非法买卖外汇罪/洗钱罪） |
| 收汇与申报不匹配 | 收汇金额、时间与实际出口不符 | 银行风控冻结+外汇局处罚 |
| 分拆逃汇 | 将大额收汇分拆成多笔小额规避监管 | 外汇局列为重点监管对象 |
| 不按时收汇 | 出口后长期未收汇 | 税务/外汇部门核查风险 |
| 虚假贸易背景 | 没有真实贸易的单向资金流 | 外汇局认定为虚假贸易 |

### 12.2 易出错点

1. **收汇金额远超申报金额**——误差超过±10%即需说明
2. **收汇方与合同方不一致**——第三方付款未留凭证
3. **收汇时间跨度太大**——出口后超过1年未收汇
4. **频繁的大额个人账户收汇**——触发银行反洗钱风控系统

### 12.3 应对措施

| 应对措施 | 具体做法 |
|---------|---------|
| ① 优先走平台对接银行 | 最合规的收汇方式 |
| ② 控制收汇匹配度 | 金额误差控制在±10%以内，确保订单关联 |
| ③ 及时收汇 | 出口后**90天内**完成收汇 |
| ④ 保留真实交易凭证 | 合同、发票、聊天记录、物流单据 |
| ⑤ 开设市场采购贸易专户 | 提前与银行沟通，开立专门的外汇账户 |
| ⑥ 警惕"地下钱庄"话术 | 任何"汇率好、没手续费、不走银行"的渠道都是违法 |

> **特别警示**：2024年后外汇管理趋严，务必通过合规渠道收汇。一旦被认定为地下钱庄，不仅面临外汇局行政处罚，还可能涉及洗钱等刑事指控。

## 十三、总结与建议

### 13.1 核心结论

1. **1039是合规出口通道，不是退税工具**——它的价值在于让"无票小商品"合规出口，替代买单出口
2. **常熟试点成熟**——常熟服装城1039试点已运作多年，"市采通"平台是全国标杆
3. **备案有门槛，但可解决**——摊位问题是主要门槛，可通过租赁实物摊位、虚拟摊位或委托代理解决
4. **收汇便利但需合规**——支持个人账户收汇、多币种收汇，但严禁地下钱庄
5. **1039和0110要分开走**——有票走0110退税，无票走1039免税，不可混用

### 13.2 综合推荐方案

#### 中小商户推荐路径：
```
有摊位/有市场货源 → 备案1039 → 通过平台对接银行或持牌支付机构收汇 → 享受免税
                                          ↓
                              无需退税烦恼，资金周转快
```

#### 较大企业推荐路径：
```
有进出口权 + 增值税发票 → 走0110一般贸易 → 申请退税（服装类约13%）
                                        ↓
                        若部分货物无票 → 同时备案1039分流
```

#### 跨境电商推荐路径：
```
B2C小包：9610（可退税）
B2B大货：9710/9810（可退税）
拼柜杂货：1039（免税）
```
