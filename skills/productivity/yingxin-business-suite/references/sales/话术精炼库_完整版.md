## 一、获客破冰话术

### 1.1 开场白4目的
1. 吸引注意力
2. 建立融洽关系
3. 介绍公司/服务
4. 打动客户（利益+专业）

### 1.2 电销开场白框架

**电话模板一（针对新公司）：**
```
"喂，是××公司的张总吧？您公司刚办好，
税务登记了没有？营业执照下来30天内要做税务登记，
超期会有罚款，您知道吗？"
```

**电话模板二（针对老公司）：**
```
"我是本地财税公司的，今天打电话主要是提醒您
税务申报的事情。您现在账务怎么做的？
我们一年××元就能帮您做全套。"
```

### 1.3 微信破冰话术

```
"张总您好，我是苏州盈信的企业服务顾问××。
以后您公司财税方面有任何问题都可以问我。
先送您一份见面礼——财税政策解读汇总，
希望对您有帮助。
日后再用我的专业去加深信任，成交水到渠成。"
```

## 二、需求挖掘话术

### 2.1 现状-问题-影响-引导 四步法

**第一步：了解现状**
- "执照下来了吗？银行开户了吗？"
- "有没有会计？专职还是兼职？"
- "现在的代账公司合作的价格多少？"
- "有自己的账目吗？数据清晰吗？"

**第二步：挖掘问题**
- "现在的会计有没有正常报税您知道吗？"
- "有问题的时候能及时回复您吗？"
- "上次核查账目是什么时候？"

**第三步：放大影响**
- "如果漏报了某个税种，金税系统会自动预警……"
- "如果票据丢失，以后查账拿不出来……"

**第四步：引导需求**
- "我们这边是这样帮客户解决的……"

### 2.2 四类性格调整沟通

| 类型 | 性格特征 | 沟通禁忌 | 沟通要点 |
|------|----------|---------|---------|
| 老虎型 | 果断、急 | 别绕弯子 | "我直接说结论" |
| 孔雀型 | 健谈、爱面子 | 别打断他 | "您讲得太对了" |
| 鸽子型 | 犹豫、怕麻烦 | 别逼太紧 | "您放心我来安排" |
| 猫头鹰型 | 谨慎、重数据 | 别吹牛 | "这是数据您可以看" |

## 三、常见异议处理（30个场景完整话术）

### 场景1：准备自己注册公司

```
"看来您对工商注册挺熟悉的。不过注册全流程挺费时间——
银行开户后还要做税种认定、三方协议、票种核定、
实名认证、会计制度备案等等。

我们注册全流程也就200块，效率高速度快，
能让公司尽快开展业务。"
```

### 场景2：现在零申报

```
"零申报虽然简单，但每个季度都要报，
年底还有企业所得税汇算清缴。
如果哪次逾期了，罚款500-2000起步。
而且会计制度备案、账册设置这些手续，
没有专业知识也容易出错。"
```

### 场景3：还没运营不需要

```
"公司注册后30天内必须税务登记，
否则直接拉入异常名录。
而且无论是否运营，每个月都要申报。
不申报=逾期=罚款+信用扣分。"
```

### 场景4：有兼职/朋友在记账（核心场景）

```
"万一她给您做错账，税务查上门，
您找朋友赔也不好意思，不赔自己吃亏。
所有风险只能自己扛。

而且个人兼职不受法律保护，
《整治清理会计人员非法代理记账》的通知
明确代账机构必须依法取得资质。

我们同样两三百一个月的服务，
比个人兼职更专业、更规范。"
```

### 场景5：同行比你们便宜（核心场景）

**话术A（温和版）：**
```
"咱们比价的同时，别忘了对比专业和服务。
同样的配置，服务也可能不一样——
做合规的能力、稽查应对的能力、从业经验。
多花5分钟了解，比未来花无数时间为便宜买单划算。"
```

**话术B（强势版——来源：洽谈沟通技术.docx）：**
```
"我相信市场上我们不是最便宜的，也不是最贵的。
如果你只关心价格，那我给你提个醒——
代理记账同质化严重，90%的公司看起来差不多。
但合作之后，做合规的能力、处理的稽查案例、
甚至人脉关系，就完全不一样了。

我承认我们不是最便宜的，
但如果你要对比价格，那一定要对比会计的成本和配套系统。"
```

**话术C（导购式对比）：**
```
"他们300元一个月是基础代账，
会计每人负责六七十家，精细度有限。
我们的合规代账，会计只负责二十家，
关注账务细节、数据匹配性、申报逻辑性。

就像开车——平直路上新老司机一样开，
遇到稽查这种'路况不好'的时候，差别就出来了。"
```

### 场景6：网上说代账不专业

```
"所有行业都有负面消息，尤其代理记账，
干掉了很多人的饭碗。
谁说坏话最多？两种人——
一是个人专职会计（很多只会基础做账报税）
二是专家大咖（本来靠咨询吃饭）

那些说代账不专业的，可以看看我们的资质——
高级会计师、TSC五级评价、上千家客户。
有这些做不了假的东西在，谁专业谁不专业一目了然。"
```

### 场景7：只要最低价

```
"您花这个钱无非买个省事放心。
正因为您对财税不太了解才找我们。
如果只为价格，有人能做更低——
但账务质量、税务合规呢？

我给您找几个低价营销出问题的案例，
您看看再决定。"
```

### 场景8：需要和股东商量

```
"当然可以。我相信几位股东和您商议后一样会选择我们。
大家都想找服务好、省事、放心的公司。
要不咱们见个面，我现场给股东们解答清楚？"
```

### 场景9：我先给一个月钱试试

```
"我们合作这么多客户都是整年签的。
如果服务不好，老客户早跑了。
要不先签一年，有任何不满意全额退款。"
```

### 场景10：某某说你们不行

```
"同行竞争说两句很正常。
您不如亲自来我们公司看看，感受一下。
我们TSC五级满分的评价做不了假，
高级会计师证书做不了假，
上千家老客户的选择也做不了假。

事实胜于雄辩。"
```

## 四、报价与逼单话术

### 4.1 三明治报价法

```
第一步（价值铺垫）：
"王总，我先给您介绍一下我们的服务标准——
代账部、咨询部、合规稽查部如何分工……
第二步（报价）：
"基于您的需求，全年的服务费是××元。"
第三步（强化价值）：
"这个价格包含的一对一会计、合规审核、
税务风险预警……行业内同等配置至少要××。"
```

### 4.2 逼单7法

**1. 从众成交法**
```
"我们合作的几千家客户里，您这个行业就有100多家。
大家都认可我们，说明专业度是靠得住的。"
```

**2. 里应外合法**
```
（提前安排同事配合）
"我看到群里说这个月的优惠名额只剩3个了，
您再犹豫可能就错过了。"
```

**3. 二选一法**
```
"您觉得A方案适合您，还是B方案？
（而不是问'您考虑好了吗'）"
```

**4. 限制法**
```
"这个价格只保留到今天下午6点，
过了这个时间就要按新报价走了。"
```

**5. 案例驱动法**
```
"我给您看看和您情况类似的王总，
他之前也在纠结价格，后来选择了我们。
结果前不久税务巡查，我们的账一次通过，
他隔壁公司被罚了××元……"
```

**6. 稀缺性法**
```
"我们会计部每个会计最多服务20家合规客户，
目前名额有限。您要是确定的话，
我先帮您锁定一个名额。"
```

**7. 价值清单法**
```
"我花5分钟给您列一下——
找我们做账 vs 自己做账的10个差别。
（逐条对比后）您看，这才是真正的价值所在。"
```

### 4.3 成交前最后一步

```
"王总，您能告诉我选择我们而不是其他公司的
三个理由吗？……好，这三个理由我保证不辜负。
那我们现在就签？"
```

## 五、客户跟进话术

### 5.1 未成交A类客户

**原则：快、准、狠**

**电话跟进节奏：**
- 第1天：报价后梳理，解决疑虑
- 第3天：利用报税节点再提醒
- 第7天：发新的优惠/政策变化信息
- 第14天：最后一次真诚跟进

**跟进话术：**
```
"张总，最近怎么样？您公司税务方面还顺利吗？
上次聊完之后，我这边又整理了一个您行业的相关政策，
发给您参考。不管您最终选不选我们，
这些信息对您应该都有用。"
```

### 5.2 朋友圈经营

**发圈节奏建议：**
- 周一：客户案例/签单喜报
- 周三：政策解读/风险提示
- 周五：专业干货/公司动态
- 周末：生活化内容（拉近距离）
