## 二、超额利润激励

### 含义

超额利润激励是指公司设定目标利润，实际利润超过目标的**增量部分**，按约定比例分配给激励对象。

### 实操流程

```
第一步：设定目标利润
  └─ 可选依据：净利润、回款额、ROE（推荐净利润）
  └─ 净利润为真实收益，确保只有公司真正盈利才有激励

第二步：确定超额分配比例
  └─ 建议设阶梯：超额10%以内分30%，10-30%分40%，30%以上分50%
  └─ 数据需根据公司实际情况测算后反推设计

第三步：年度核算+发放
```

### 关键注意事项

| 优点 | 缺点 |
|------|------|
| ✅ 增量分配，不伤原有股东利润 | ⚠️ 可能诱发短期行为（牺牲长期利益冲短期业绩） |
| ✅ 员工有直接获得感（算得出自己的奖励） | ⚠️ 目标设定争议大（定太低=白送，定太高=无激励） |
| ✅ 不涉及股权变更，操作简单 | ⚠️ 核心人才没有"股权绑定"，离职率不降 |

### 超额利润激励表示例

| 年度 | 营业收入 | 目标利润 | 实际利润 | 超额利润 | 激励比例 | 激励金额 |
|------|---------|---------|---------|---------|---------|---------|
| 2025 | 1000万 | 100万 | 150万 | 50万 | 40% | 20万 |

## 三、分红权激励

### 含义

分红权是股权概念中的子权利（不包含投票权/表决权/继承权），激励对象按授予份额，在每年特定时期享有公司分红收益。

### 三种实操方式

| 方式 | 说明 | 适用 |
|------|------|------|
| ① 直接分红 | 激励对象以虚拟份额直接参与分红 | 员工数少、结构简单 |
| ② 穿透分红（代持） | 股东甲代持激励份额，先接公司分红→再分给激励对象 | 不想让员工出现在工商登记 |
| ③ 递延支付 | 不一次性全额支付，分阶段支付（如分3年） | 绑定长期、减少一次性税务压力 |

### 递延支付设计

```
第一年：支付总额的50%
第二年：支付总额的30%
第三年：支付总额的20%
如中间离职：未支付部分不再发放
```

### 分红权激励协议条款示例

```text
（1）根据公司分红权激励制度，甲方（公司股东）以丙公司（激励公司）
每分红年度流水的（）%为公司年度固定可分红值，乙方（激励对象）有权
以认购的分红权份额为基础核算并获得年度分红。

（2）乙方应当遵守岗位职责，积极完成以下考核标准：
  - 截至第一年度结束，乙方负责管理区域在第一分红年度实现_____
  - 截至第三年度结束，乙方负责管理区域实现_____
```

## 四、期权激励

### 含义

公司授予激励对象在未来规定期限内，按预先约定的价格和条件购买本公司股权的权利。

关键特征：
- **权利而非义务**：激对对象可以选择行权或放弃行权
- **等待期**：从授予到可行权之间有一段时间（通常1-3年）
- **行权条件**：业绩达标或服务期满

### 实操流程

```
授予日 → 等待期（1-3年） → 行权期 → 持有/退出
  │                         │
  约定行权价格              激励对象决定是否行权
  设定业绩条件              ├─ 行权 → 缴纳认购款 → 工商登记
                             └─ 放弃 → 期权失效
```

### 关键条款

| 条款 | 建议设置 | 原因 |
|------|---------|------|
| 行权价格 | 低于外部股价（建议≤外部股价的60%-80%） | 有激励空间 |
| 等待期 | 1-3年 | 绑定绩效+防止离职 |
| 行权期长度 | 1-2年 | 足够时间决策 |
| 行权失效条件 | 离职/违纪/未达标 | 必须明确 |

### 期权合同要点

- 授予股份数量
- 授予价格和行权价格
- 行权期安排
- 若公司经济效益不好，股价低于行权价，激励对象可选择部分行权或放弃行权
- 行权后及时办理股权工商登记（直接持股或持股平台）

> "进入行权期后，若公司经济效益不好，股价低于____元/股，则激励对象可选择部分行权或放弃行权。此时，期权的授予是权利而非义务，激励对象可以基于自己的判断和公司财务情况、市场表现来决定是否行权。非特殊情况，公司不得拒绝激励对象行权。"

## 五、实股激励

### 含义

向长期为核心贡献的员工授予**工商登记的股权**。适用于经过长期考验的老员工。

⚠️ 对新员工和"空降兵"慎用实股激励。

### 两种持股方式

| 方式 | 说明 | 适用 |
|------|------|------|
| ① 直接持股 | 激励对象直接登记为工商股东，含全部股东权利 | 极少数核心层（≤3人） |
| ② 间接持股 | 通过持股平台（有限合伙）持有股权 | 激励对象较多时 |

### 身股保留权（关键条款）

激励对象在公司连续工作满一定年限（如10年）且无过错退出，承诺不从事同业竞争、遵守保密义务，则有权在退出后**一定期限内继续享受分红收益**。

**条款示例：**

```text
参与本制度并取得激励实股的激励对象，自《股权激励协议》生效之日起，
在公司连续工作满10年且无过错退出的，若承诺不从事同行业工作、
不与公司构成同业竞争（经公司股东会批准的除外），并遵守保密等义务，
且配合公司完成全部股权回购及工商变更等手续，则有权以其退出时
所持激励股权份额为依据，继续在退出后____年内享受公司分红收益。
```

### 强制回购触发条件

| 触发条件 | 价格 |
|---------|------|
| 正常离职 | 按净资产或出资额孰低回购 |
| 过错离职（违反竞业/泄密/违纪） | 1元回购或出资额原值回购 |
| 满10年无过错退出 | 净资产回购+身股保留权 |

## 六、城市合伙人激励

### 含义

与传统加盟模式不同——城市合伙人是**股权合作关系**，不是付费加盟。合伙人与总部共同设立城市子公司，获得该城市品牌独家运营权。

### 核心设计要素

| 要素 | 说明 |
|------|------|
| 出资 | 合伙人出资+总部出资，按比例持股 |
| 运营权 | 城市独家运营权，总部不再开放其他名额 |
| 开店承诺 | 合伙人承诺1-2年内开设约定数量分店 |
| 利润分配 | 按持股比例分配利润 |
| 退出 | 股权转让或公司回购 |

### 三步走战略示例

| 阶段 | 时间 | 目标 |
|------|------|------|
| 第一步：超额利润激励 | 1-3年 | 做大流水与净利润，满足高管现金流需求 |
| 第二步：期权激励 | 3-5年 | 核心高管从"打工心态"转向"合伙人心态" |
| 第三步：实股激励/城市合伙人 | 5年+ | 长期绑定+城市扩张 |

## 七、工具选择决策树

```text
Q1: 目的是什么？
 ├─ 短期激励（1-3年业绩冲刺） → 超额利润激励 or 分红权激励
 ├─ 中期绑定（3-5年核心层）   → 期权激励
 └─ 长期绑定（5年+灵魂人物）  → 实股激励
      └─ 城市拓展            → 城市合伙人

Q2: 给谁？
 ├─ 全员/销售团队 → 超额利润激励（可直接关联业绩）
 ├─ 中层管理       → 分红权激励 or 期权激励
 └─ 核心高管       → 实股激励（直接持股 or 持股平台）

Q3: 是否接受有工商变更？
 ├─ 不接受 → 超额利润激励 or 分红权激励（无工商变更）
 └─ 接受   → 期权（行权后办） or 实股 or 城市合伙人

Q4: 有多少激励对象？
 ├─ ≤3人 → 直接持股（实股）
 ├─ 3-20人 → 持股平台（有限合伙）
 └─ 20+人 → 分红权激励为主
```
