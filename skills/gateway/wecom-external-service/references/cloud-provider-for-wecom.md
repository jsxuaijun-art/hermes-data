# 企业微信机器人部署 — 云服务器选型参考

## 核心原则

**企微回调走的是公网 HTTPS，与服务器厂商无关。** 阿里云和腾讯云在企微消息投递路径上没有区别，不存在"腾讯旗下所以企微有内部通道"的情况。

## 推荐配置

对于跑 Hermes Gateway + WeCom 回调服务，最低需求：

| 维度 | 要求 | 说明 |
|:--:|:--|:--|
| CPU | 2核 | Hermes Gateway + Nginx + Python 进程 |
| 内存 | 2G | 足够跑 Gateway，无需 swap |
| 系统盘 | 40GB | 系统 + 日志，一年用不完 |
| 带宽 | 3M 固定 / 200M 峰值 | 纯文本消息+LLM token 回传，带宽需求极低 |
| 流量 | 不限 / 300GB+ | 纯文本服务流量极小，不用担心 |

## 阿里云 vs 腾讯云 对比（2025.5 价格）

| 项目 | 阿里云 轻量 | 腾讯云 轻量 |
|:--|:--|:--|
| 2核2G 新客价 | ¥68~79/年（视活动） | ¥68~99/年（视活动） |
| 带宽 | 200M 峰值版 / 3M 版 | 3M~4M |
| 流量 | 不限流量 | 300GB/月起 |

## 推荐选择

- **首选阿里云轻量 ¥68~79/年** — 同价位带宽更宽、不限流量
- 如果已有腾讯云账号且懒得注册阿里云 → 腾讯云完全可用，不影响后续操作
- 购买时地域选**上海**（华东2），离苏州最近延迟最低
- 镜像选 **Ubuntu 22.04**，不要选 24.04（兼容性更好）
- **数据盘不用加** — 系统盘40~50GB足够
- 实际购买价可能不同（如 ¥79/年 vs ¥68/年），取决于活动档位，¥79 仍是地板价

### 实战备忘：阿里云购买完成后的确认项

买完后必须在控制台做以下操作才能 SSH 连上：
1. 进入"轻量应用服务器"控制台 → 找到实例
2. **重置密码**（默认不设密码）
3. 点**重启**使密码生效
4. 拿到**公网 IP**（在实例列表可见）

```
验证连通性：
ssh root@<公网IP>
或
sshpass -p '<密码>' ssh -o StrictHostKeyChecking=no root@<公网IP> "uname -a"
```

## 域名管理平台 — 西部数码（west.cn）DNS 配置

如果域名托管在**西部数码**，其 DNS 管理界面字段名称与阿里云/腾讯云不同，容易让新手困惑。对照表：

| 西部数码字段名 | 含义 | 你的填法 |
|:--|:--|:--|
| 主机名 / 主机记录 | 子域名前缀 | 如 `callback`（实现 callback.yingxinkuaiji.com） |
| 记录类型 | 选 A | A（IPv4 地址） |
| 线路类型 | 默认/一般 | 选**默认**，不用改 |
| 记录值 | 服务器 IP | 填公网 IP，如 `47.103.27.171` |
| 优先级 / MX优先级 | 只影响邮箱记录 | **留空不填** |
| TTL | 缓存时间 | 默认 600 或更低 |

**配置示例：**

```
主机名：callback
记录类型：A
线路类型：默认
记录值：47.103.27.171
优先级：（空）
TTL：600
```

**生效确认：** 配置后等 1~10 分钟，用 `ping callback.yingxinkuaiji.com` 或 `nslookup callback.yingxinkuaiji.com` 验证解析是否指向你的服务器 IP。

## ICP备案

两家都需要备案（中国大陆服务器），和厂商无关。

### 核心事实：备案状态 ≠ 域名控制台显示状态

❗ **重要陷阱**：域名在西部数码（或其他域名平台）的控制台里可能显示"备案已删除"、"无备案信息"等，**但这不代表工信部数据库里也删了**。域名平台的"注销备案"多数只是解除域名与备案号的绑定，备案号本身可能仍以"空壳主体"或"有效主体"形式保留在工信部。

**实战案例（2026.5.12 — 苏州盈信）：**
用户称"域名以前备过案但删了"，指向了现有的备案号 **苏ICP备15030316号-1**。查询后发现：
- 备案主体：**苏州盈信企业管理有限公司**（用户自己的公司）
- 域名：yingxinkuaiji.com
- 最后更新：**2026-05-02**（上个月刚更新）
- 状态：**完全活跃，根本没有删除**

用户对"已删除"的印象来自于域名平台控制台，而非工信部数据库状态。

### 验证备案是否仍有效的方法

工信部官网（beian.miit.gov.cn）依赖 JavaScript，无法直接通过命令行工具抓取。推荐使用第三方查询平台：

| 平台 | URL | 可用性 |
|:---|:---|:---:|
| icplishi.com | https://icplishi.com/{备案号编码}/ | ✅ 无需验证码 |
| beianx.cn | https://www.beianx.cn/ | ⚠️ 偶尔需要验证码 |

**查询步骤：**
1. 拿到任何已知的备案号（如苏ICP备15030316号）
2. 构造 URL：`https://icplishi.com/苏ICP备15030316号/`（URL 编码为 `https://icplishi.com/%E8%8B%8FICP%E5%A4%8715030316%E5%8F%B7/`）
3. 用 `web_extract` 或 `curl` 获取页面，查找：
   - 备案主体名称（确认是否与当前公司一致）
   - 子备案号对应的域名
   - **更新时间**（关键！近期的表示活跃，旧的可能已注销）

### 状态分类及对应操作

| 状态 | 特征 | 操作 | 时长 |
|:---|:---|:---|:---:|
| ✅ **有效活跃** | 主体+域名对应，更新时间在近1年内 | **新增接入**（接入备案）— 在阿里云备案系统选"接入备案"，不是"首次备案" | **1~2天** |
| ⚠️ **空壳主体** | 主体存在但无网站接入信息 | 先在阿里云做"新增接入"，管局自动激活 | 2~3天 |
| ❌ **完全注销** | 查询不到任何记录 | 从零做"首次备案" | 3~7天 |

### 新增接入（接入备案）流程

当备案仍然有效时（最常见情况），操作步骤：

1. 登录阿里云备案系统 → https://beian.aliyun.com/
2. 选择 **"新增接入备案"**（不是"首次备案"！）
3. 输入现有备案号（如 苏ICP备15030316号）
4. 系统自动拉取原备案信息，补充阿里云服务器接入信息
5. **不需要幕布拍照**，线上核验即可
6. 1~2个工作日内审核通过

### 注意事项

- **接入备案期间不影响现有网站访问** — 不需要关站
- 阿里云系统会自动检测原接入商（如西部数码），如果原接入商未解除绑定，阿里云会提示"该备案号已有接入商"，需要先在原平台"取消接入"（不是注销备案号）
- 取消接入后，备案号本身仍然有效（只是少了一个接入商），阿里云即可接入
- 如果域名平台确实显示已注销，以工信部数据库查询结果为准 — 域名平台的"删除"可能只是移除DNS关联，备案号仍然留在工信部
