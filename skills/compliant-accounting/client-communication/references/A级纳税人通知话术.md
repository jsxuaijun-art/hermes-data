# A级纳税人通知话术

## 场景
客户公司被税务评为A级纳税人（最高信用等级）。借此机会：
1. 恭喜客户
2. 暗示"这是专业能力的体现"
3. 树立"不走低价路线"的品牌形象
4. 提高客户粘性

## 中和版（默认推荐）

X总，刚看到咱们公司评上A级纳税人🎉 这是税务最高信用等级——不是说营收大就能拿的，关键是申报数据逻辑性强、经得起税务标准检验。能评上，说明咱们财税管理这块在局里是被认可的。

评上之后好处不少：发票额度高、审批快、稽查概率低，后面经营会更省心。

这也是咱们长期配合的结果。我们团队一直坚持专业路线，不卷低价，帮客户把账做规范、把风险挡外面，才有这个底气。希望能一直保持下去，继续配合好💪
