### 黄金前件 ⚠️ — 所有文案必须先执行此步骤

无论写什么类型的文案，**开头3秒必须按「黄金三秒钩子」规则设计**：
- 用户平均1.7秒滑动切换，六成观众3秒内决定去留
- 平台算法以3秒完播率为核心指标
- 六大开头公式任选：数字冲击型、冲突悬念型、情感共鸣型、反常识揭秘型、利益承诺型、提问好奇型

### 视频制作手法反推工作流（Production Forensics）

当江姐要求「拆解这个视频的制作风格/工具/手法」时，执行以下流程：

**Step 1 — 定位文件 + 查元数据**
```bash
ffprobe -v quiet -print_format json -show_format -show_streams <videopath>
```
关键看：
- `encoder` / `Lavf` 版本 → 是否被转码过
- `comment` 字段（如 `vid:v0300fg...`）→ 微信视频号/腾讯视频 ID
- `vendor_id` → 原始录制设备
- `codec_name`, `profile`, `level` → 技术规格
- `bit_rate` → 压缩程度（低码率高分辨率 = 网络视频）
- `sample_fmt`, `sample_rate`（音频）→ 人声编码模式

**Step 2 — 像素级画面分析（无 vision API 替代方案）**
```bash
# 场景切换检测（了解剪辑节奏）
ffmpeg -i <videopath> -filter:v "select='gt(scene,0.1)',showinfo" -f null - 2>&1 | grep "pts_time:"
```
解读输出中的 `mean:[Y U V] stdev:[Y U V]`：
- **U/V 均值≈128 + 极低标准差** → 画面去色/黑白/文字卡（知识卡片类）
- **U/V 有变化 + 较高标准差** → 彩色画面/真人出镜/实拍
- **Y 均值** → 画面亮度（85-100=暗色底，150+=亮色底）
- 场景切换少 + 色度中性 → 纯文字卡片/知识分享类

**Step 3 — 关键帧分析**
```bash
# 统计帧类型分布
ffprobe -v quiet -select_streams v -show_entries frame=pict_type \
  <videopath> 2>&1 | grep "pict_type" | sort | uniq -c | sort -rn
```
- I帧很少（5个/29秒）→ 极少量剪辑切换，画面静态为主
- I帧很多 → 频繁画面切换/快速剪辑风格

**Step 4 — 工具链推断启发式**

| 元数据信号 | 推断 |
|-----------|------|
| `compatible_brands: isomiso2avc1mp41` + `Lavf` | 被 FFmpeg 重编码过，非原始导出 |
| `comment: vid:v0300f...` | 来自微信视频号/腾讯视频 |
| HE-AACv2 + 低码率 | 平台方压缩（抖音/视频号） |
| 无 `com.apple.*` / `com.android.*` 标签 | 原始工具信息被抹去 |
| 音频 44.1kHz + 62kbps | 人声优化 |

**Step 5 — 制作链路还原**
组合所有证据 → 填写标准输出模板：

```
🎬 视频制作手法还原报告
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
来源：微信视频号/抖音下载
分辨率：1920×1080
时长：29秒 / 2.1MB
画面类型：暗底+白字 知识卡片（去色/黑白）
场景切换：5次（约每5.8秒一切，5个知识卡片）
工具推测：剪映(CapCut) / Canva 文字模板
配音推测：TTS（剪映自带/魔音工坊）
制作流程：
  ① 文案600-800字 → ② 分5-6段为卡片
  ③ 暗底+大标题+关键词高亮 → ④ 加BGM
  ⑤ TTS配音 → ⑥ 导出1080p → ⑦ 平台压缩
可复制性：✅ 适合财税知识科普类视频
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 批量生成工作流（高优先级）

当江姐要求"批量出文案"时，执行以下精简流程：

**输入条件：** 江姐给1个业务方向 + 1个热点事件（可选）
**输出目标：** 10-15条独立文案框架，覆盖不同角度

**操作步骤：**
1. 先问有没有相关视频（见上方热点事件处理流程）
2. 确认业务方向和转化目标
3. 搜索热点事件或从素材库调取
4. 提取事件内核→商业规律→业务映射（详见行业自然流skill中的方法论）
5. 切10-15个不同角度，每条独立成文
6. 用表格给出选题速览供筛选
7. 江姐筛选后，把选中的深化为完整拍摄脚本

**10条框架技巧：** 同一个热点事件可以切出10+个角度，核心方法：
- 变主角（韩红→你→你的会计→你的公司）
- 变视角（旁观者→参与者→受害者→受益者）
- 变深度（现象→规律→本质→解决方案）
- 变场景（个人→行业→社会→历史）
- 变语气（讲故事→讲道理→讲案例→讲数据）

**深化脚本模板（拍摄脚本格式）：**

每条拍摄脚本包含：
```
📍 标题：《xxx》
时长：60-90秒
风格：直面镜头口播
BGM：轻节奏感

口播文案（带时间轴）：
【0-5s】钩子开场
【5-xxs】价值输出
【xxs-结尾】CTA转化："找江姐就对了"

分镜：时间→景别→动作
```

**结构铁律：** 热点钩子(0-5s)→专业价值(5-70s)→CTA转化(最后5-10s)
- CTA必须自然嵌入，不是硬广："找江姐""找江姐聊聊""找江姐就对了"
- 每条结尾都指向一个具体动作（注册/变更/注销/免费咨询）

**数据反馈循环：**\n- 江姐发完视频后收集：完播率 > 收藏率 > 互动率\n- 完播率高+收藏率高 → 按此模式批量出5条（上限，江姐没时间拍更多）\n- 完播率高+收藏率低 → 优化结尾CTA，加实操干货\n- 完播率低 → 换开头钩子，重做前5秒\n\n**产出规范（2026.6.27 江姐确认）：**\n- 脚本直接放桌面 `D:\\360MoveData\\Users\\Admin\\Desktop\\`，不存memory\n- Skill和方法论才存memory和上传github\n- 单次批量不超过5条，不够再补

### 抖音算法核心认知（2026年6月深度版） 🔥

#### 一、整体调整

| 维度 | 旧模式 | 新模式 |
|:--|:--|:--|
| 匹配方式 | 标签匹配（关键词/画面/话术） | 行为预测（AI大模型预判用户喜好） |
| 考核周期 | 24小时爆发 | 延长至7天，持续回收深度行为 |
| 直播考核 | 综合指标 | 15分钟赛马单元×内容/交易/合规三维 |

#### 二、数据指标权重排序（新）

```
收藏率 > 复访率 > 铁粉互动 > 5秒完播 > 整体完播 > 评论 > 点赞 > 转发
```

**核心逻辑**：算法不再均等看待所有数据。收藏=用户主动认可内容长期价值，复访=内容有持续吸引力。这两项权重远超点赞转发。

#### 三、视频六阶流量池（逐级晋升制）

```
种子池（300-500曝光）→ 兴趣池（500-5000）→ 价值池 → 扩量池 → 高热池（200万级）→ 超级池（千万级）
```

各阶段考核重点：
- **种子池**：查重、违规词筛查，合规通过才能进入下一阶段
- **兴趣池**：验证内容与用户匹配度（互动数据判定）
- **价值池**：完播率+完播时长+互动收益，计算内容权重
- **扩量池**：传播价值+商业价值双维度评估
- **高热池**：全网传播潜力监测
- **超级池**：现象级内容，全网分发

#### 四、流量权重计算公式

```
总分（满分1分）= 0.35 × VPS + 0.25 × ENG + 0.20 × RET + 0.15 × TAG + 0.05 × ACCT
```

| 符号 | 含义 | 权重 | 核心含义 |
|:--:|:--|:--:|:--|
| VPS | 内容价值得分 | 35% | **内容硬实力**——最核心权重 |
| ENG | 互动数据 | 25% | 流量热度的核心动力 |
| RET | 留存与回访 | 20% | 长期吸引力的试金石 |
| TAG | 标签精准度 | 15% | 内容与目标人群的匹配度 |
| ACCT | 账号健康分 | 5% | 基础线，达标即可 |

**晋级门槛**（从初级池进入下一池必须全部达标）：
- ✅ 有效完播率 ≥ 38%
- ✅ 互动指标 ≥ 12%
- ✅ 留存加速 ≥ 18%
- ✅ 标签纯度 ≥ 65%
- ✅ 账户健康度 ≥ 70%

**运营二八法则**：80%精力砸在 VPS（内容价值）和 ENG（互动设计）上，后三个及格线过了就别再纠结。

#### 五、直播六阶赛马机制（15分钟/考核单元）

| 层级 | 场观范围 | 核心考核 |
|:--:|:--:|:--|
| 冷启动 | 200-500 | 基础留人能力 |
| 基础验证 | 1,000-5,000 | 基础留人能力 |
| 流量加速 | 5,000-30,000 | 商业变现验证 |
| 稳定增长 | 3万-20万 | 商业变现验证 |
| 增长爆发 | 20万-100万 | 全维度精细化 |
| 超级顶流 | 100万+ | 全维度精细化 |

**三大核心考核指标**：
1. **内容能力**：三秒留人、停留时长、互动、转粉
2. **交易能力**：电商=GMV / 非电商=打赏、灯牌等付费行为
3. **合规性**：一票否决——违规直接断流

#### 六、付费投放底层逻辑

```
eCPM = eCTR（预估点击率）× eCVR（预估转化率）× 出价 × 1000
```

- **eCTR**：素材质量决定（视频封面、直播画面）
- **eCVR**：落地页承接转化能力
- **出价**：单次转化最高成本

**付费流与自然流的协同机制**：
- ✅ **正向协同**：付费引入精准用户 → 产生停留/互动/下单 → 算法提权 → 撬动免费自然流量
- ❌ **负向协同**：素材劣质/流量不精准/转化差 → 拉低账号评分 → 压制自然流量

**核心认知**：付费流与自然流已并轨运行，内容质量是唯一底座，不分付费/自然。

#### 七、六大核心规则（速查）

1. **负反馈 > 正反馈（乘法级 vs 线性累加）**
   - 点赞/收藏/评论/关注 = 线性加分
   - 负面评论/断崖跳出/举报 = 乘法级扣分
   - 保评论区情绪 > 追正向数据

2. **留存动态曲线取代完播率**
   - 旧玩法：夸张开头+灌水强行拉完播 → **已失效**
   - 新规则：算法分析留存曲线——平缓下降=优质，前期高+几秒断崖=劣质扣分

3. **评论区情绪是权重因子**
   - 满屏负面评论 = 直接扣分
   - 高净值人群给负反馈 → 平台缩推荐 → 人群标签混乱 → 变现完蛋

4. **推流双价值：留存价值 + 电商价值**
   - 留存价值：能留人、带动深层互动
   - 电商价值：帮助用户建立认知、引导搜索和交易
   - 纯娱乐/纯段子 → 只有留存价值 → 推流天花板低

5. **付费流 & 自然流并轨运行**
   - 正反馈双向放大，负反馈双向压制
   - 内容质量是唯一底座

6. **人群资产 × 二次触达窗口 15-90天**
   - 产生深度行为的用户被打上账号专属标签
   - 15-90天内可反复触达

#### 八、落地检查清单（每篇文案生成后自检）

- [ ] 开头是否避免「虚张声势的夸张悬念」？（防断崖式跳出）
- [ ] 内容是否能维持留存曲线平缓下降？（无灌水段）
- [ ] 评论区是否会引发负面情绪？（防扣分）
- [ ] 是否有电商价值或深度留存价值？（推流天花板）
- [ ] 最终能否打上精准人群标签？（而非泛流量）
- [ ] 完播率是否 ≥ 38%？（晋级门槛）
- [ ] 是否有引导收藏/复访的钩子？（最高权重指标）
- [ ] 内容是否逐层递进而非开头全抛？（保留存曲线平滑）

## 类型二：趣味破播类 (short-video-fun-breakout)

### 核心执行流程
1. **收到用户原始文案后** → 先主动询问观点/目的/故事
