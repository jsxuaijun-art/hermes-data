# 股权激励全流程 "四步十问" 设计法

> 来源：《一本书讲透股权》第十二章精华
> 用于 equity-architecture-guide 的股权激励全流程设计

---

## 一、四步框架

```text
第一步：定人（Who）——用谁激励、激励谁、怎么持股
第二步：定钱（Price）——公司估值、激励价格
第三步：定量（Quantity）——分多少、每个人分多少
第四步：定源（Source）——股权来源、出资方式、时间安排

退出机制贯穿始终
```

---

## 第一步：定"人"

### 1.1 用谁激励（确定激励主体）

| 场景 | 建议 |
|------|------|
| 单一公司经营 | 以该公司为激励主体 |
| 多家关联公司 | 优先选**劳动关系所在公司**为激励主体 |
| 多公司交叉用工 | 建议用控股公司作为激励主体（→ holding-company-firewall） |
| 拟上市主体 | 以上市主体为激励主体，通过持股平台间接持股 |

### 1.2 激励给谁（确定激励对象）

**马斯洛需求层次 × 股权激励适配表：**

| 员工层级 | 需求层次 | 匹配激励工具 | 持股方式 |
|---------|---------|-------------|---------|
| 基层员工 | 生理+安全 | 超额利润/分红权 | 虚拟份额 |
| 中层骨干 | 归属感 | 分红权/期权 | 虚拟/持股平台 |
| 核心高管 | 尊重+自我实现 | 期权/实股 | 持股平台/代持 |
| 灵魂人物 | 自我实现 | 实股（家族公司等同实控人） | 直接持股 |

**激励对象入选条件参考：**

| 维度 | 标准 |
|------|------|
| 未来创造价值 | 股权激励本质是买未来，不是奖励过去 |
| 过去贡献 | 老员工可用贡献折股（注意公平性） |
| 岗位重要性 | 核心岗位优先 |
| 职业匹配度 | 文化认同+长期发展意愿 |
| 年限 | 建议满1-2年正式员工 |

**排除条件：**
- 近3年有严重违纪/重大过失
- 竞业限制期内
- 有公司法规定的不得担任董事/高管情形的

### 1.3 如何持股（三种方式对比）

| 持股方式 | 说明 | 适用 | 优缺点 |
|---------|------|------|--------|
| ① 直接持股 | 登记为工商股东 | 极少数核心层（≤3人） | ✅ 权利完整；❌ 股东多决策慢 |
| ② 间接持股（有限合伙） | 通过有限合伙平台持股，GP控制决策 | 3-20人的核心层 | ✅ GP即使1%也能控制；❌ 管理成本略高 |
| ③ 代持持股 | 由大股东代持 | 临时安排/人数少 | ✅ 灵活；❌ 法律风险（名义股东擅自处分） |

> **推荐路径：** 核心层直接持股（如果人数极少）→ 多数通过有限合伙平台间接持股
> 代持持股仅作为过渡安排，后续应还原或转入持股平台

## 第二步：定"钱"

### 2.1 公司估值方法

| 方法 | 计算方式 | 适用 |
|------|---------|------|
| 净资产法 | 净资产×溢价倍数 | 重资产公司 |
| 市盈率法（PE） | 年净利润×PE倍数（行业经验值） | 有稳定盈利的公司 |
| 市销率法（PS） | 年营收×PS倍数 | 互联网/高增长但未盈利公司 |
| 现金流折现（DCF） | 未来现金流折现 | 成熟期公司 |

**非上市公司的"玄学估值"：**
> "公司的估值不仅包含当前的净资产、净利润，还涵盖创始人团队自企业成立以来投入的时间与心血，以及公司未来的潜力和发展空间。所以，估值需要'喊出来'，需要充分展示并让投资/激励对象相信，企业未来能成长到什么样的规模。"

### 2.2 外部股价 vs 内部激励股价

| 项目 | 计算方法 | 说明 |
|------|---------|------|
| 外部股价 | 市盈率法：公司估值÷总股本 | 投资方入股价 |
| 内部激励股价 | 外部股价×内部折扣率（通常60%-80%） | 员工认购价 |

**示例：**
```
A公司：注册资本1000万，年净利润300万
外部估值：300万×10倍PE=3000万 → 外部股价3元/股
内部激励价：3元×60%=1.8元/股（员工认购价）
```

### 2.3 兜底承诺

实操中有实际控制人对员工做出兜底承诺：
- 员工认购资金的本金兜底补足
- 股票升值收益归员工
- 下跌损失由实控人承担

⚠️ **税务风险提示：** 兜底承诺的支出不能税前扣除，且实控人承担无限兜底义务。

## 第三步：定"量"

### 3.1 激励总量建议

| 公司阶段 | 激励总量 | 说明 |
|---------|---------|------|
| 初创期（0-3年） | 10%-20% | 预留足够空间吸引核心人才 |
| 成长期（3-7年） | 10%-15% | 分批释放，配合业绩增长 |
| 成熟期（7年+） | 5%-10% | 存量调整+增量补充 |
| 拟上市 | ≤10% | 上市后股权激励受严格监管 |

### 3.2 分配方法一：分类比例推定

```
假设：激励总份额=10%
      5位核心高管（实股）→ 占总激励份额60%
      15位中层（分红权）→ 占总激励份额40%

核心高管5人分配：
  总经理         → 20%（占总激励份额的20% = 总股本的2%）
  副总经理(2人)  → 各12.5%（各1.25%）
  部门总监(2人)  → 各7.5%（各0.75%）
```

### 3.3 分配方法二：岗位价值评估法

**评估维度（每项0-10分）：**

| 维度 | 权重 | 说明 |
|------|------|------|
| 工龄 | 20% | 满1年1分，10年封顶 |
| 岗位职级 | 30% | 骨干2分→主管4分→经理6分→总监9分→副总10分 |
| 业绩贡献 | 30% | 近3年业绩表现 |
| 不可替代性 | 20% | 离职后影响程度 |

**计算：**
```text
个人得分 = Σ(维度得分×权重)
个人占比 = 个人得分 ÷ 激励对象总得分
个人获授股数 = 激励总量 × 个人占比
```

### 分配注意事项

| 要点 | 说明 |
|------|------|
| 差距不宜过大 | 最高和最低建议≤5倍，过大则内部不平衡 |
| 预留调整空间 | 预留10%-20%给未来新员工 |
| 明确授予条件 | 考核期内必须达标 |
| 避免"平均主义" | 大锅饭=没激励 |

## 第四步：定"源"

### 4.1 股权来源

| 来源方式 | 操作 | 适用 |
|---------|------|------|
| 大股东转让 | 大股东直接转让部分股权给激励对象 | 股权结构简单 |
| 增资扩股 | 公司增加注册资本，激励对象认购 | 需要同时融资 |
| 预留股权 | 设立时预留（大股东代持） | 有上市规划 |
| 持股平台 | 有限合伙预留份额 | 多人激励 |

**实操建议：**

```text
方式① 大股东预留：在公司设立或调整时，大股东先认缴较多股权，
         后面再转让给激励对象。
         → 适合：股权结构简单，≤5位激励对象

方式② 设立有限合伙：成立一家有限合伙企业作为持股平台，
         平台持有公司股权，激励对象作为LP。
         → 适合：多位激励对象，需保持决策集中

方式③ 股东协议预留：股东协议约定预留X%的股权，
         三年内用于员工股权激励。
         → 适合：股权结构已定，不方便立即变更
```

### 4.2 出资方式

| 方式 | 操作 | 注意 |
|------|------|------|
| 自有资金 | 激励对象自掏腰包 | 效果最好，有真金白银投入 |
| 借款出资 | 实控人借款给员工 | 签借款协议，与在职状态挂钩 |
| 延期支付 | 从未来分红中扣除 | 自循环，降低员工出资压力 |
| 赠与 | 实控人赠送股权 | 税务问题（赠与个税） |

**借款出资条款示例：**

```text
双方签署书面的《借款协议》，明确约定借款期限、还款计划，
且还款计划应与员工的在职状态挂钩（离职前一次性还清）。
```

### 4.3 时间安排

| 要素 | 建议设置 | 说明 |
|------|---------|------|
| 有效期 | 3-5年（分红权）/ 长期（实股） | 分类型设定 |
| 授予日 | 方案批准日 | 明确起点 |
| 等待期/锁定期 | 3-5年 | 期间不可转让/退出 |
| 行权期 | 1-2年 | 期权激励专用 |
| 分红窗口期 | 每年4月 | 财务数据整理完成后 |
| 考核周期 | 年度 | 公司层面+个人层面 |

**考核机制双层设计：**

```
公司层面考核（如净利润达标≥90%）
    └─ 达标 → 进入个人层面考核
         ├─ 个人达标率≥90% → 100%分红
         ├─ 70%≤个人达标率<90% → 按比例发放
         └─ 个人达标率<70% → 不分红
```

## 退出机制

### 退出情形与价格

| 退出情形 | 退出价格 | 说明 |
|---------|---------|------|
| 正常离职（满服务期） | 按净资产或出资额孰高 | 奖励长期贡献 |
| 正常离职（未满服务期） | 按出资额或孰低回购 | 本金保障但不盈利 |
| 过错离职 | 1元回购 或 出资额原值 | 违约惩罚 |
| 死亡/丧失劳动能力 | 按净资产回购 | 人道处理 |
| 公司上市 | 按上市规则 | 锁定+解禁 |

### 退出程序

```
发生退出情形 → 书面通知 → 价格核算 → 协议签署
  → 回购款支付（分期或一次性） → 工商变更/平台调整
```

### 勤勉尽责清单（激励制度应包含的内容）

```
□ 激励对象入选条件与排除条件
□ 不同激励工具的适用场景与选择逻辑
□ 授予价格与行权价格确定机制
□ 考核机制（公司层+个人层）
□ 有效期、授予日、等待期、锁定期
□ 分红窗口期与分红方式
□ 退出情形与退出价格
□ 尽职调查流程（公司财务数据+员工背景）
□ 协议签署与工商变更流程
□ 制度管理与动态调整机制
```

## 客户沟通话术

### 初次聊股权激励

> "老板，你刚才说想给核心员工分点股份——这是个好想法，也是我见过最容易出问题的动作。我给你讲讲我的一些实操经验。股权激励不是说把股份送出去就完了，最怕的是分完以后反而把关系搞僵了。核心要解决四个问题：分给谁、分什么、分多少、钱从哪里来。每个都想清楚再做。我们帮很多企业设计过方案，要不要先给你理一下你的情况？"

### 聊到工具选择

> "股权激励有五种常用工具：超额利润激励、分红权、期权、实股、城市合伙人。它们的区别主要在于'绑的程度'不一样。超额利润激励效果最快——今年业绩好今年就分钱，但不绑定长期；实股激励最彻底——直接给工商登记的股份，但给了就不好收回来。所以一般建议：先用超额利润或分红权做一年，大家感受到股权激励的好处了，再从里面挑最优秀的人给期权或者实股。"

### 聊到员工出资

> "要不要让员工出钱买？我的建议是：一定要出。哪怕你借给他也要出。为什么？不出钱的东西没人珍惜。你白送给员工股份，他不会当回事，哪天觉得公司不好了拍拍屁股就走了。交过钱的员工，心态是不一样的——他从'打工'变成了'我的公司也要赚钱'。这叫真金白银的绑定。"
