## 📍 脚本 #4：《注销比注册难十倍》

**时长：** 65-80秒 | **风格：** 揭秘警示型

**口播文案：**

```
【0-5s】韩红走个面儿被骂了三天。
但你知道公司注销比注册难多少倍吗？

【5-15s】很多老板觉得注册公司难，注销简单。
我告诉你——错了。
注销比注册难十倍，这是行业共识。

【15-30s】为什么？因为你注册的时候，
税务、工商、银行全是零状态。
但注销的时候——欠的税要补、该报的表要报、
所有的票据要清，少一样都走不了。

【30-45s】我见过最惨的一个老板，
公司不经营了扔在那不管。
三年后想再注册新公司，发现被拉黑了。
想注销？先处理三年的税务问题。
补税罚款加滞纳金，比注册一个新公司贵得多。

【45-60s】所以不经营的公司，赶紧注销。
别等麻烦找上门了才着急。

【60-75s】在苏州注册、变更、注销，
找江姐帮你一次搞定。
不留后患，不花冤枉钱。
```

**分镜：** 00s中景→15s摇头→30s手指数→45s故事表情→60s直视镜头→70s字幕

## 📍 脚本 #5：《变更公司，别自己瞎折腾》

**时长：** 60-75秒 | **风格：** 实用干货型

**口播文案：**

```
【0-5s】韩红的面子能走一次两次。
但公司变更，可不是走一次就完的事。

【5-15s】公司经营过程中，变更是家常便饭——
法人变了、经营范围变了、
注册资本变了、股东变了。
听起来简单吧？

【15-30s】但你知不知道——
法人变了，税务要不要同步？
银行要不要同步？合同要不要重签？
许可证要不要重办？
很多人以为变一个地方就行，
漏了一个，后面全是窟窿。

【30-45s】这就是为什么我不建议你自己跑变更。
不是说你办不了，是你不知道哪些要同步。
专业的，一次到位。
不专业的，改了白改。

【45-60s】我们在苏州做了16年，
每一个变更的流程、每一个要同步的部门、
每一份要签的文件，都清清楚楚。
你只负责签字，剩下的我来。

【60-70s】公司要变更？找江姐，一个电话搞定。
```

**分镜：** 00s中景→10s摊手→25s手指列举→40s对比语气→55s自信微笑→65s字幕

## 📍 脚本 #6：《你的经营范围和实际业务对得上吗？》

**时长：** 60-75秒 | **风格：** 提问破局型

**口播文案：**

```
【0-5s】韩红去冯小刚的场子走面儿。
但如果走错了场子呢？尴尬不尴尬？

【5-15s】很多老板注册公司的时候，
经营范围随便写。
结果实际业务和执照上写的，完全对不上。
这叫什么？超范围经营。

【15-30s】超范围经营的后果是什么？
轻的，工商警告。
重的，税务直接认定你偷税漏税。
你开的是餐饮店，执照上写"销售服装"，
税务局一查——你的票怎么全是餐饮？
说不清楚。

【30-45s】所以经营范围一定要写准、写全。
多写没关系，大不了不做。
少写，就是给自己挖坑。

【45-60s】我们帮客户注册第一步就是梳理经营范围。
你做什么业务、未来可能做什么，
全列进去，一步到位。

【60-70s】不懂经营范围怎么写的？找江姐聊聊。
免费帮你规划，避免超范围经营风险。
```

**分镜：** 00s中景→10s皱眉→25s手势加重→40s轻松→50s自信→65s字幕

## 📍 脚本 #7：《个体户vs有限公司，选错悔三年》

**时长：** 70-85秒 | **风格：** 对比分析型

**口播文案：**

```
【0-5s】韩红选的是走谁的面儿。
你选的是——走哪条路经商。

【5-15s】很多老板刚开始创业，
第一个问题就是：
我注册个体户好，还是有限公司好？
选错了，后面想改，成本高得吓人。

【15-30s】我简单给你讲清楚——
个体户：税低，但无限责任。
出了事，拿你个人财产赔。
有限公司：税高一点，但有限责任。
出了事，只赔公司资产，不牵连个人。

【30-45s】你做小本生意、风险低、不融资，
个体户够了。
你想做大、有合伙人、要融资、要招投标，
必须有限公司。
选错了，想从个体户转公司？
相当于注销重来，几个月的时间加几千块成本。

【45-60s】所以注册之前，先想清楚你未来三年怎么走。
想清楚了再选，别选错了再后悔。

【60-75s】在苏州搞不清楚选什么的？
找江姐，16年经验帮你分析。
选对了再注册，一步到位。
```

**分镜：** 00s中景→10s手指对比→25s左右手分述→45s直视镜头→60s点头→70s字幕

## 📍 脚本 #8：《注册资本写多少？写多了反而麻烦》

**时长：** 60-75秒 | **风格：** 破常规认知型

**口播文案：**

```
【0-5s】韩红走面儿讲的是身份匹配。
你注册资本写多少，也讲身份匹配。

【5-15s】现在注册公司是认缴制，不用实缴。
有些老板就飘了——
顺手写个500万、1000万。
看着挺唬人，但其实是在给自己挖坑。

【15-30s】为什么？
注册资本写高了，股东责任就大了。
公司破产了、出事了，
你认缴的500万就得真掏出来。
你赔得起吗？

【30-40s】写低了呢？
跟合作伙伴谈生意，
人家看你注册资本10万，
直接不跟你玩了。

【40-55s】所以写多少合适？
看你的行业、看你的业务、看你的客户。
不是越高越好，也不是越低越好。

【55-70s】注册资本拿不准的，
注册前来找江姐聊聊。
我帮你算一个最适合的数字，
不给自己留坑，也不在客户面前丢分。
```

**分镜：** 00s中景→10s摇头无奈笑→25s手势加重→40s摊手→55s微笑→65s字幕
