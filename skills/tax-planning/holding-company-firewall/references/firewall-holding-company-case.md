## REV1（原始方案，2026.6.2）— 新设控股公司

### 场景描述

自然人A（夫）和B（妻）名下共有4家公司：
- A持有：苏州S桥国际旅行社有限公司（100%）
- B持有：苏州KB虾商务科技有限公司（100%）、苏州ls达汽车服务有限公司（80%）、苏州EU环境技术有限公司（100%）
- ls达汽车服务有外部股东C和D各10%

目标：新设一家控股公司，由控股公司持有各运营子公司股权，实现防火墙隔离效果。

## 核心架构

```
                      ┌──────────────────┐
                      │  A 44% + B 44%    │
                      │  + 第三方 12%     │  ← 一致行动人协议(A+B)
                      │  (控股公司)        │
                      └────────┬─────────┘
                               │
            ┌──────────────────┼──────────────────────┐
            │                  │                      │
       ┌────▼────┐       ┌────▼────┐            ┌────▼────┐
       │ S桥国际 │       │ KB虾   │            │ ls达   │
       │ 100%    │       │ 100%   │            │ 80%    │
       └─────────┘       └─────────┘            └─────────┘
                                            C 10% / D 10%(维持)
                                         ┌────────────┐
                                         │ EU环境     │
                                         │ 100%       │
                                         └────────────┘
```

## 关键设计要点

### 1. 第三方持股比例 ≥12%

| 持股方案 | 防火墙效果 | 控制权 |
|---------|-----------|--------|
| A 44% + B 44% + 第三方 12% | ✅ 强（打破实质一人公司认定） | 一致行动人协议（A+B合计88%） |
| A 50% + B 50%（无第三方） | ⚠️ 中（夫妻公司被穿透风险） | A+B合计控制 |

**Why 12% rather than <10%?** 10%以下法院可能认定为"象征性持股"，仍然穿透认定为实质一人公司。12%以上可主张"非一人公司"。

### 2. 外部股东（ls达）处理策略

| 外部股东关系 | 推荐策略 | 法律工具 |
|------------|---------|---------|
| 紧密（朋友/合伙人） | 策略B：收购100%（含C和D的20%） | 协商转让 |
| 一般/关系不好 | 策略A：收购B的80%，C和D各10%保留 | 控股公司持股80%>50%实现控股 |

**法律关键**：《公司法》(2023)第84条——B向控股公司转让80%股权属于"对外转让股权"，必须书面通知C和D，C和D有优先购买权。30日内未答复视为同意。

### 3. 股权转让税务路径

| 路径 | 个税 | 条件 |
|-----|------|------|
| 平价转让 | 无（增值=0） | 注册资本原值转让，净资产未大幅增值 |
| 非货币性资产投资 | 20%但可5年分期 | 财税〔2015〕41号 |
| 溢价转让 | 差额×20% | 净资产>注册资本 |

- 契税：股权转让不征（财税〔2021〕29号）
- 印花税：转让金额万分之五
- 后续分红：子公司→控股公司免企业所得税（企业所得税法第26条）；控股公司→A/B个人按20%缴纳

## 执行路线图

```
① 确定第三方人选 → 谈妥持股12%条件
② 新设控股公司（注册资本100-500万，5年实缴）
③ 经营范围含"企业管理咨询""股权投资"（不含"投资管理"等需金融牌照范围）
④ 签署一致行动人协议（A+B联合控制）
⑤ 评估各公司净资产 → 确定转让价格
⑥ 通知ls达外部股东C/D → 获取放弃优先购买权声明
    ⚠️ 外部股东反对→只能走策略A（收购B的80%）
⑦ 依次办理股权转让工商变更
⑧ 各子公司修改章程（股东变更为控股公司）
⑨ 建立财产独立管理制度（独立账户/账簿/决策记录）
```

## 关键法规索引

- 📖 公司法（2023修订）第3条、第23条、第84条
- 📖 民法典第83条
- 📖 最高院九民纪要（2019）第10-12条
- 📖 财税〔2015〕41号（非货币性资产投资分期纳税）
- 📖 财税〔2021〕29号（股权转让不征契税）
- 📖 企业所得税法第26条（居民企业之间分红免税）

## 注意事项

1. **防火墙不等于绝对隔离**——公司法第23条是防火墙的例外：如果A/B用控股公司逃债，法院会刺破面纱要求连带清偿
2. **财产独立是防火墙的基础**——即使持股架构完善，账户混同/账簿混同/业务混同都会让防火墙失效
3. **必须先做信息核实再做方案**——各公司净资产、外部股东意向、征收方式、营收数据必须逐一确认后才能做精确税务测算
4. **客户沟通话术**：核心卖点是"花最小的代价（引入第三方12%）让防火墙真正生效"，而不是"控股公司能帮你完全隔离"

## REV2（方案变更，2026.6.4）— 收购苏州HP旅游汽车有限公司

### 变更触发

| 维度 | 原始方案(REV1) | 变更后(REV2) | 变化原因 |
|------|---------------|-------------|---------|
| 主体来源 | 新设公司 | 收购已有公司 | 获得现成的 道路运输经营许可证 |
| 被收购公司 | 无 | 苏州HP旅游汽车有限公司 | 有3位自然人股东，持有道路运输经营许可证 |
| 控股公司定位 | 纯持股平台 | 持股 + 实际运营 | 不仅是控股公司，还要经营运输业务 |
| 收购对象性质 | — | 有存量业务+资质的运营实体 | 许可证不能单独转让，需连公司一起买 |
| 操作复杂度 | ★★★ | ★★★★★ | 涉及股权收购、许可证维系、3位股东协调 |

### REV2 新增设计要点

#### 1. 收购路径的核心矛盾

收购苏州HP公司作为控股公司，与REV1"新设控股公司"的本质区别：

**矛盾一：许可证不可分离性**
- 道路运输经营许可证是附着在公司主体上的行政许可，不能单独过户
- 必须通过股权收购获得公司控制权→许可证维持在公司名下
- ⚠️ 收购完成后需确认许可证是否需变更或续期（涉及交通运输管理部门审批）

**矛盾二：存量业务与防火墙隔离**
- HP公司有存量业务（需确认营收、合同、人员、债权债务）
- 防火墙架构要求HP公司作为控股公司持有各子公司股权
- 但HP自身也有业务 → 意味着HP既是控股公司又是运营实体
- **设计挑战**：如何隔离HP公司自身的运营风险，不让其影响控股平台的安全

**矛盾三：3位股东退出安排**
- HP公司有3位自然人股东 → 股权收购需全部协商一致
- 收购合同、交割安排、过渡期管理都比新设公司复杂

#### 2. 新架构示意图（REV2 初步方向）

```
                              ┌──────────────────────────┐
                              │  A 44% + B 44%            │
                              │  + 第三方 12%              │  ← 一致行动人协议
                              │  (苏州HP旅游汽车有限公司   │
                              │   收购后改制为控股平台)    │
                              └────┬─────────────────┬────┘
                                   │                  │
                         控股+运营 │            需隔离  │
                                   │                  │
                              ┌────▼────┐       ┌────▼──────────┐
                              │ HP公司   │       │ 原4家子公司   │
                              │ 运输业务  │       │ (S桥/KB虾/   │
                              │ (需单独   │       │ ls达/EU)      │
                              │  隔离)    │       │               │
                              └──────────┘       └───────────────┘
```

**关键设计要求：**
- HP公司收购后需在组织架构上做**业务隔离**：运输业务单独设部门/分公司/子公司
- 控股职能（持有其他公司股权）与运营职能（运输业务）**账务分离**
- 许可证维系条件必须核查：公司名称变更是否影响许可证有效？

#### 4. 许可证核查清单（收购前置）

道路运输经营许可证收购前必须逐项核查：

- [ ] 许可证有效期是否≥1年（太短需先续期）
- [ ] 许可证上公司名称与营业执照是否一致
- [ ] 许可经营范围是否包含"道路旅客运输"或所需具体类别
- [ ] 是否有行政处罚或违规记录（影响年审）
- [ ] 车辆数量是否符合许可证等级要求（低于最低车辆数可能被吊销）
- [ ] 安全生产管理制度是否健全（高客运输有特别要求）
- [ ] 公司名称变更后许可证是否需重新核发或变更

> ⚠️ **道路运输经营许可证的特殊性：** 该许可是属地核发（苏州交通运输局），收购后即使法人/股权变更，许可证主体仍是苏州HP旅游汽车有限公司，不影响许可证效力。但如果要更改公司名称，需要重新向交通运输管理部门申请许可证变更。**建议收购后两年内不改名，或确认变更流程后再操作。**

#### 7. 常见陷阱警告

| # | 陷阱 | 后果 | 应对 |
|---|------|------|------|
| 1 | 许可证未核查有效期就签约 | 收购后发现许可证即将到期，无法续期 | 签意向书之前先验许可证 |
| 2 | 未发现HP公司的或有债务 | 收购后债务追上门，防火墙失效 | 尽职调查+出让方担保条款 |
| 3 | 3位股东中1位不同意 | 收购失败 | 提前确认3位股东意愿，签约前签意向书 |
| 4 | 收购后立即改公司名 | 许可证需重新申请，可能影响业务连续性 | 建议2年内不改名 |
| 5 | 未做业务隔离 | HP公司运输业务亏损拖累控股平台 | 组织架构+账务分离 |
