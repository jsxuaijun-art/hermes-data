# 裁判文书网 (wenshu.court.gov.cn) 爬取限制记录

## 最终结论 (2026.5.13 验证)

**完全无法从 CLI 通过编程方式搜索或获取判决书。**

## 尝试过的技术手段

| 方法 | 结果 | 备注 |
|------|------|------|
| GET 首页 | 200 OK, 返回HTML | 含JS动态渲染 |
| POST `/website/parse/rest.qz` (JSON) | 404 错误页 | API 已更换或被屏蔽 |
| POST `/website/parse/rest.qz` (form-urlencoded) | 404 错误页 | 同上 |
| GET search page with URL params | 200 但只返回首页框架 | JS渲染内容不可见 |
| 搜索引擎 site:wenshu.court.gov.cn (Bing/搜狗) | ❌ 无结果 | 深页不被索引 |
| Google site: 搜索 | ❌ 超时/被墙 | 网络环境限制 |

## 反爬机制识别

1. **验证码** (CAPTCHA) — 访问搜索页或API前需滑块验证
2. **动态Token** — 每次请求需携带从JS生成的动态token
3. **IP限流** — 短时间内多次请求会被封IP
4. **User-Agent 检测** — 需真实浏览器UA
5. **Referer 校验** — 需从特定页面跳转
6. **JS动态渲染** — 搜索结果通过JS异步加载，不会出现在静态HTML中

## 替代方案 (已验证可行)

1. **用户直接在浏览器打开 wenshu.court.gov.cn 手动搜索** — 最可靠
   - 建议关键词: "终身使用权" + "软件" + "违约"
   - 或 "合同目的不能实现" + "软件"
2. **法律平台类案分析** (lawtime.cn, 律霸网, 找法网) — 能找到法律咨询/分析但非原始判决
3. **法律数据库** (北大法宝 pkulaw.com, 威科先行) — 需付费订阅
4. **最高法官网案例库** (https://www.court.gov.cn/) — 仅公布指导案例
5. **微信公众号文章** — 很多律师写类案分析，搜狗微信搜索可用

## 从本次会话获取的关键经验

- 代理记账公司的法律风险中也涉及软件服务合同（如财务软件、代账软件）
- 用户知道自己维权时的法律依据（民法典第582条的"更换"补救措施），代理人只需放大论证
- "更换等同价值其他软件" 这个诉求在法庭上比"退费"更理直气壮，应在文书中着重强调用户善意
