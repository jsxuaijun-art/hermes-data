# 公司法·注册资本/股东出资责任·类案裁判要点

## 法律框架（2024年新《公司法》）

| 法条 | 内容 | 适用场景 |
|------|------|---------|
| 第50条 | 股东未按期足额缴纳出资 → 赔偿公司损失 | 出资违约 |
| 第52条 | 股东失权制度：催缴书 → 60天宽限期 → 董事会决议 → 失权通知 → 丧失未出资股权 | 股东持续不履行出资义务 |
| 第54条 | 加速到期：公司不能清偿到期债务 → 公司或债权人有权要求已认缴但未届期的股东提前缴纳出资 | 公司资不抵债 |
| 第88条第1款 | 未届出资期限转让股权 → 受让人承担出资义务；受让人未按期缴足 → 转让人承担补充责任 | 股权转让后的出资归属 |
| 第88条第2款 | 瑕疵出资（已到期未缴）转让股权 → 转让人与受让人连带责任 | 瑕疵股权转让 |

**配套规定**：国务院令第784号《关于实施〈中华人民共和国公司法〉注册资本登记管理制度的规定》
- 2024年6月30日前设立的公司 → 剩余出资期限自2027年7月1日起超过5年的 → 应于2027年6月30日前调整至5年内
- 有限责任公司股东应当在调整后的期限内足额缴纳出资

## 核心裁判规则

### 规则一：加速到期的触发条件

**新《公司法》第54条** — 三要件：
1. 债权已到期
2. 公司不能清偿到期债务
3. 股东已认缴出资但未届出资期限

三个条件**同时满足**才能启动加速到期。

**认定"不能清偿到期债务"的标准**：
- 公司作为被执行人的案件，法院穷尽执行措施无财产可供执行（已具备破产原因但未申请破产）
- 或公司明显缺乏清偿能力（停止支付、资产不足以清偿债务等）

### 规则二：失权制度的程序流程

第52条要求严格程序：
1. 公司发出书面催缴书（可载明不少于60天的宽限期）
2. 宽限期届满后，股东仍未履行
3. 公司经**董事会决议**向股东发出失权通知（书面形式）
4. 通知发出之日 → 股东丧失未缴纳出资对应的股权
5. 丧失的股权 → 6个月内转让或注销；未处理的 → 其他股东按出资比例补足

### 规则三：股权转让后的追责链

第88条第1款（新法重点）：
- 转让人 →（股权转让）→ 受让人
- 受让人承担缴纳出资的义务（主责）
- 受让人未按期缴足 → 转让人承担**补充责任**
- 即：债权人先追受让人，追不到再追转让人

## 类案裁判要点

### 案例1：北京西城法院首例第54条加速到期案（2024年7月1日一审）

**案由**：债权人李某申请追加股东张某为被执行人
**事实**：
- 张某持有公司60%股权，认缴180万元，出资日期为2052年3月15日
- 公司欠李某债务经法院判决后未能清偿
- 执行程序穷尽措施后无财产可供执行
- 法院追加张某为被执行人

**裁判结果**：张某在180万元未出资范围内对李某承担补充赔偿责任
**裁判依据**：新《公司法》第54条

### 案例2：平顶山石龙区法院（2025年）

**事实**：
- 甲公司欠天某公司货运代理费78529元
- 执行无果后终本
- 甲公司股东王某、李某认缴出资未实缴（出资期限2037年），2023年3月以0元转让给丁某
- 受让人丁某认缴5000万元

**裁判结果**：
- 丁某在5000万元范围内对甲公司债务承担补充赔偿责任
- 王某、李某（原股东/转让人）对丁某的债务承担补充责任
- 适用第88条第1款：转让人对受让人未缴出资承担补充责任

## 实操建议（给企业客户）

### 对注册资本认缴但未实缴的企业

1. **实缴能力评估**：在认缴期限（含过渡期调整后）内，确保有实缴能力
2. **最低资产保全**：即使认缴期限未届满，公司账户应保持至少覆盖短期债务的流动资产
3. **避免资不抵债**：一旦公司出现"不能清偿到期债务"的迹象，加速到期风险触发
4. **股权转让风险提示**：以0元或低价转让认缴未实缴股权 → 转让人仍可能被追偿（第88条）
5. **出资凭证保管**：实缴后及时验资、公示、保留银行流水证明

### 对债权人/合作方

1. **签约前核查**：通过国家企业信用信息公示系统查询对方注册资本实缴情况
2. **加速到期诉讼**：发现对方已不能清偿债务 → 可同时起诉公司和未实缴股东
3. **追加被执行人**：执行程序中直接申请追加未实缴股东
