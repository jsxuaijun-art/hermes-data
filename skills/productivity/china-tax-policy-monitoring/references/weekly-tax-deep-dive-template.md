# 财税战略情报中心 — 周度报告模板

## 模板结构说明

配合 `weekly_tax_loader.py` 使用。模板中的日期占位符由加载脚本在运行时替换：
- `{{PREVIOUS_MONDAY_YYYY-MM_DD}}` / `{{PREVIOUS_SUNDAY_YYYY-MM_DD}}` — 上周范围
- `{{WEEK_NUM}}` — ISO周数
- `{{NEXT_30D}}` / `{{NEXT_60D}}` / `{{NEXT_90D}}` — 未来日期

实际文件路径：`~/.hermes/cron/weekly_tax_deep_dive.md`

## 输出格式

报告周期：{上周一} 至 {上周日}（第{X}周）

- 新政策解读、政策调整方向、影响评估

- 典型案例深度拆解

### 三、高频风险点
- 金税四期比对逻辑

- 跨部门数据共享机制

- 税负率对比
