# 收购现有公司作为控股公司 — 信息采集分组问卷

> 本问卷适用于以下场景：夫妻/实控人计划收购一家已存续公司（尤其是带特殊经营许可证的），
> 将其改造为控股母公司并实际运营，同时覆盖风险隔离和架构重组需求。
>
> 一共4组问题，建议分步骤发给客户填写。

---

## 组A：夫妻AB现有公司体系

> 了解"被收购方"的全景——多少家公司、股权分布、财务状况。

| # | 问题 | 目的 |
|---|------|------|
| A1 | 收购方（夫妻A/B）目前名下共有几家公司？分别什么名字？个人持股比例各是多少？ | 明确收购主体全景 |
| A2 | 各公司的注册资本分别是多少？**实缴**了多少？ | 判断净资产、转让成本 |
| A3 | 各公司**净资产**大概是正还是负？金额能说个大概范围吗？ | 转让定价、个税测算基础 |
| A4 | 各公司当前**经营状态**：正常经营/停业/亏损/盈利？ | 防火墙覆盖范围 |
| A5 | 各公司有无**外部股东**？各占多少？跟夫妻的关系如何？ | 外部股东处理策略前置 |
| A6 | 各公司有无**大额负债、对外担保、涉诉**？ | 防火墙有效性评估——避免把雷带进新架构 |
| A7 | 各公司当前的**征收方式**：一般纳税人/小规模？查账/核定？ | 税务衔接 |

---

## 组B：标的公司——苏州HP旅游汽车有限公司 详情

> 标的公司本身的信息——这是收购决策的基础。

### B组-1：基本面

| # | 问题 | 目的 |
|---|------|------|
| B1 | 注册资本是多少？**实缴**了多少？ | 股权取得成本 |
| B2 | 3位自然人股东各持多少比例？他们之间是什么关系（亲属/朋友/合伙）？ | 谈判策略、优先购买权安排 |
| B3 | 公司哪一年成立？是否正常经营中？ | 判断是否有合规/税务历史遗留 |
| B4 | 公司**净资产**大概多少？最新一期财报有吗？ | 转让定价基础、税务局核定风险 |

### B组-2：财务状况

| # | 问题 | 目的 |
|---|------|------|
| B5 | 资产端：固定资产（车辆等）大概值多少钱？有无房地产/土地使用权等不动产？ | 涉税核定门槛 |
| B6 | 负债端：有无银行贷款、其他借款、对外担保？ | 收购后将承继的负债 |
| B7 | 有无**欠税**、**涉诉**（包括潜在诉讼）、**行政处罚**？ | 收购风险——避免收了一堆雷 |
| B8 | 近三年盈亏趋势怎么样？目前有未分配利润吗？ | 后续分红个税 |

### B组-3：道路运输经营许可证

| # | 问题 | 目的 |
|---|------|------|
| B9 | 许可证**发证机关**是谁（苏州市交通运输局？省厅？） | 确认变更审批机关 |
| B10 | 许可证**有效期**到什么时候？每年需要**年审**吗？ | 判断收购后是否需要马上续期 |
| B11 | 许可证**跟公司主体绑定**到什么程度——变更股东、法人、注册资本会影响许可吗？ | 核心——弄清收购后许可是否自动延续 |
| B12 | 许可证是否有**经营区域/类型限制**（比如仅限苏州境内/仅限旅游客运）？ | 判断未来业务扩展空间 |
| B13 | 许可对应的**车辆和司机**情况——公司的车是自购还是挂靠？司机是自有还是外聘？ | 运营成本和安全责任评估 |

## 组C：收购方案方向

> 明确收购方的意图和约束条件。

| # | 问题 | 目的 |
|---|------|------|
| C1 | **由谁来收购？**——夫妻中的一方个人？还是用他们现有的某家公司去收购？ | 决定交易主体和税务路径 |
| C2 | **收购比例**：打算全资收购（100%）还是只收购控股权（≥67%）？ | 3位股东的退出安排 |
| C3 | **收购对价**：双方初步协商的价格是多少？定价依据是什么（净资产/评估值/协商）？ | 个税测算、印花税基数 |
| C4 | **资金来源**：自有资金？还是需要过桥借款/类贷款？ | 收购能否顺利执行 |
| C5 | **3位股东的态度**：都愿意卖吗？有没有不愿意退出的？如果不退，是否接受保留小股东身份？ | 交易可行性前提 |
| C6 | **交易时间表**：打算什么时候完成收购？有急迫性吗？ | 确定优先级和路线图节奏 |

## 组D：许可证专项（增补版）

> 如果标的公司的许可证是关键收购动机，这些问题必须深入。

| # | 问题 | 目的 |
|---|------|------|
| D1 | 道路运输经营许可证是**到期自动续期**还是需要重新申请？续期条件是哪些？ | 评估许可的长期可持续性 |
| D2 | 许可证对**车辆数量、类型、安全管理制度**有什么要求？ | 收购后运营合规门槛 |
| D3 | 如果收购后成立一家新的**运营子公司**由这个子公司运营，许可证能否由母公司持证→子公司运营（业务委托/挂靠模式）？ | 混合型控股的关键隔离策略 |
| D4 | 有没有类似企业做过"股东变更后许可继续有效"的先例？可以请标的公司提供往年股东变更记录供参考。 | 实务中许可延续的可行性验证 |

## 组E（可选）：财税业务增值

> 如果客户已有全套代理记账/税务合规需求，以下可作为追加服务切入点：

| # | 问题 | 服务转化机会 |
|---|------|-------------|
| E1 | 标的公司目前的**记账方式**：自己做？还是已有代理记账公司？ | 接手账务 |
| E2 | 标的公司目前的**税务合规状况**：有没有被专管员约谈/被风控/被稽查过？ | 合规体检+整改 |
| E3 | 收购完成后，你们各公司想做**合并报表**还是各自独立做账？ | 财务管理转型增值服务 |
| E4 | 标的公司目前的**发票领用/开具**情况怎么样？是自开票还是代开？ | 发票管理优化 |
