# 低价营销反击战术卡 v2.0

> 来源：1面对低价营销.docx + 价格上调案例.docx + 中宏价格（一）.docx + 全网搜索
> 使用方式：遇到客户压价时，组合使用

---

## ⚡ 核心原则（先背下来）

1. **绝不谈价先谈价值** — 价值没到位，价格永远是"贵"
2. **降价必须换条件** — 减少服务/缩短期限/增加预付款
3. **客户砍的是不安全感，不是钱** — 给他安全感比便宜更重要
4. **你的底气来自资质** — 高级会计师、TSC五级、16年实战

## 🗡️ 战术卡1：价值对比法

**触发场景：** 客户说"XX家比你们便宜XXX"

**使用时机：** 客户已经比完价，抱着别家报价回来压你

**话术：**
```
"王总，同样的价格比服务，同样的服务比质量。
您知道为什么同样的产品价格不同吗？

（方法一：类比法）
就像菜市场——时间长的菜、水里泡过的才便宜处理。
超市里，快过期的食品才打折。

（方法二：二手车销售类比）
您看二手房，一个亿的房子和一百万的房子，
都要经历带看、撮合、签约、过户的流程。
我们的服务流程也一样——不管客户开多少发票，
该核对的票据、该做的分录、该出的报表，一步不能少。
"

（方法三：电梯轿厢类比，来源：于峻晨降价调低.docx）
"记账就像电梯安装——轿厢大一点小一点，
都要上门测量、安装、调试。
会计行业有更低的，但低有低的做法——
节约流程你是外行看不出来的。"
```

**输出物：** 发给客户一张"价格构成对比图"

## 🗡️ 战术卡2：成本拆解法

**触发场景：** 客户说"你们不就是做做账报报税，怎么这么贵"

**话术：**
```
"王总，我给您拆一下成本——

（拆人工）
每个会计人员工资+社保+公积金：8000-15000/月
一个人最多服务40-60家标准客户
平均每家的直接人工成本：150-300元

（拆软件）
正版记账软件：每户20元/月
发票查验系统、电子税务局申报系统：持续成本

（拆办公）
商务楼租金、办公耗材、凭证纸、打印设备

（拆风险）
高级会计师把关、税务风险审核、稽查应对
这些无形的成本，才是真正值钱的地方

低价公司不负担房租、不用正版软件、
不招有经验有资质的会计、
减少流程、压缩人工——
能用这样的服务吗？"
```

## 🗡️ 战术卡3：失败案例震慑法

**触发场景：** 客户犹豫要不要选低价方案

**话术：**
```
"王总，我给你看看几个真实案例——

（案例1：年报漏报）
有个客户从我们这里转出去就为了省几百块，
结果连工商年报都忘记申报，
上了经营异常名录。
虽然能补救，但留下了记录。

（案例2：税务漏申报）
小税种最容易漏，低价公司流程粗、
缺乏系统监控，一个附加税没申报，
罚款+滞纳金就打水漂了。

（案例3：账务错乱）
低价公司会计经验不足，
做出来的账报表不平、勾稽关系对不上。
这还不是最可怕的——
最怕金税系统预警查上门，
才发现前几年的账都有问题。
那时候花的钱，就不是几百几千了。"

对了，范冰冰没有被判刑，
她的助理倒判刑了——
为什么？账是她做的。
会计责任重！"
```

## 🗡️ 战术卡4：高级会计师背书法

**触发场景：** 客户对专业度有疑虑，或需要"终极说服"

**话术：**
```
"王总，您选择代账公司，最应该看什么？
不是装修、不是价格、不是口头承诺。

（亮出王牌）
苏州盈信，全江苏省唯一以代理记账公司名义
通过高级会计师评审的公司。

2018年江苏省八九百人通过高级会计师评审——
其中强生医疗、东吴人寿、中衡设计的财务总监都有。
但以代账公司名义通过评审的，就我们一家。
注册会计师靠考试，高级会计师靠的是
真实力+真案例+专家团评审。

而且我们还获得了税务局评定的TSC五级满分评价。
这些东西做不了假——
每个都能在财政部、税务总局官网查到。"
```

## 🗡️ 战术卡5：价值清单法

**触发场景：** 客户说"你能不能给我列一下到底贵在哪"

**准备工作：** 提前打印或准备好一页纸的"对比清单"

| 对比项目 | 低价代账（200-300/月） | 盈信标准（300-600/月） | 盈信合规（800-1500/月） |
|---------|----------------------|----------------------|----------------------|
| 会计负责客户数 | 80-100家 | 40-60家 | 20家 |
| 会计资质 | 不限（常有实习生） | 中级职称起步 | 高级会计师审核 |
| 账务系统 | 盗版/免费软件 | 正版专业软件 | 正版+税务对接系统 |
| 发票查验 | 不查 | 逐张查验真伪 | 自动+人工双重校验 |
| 银行回单 | 客户自行提供 | 每月提醒+提供 | 系统自动拉取 |
| 年度/季度报告 | 没有 | 有 | 详细分析报告 |
| 税务预警响应 | 不管/收费才处理 | 提醒+基础应对 | 全流程代理 |
| 稽查应对 | 没有经验 | 有案例支撑 | 高级会计师带队 |
| 服务时间 | 上班时间 | 微信全天响应 | 专人对接 |
| 客户违约保障 | 无 | 资质备案有保障 | TSC五级背书 |

## 🗡️ 战术卡6：加减法谈判术

**触发场景：** 客户坚持要降价，谈判陷入僵局

**加法策略（加服务不降价）：**
```
"王总，价格确实做不了。
不过我可以向公司申请，给您加一项——
半年的税务体检分析服务。
这个单买要3000/年，
等于给您送了一个超值服务。"
```

**减法策略（降价换条件）：**
```
"如果一定要降价，我们可以调整服务范围——
去掉税务风险预警服务，只做基础代账报税。
价格可以降到××，
但遇到税务预警您要自己应对。
您看这样可以吗？"
```

**共度难关策略（来源：于峻晨降价调低.docx）：**
```
"考虑到您的情况，我们可以共度难关。
降价方案：
A. 集中一个公司开票，其他的降到零申报水平
B. 统一降一些

但有一条——如果恢复之前经营水平，
价格也要恢复。要不就只共苦不同甘了。"
```

## 📊 话术效果自检表

每次使用后问自己：
- [ ] 我有没有先展示价值，再说价格？
- [ ] 我有没有说清楚"不选我的代价"？
- [ ] 我有没有拿资质/证书/案例做背书？
- [ ] 我有没有给出可验证的信息（官网可查的资质）？
- [ ] 我有没有主动引导下一步动作？

> 效果不好的原因：80%是因为价值没说透就急着降价了。
