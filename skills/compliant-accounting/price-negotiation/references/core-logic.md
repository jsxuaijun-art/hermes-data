# 价格谈判核心逻辑：服务费 = 买断工作时间

## 一句话版

> 服务费买的是会计的工作时间——时间花在开票上，做账时间就少，风险就多了。

## 完整论述版

一个人的时间是有限的。服务费相当于买断了一定的工作时间。

总的工作时间就这么多——

- 花在开发票这些不需要专业的事情上（而且是做在明面上的，让你有感觉的，让大家开心的事情）
- 那么用来做账的时间就少了
- **做账是看不见的无形服务，做多做少花多少时间根本不知道**
- 现在税务大数据监管这么严格，风险也就来了

上帝对每个人都公平——一天24小时，能用来工作的时间就那点。大家都是靠出卖时间来赚钱的。

同行说"我们免费帮你开票"——你要知道：开发票没有技术含量，只是耗时间。他把分配给你的时间花在开票上了，分给认真做账的时间还剩多少？

**做账才是真正的专业，而且是看不见的无形服务。**
