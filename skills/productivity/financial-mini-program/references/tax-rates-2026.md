## 二、核心公式

```
应纳税所得额 = 年收入 - 60,000(起征点) - 三险一金 - 专项附加扣除 - 其他扣除
应纳税额 = 应纳税所得额 × 税率 - 速算扣除数
```

**年终奖计税公式:**
```
查找税率: bonus / 12 → 查月度表得 rate, quickDeduction
应纳税额 = bonus × rate - quickDeduction
```

## 四、累计预扣预缴法

每月累计到当月：
```
累计应纳税所得额 = 累计收入 - 累计起征点(5000×月份) - 累计三险一金 - 累计专项附加扣除
本月应预扣税额 = (累计应纳税所得额 × 税率 - 速算扣除数) - 以前月份累计已预扣税额
```
