## 三、Teaser模式 — 新5问（v2版）

### 第1步：亮身份锚点（30秒）
> 「我是盈信财税的XX。苏州做代账的公司几百家，但以代理记账公司名义独立通过**江苏省高级会计师评审**的，只有我们盈信一家。我们是**TSC五级**涉税专业服务机构——438.11分全省前列。」

### 第2步：口头采集5个关键问题（5-10分钟）

```
① 几个股东？你占多少？
② 年营收大概什么规模？100万以下？500万？1000万？
③ 财务是你自己管还是请了人？代账还是专职会计？
④ 你现在感觉交的税多不多？
⑤ 你是做什么行业的？
   ├─ 贸易/电商  → 「你供货商能开发票给你吗？」
   ├─ 餐饮/食品  → 「你食材采购一个月多少？能拿到发票吗？」
   ├─ 工程/装修  → 「你包工头的工资走什么渠道发的？」
   ├─ 科技/互联网→ 「你有专利或软著吗？」
   ├─ 制造业    → 「你卖废料的钱走哪里？」
   ├─ 零售/实体  → 「你几个店？每个店独立执照还是统一执照？」
   ├─ 医疗/教育  → 「你有行业资质证吗？好办吗？」
   └─ 服务/咨询  → 「你的客户要你开票的多不多？」
```

**关键技巧：每个问题先说「我问这个是为了判断XX」→ 显得在诊断不是在查户口。**

### 第3步：输出3个发现+1个悬念（5-8分钟）

#### 发现A：税务风险（最痛，第一个讲）
根据动态定位（看客户回答的税感知+财务负责人+行业追问）：
- 例① 税交得多+财务自己管 → 「你税交得多不一定是赚得多，是成本没扣除到位」
- 例② 税交得还行+请了代账 → 「代账只报税不做规划，两个概念」
- 例③ 工程+包工头 → 「包工头现金发工资，不能抵成本还有个税风险」

#### 发现B：股权/架构风险
- 67%以下→无绝对控制权
- 50%/50%→僵局风险
- 33%/33%/33%→任何两人联合推翻第三人
- 创始人持股过低+无一致行动人→被踢出局

#### 发现C：利润提升点（转负为正）
- 例：业务拆分→综合税负降30-50%
- 例：供应商谈开票→实际成本反而低了

#### 悬念D：预约下回的钩子
> 「完整的方案需要根据你的数据仔细算。你看我们约个时间，我花1-2小时给你做一次完整的企业诊断？出份书面报告给你。」
