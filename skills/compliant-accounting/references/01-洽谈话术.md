## 01 餐饮、酒店、连锁

### 行业痛点

| # | 痛点 | 本质 |
|---|------|------|
| 1 | 个人卡流水太大，税务稽查风险（门店50-80%个人卡收款） | 隐匿收入→偷漏税 |
| 2 | 收银存在很大漏洞（POS机、小票、吧台、采购） | 内控缺失→跑冒滴漏 |
| 3 | 结算方式多种，对账工作量大（对公户/个人卡/微信） | 财务混乱→效率低 |
| 4 | 三高一低：毛利高、成本高、费用高、利润低 | 结构性问题 |
| 5 | 成本发票无法取得（菜市场采购、路边小店、供应商不开票） | 缺进项→税负高 |
| 6 | 人员流动性强，很多员工没交社保，临时用工 | 用工风险→社保稽查 |
| 7 | 采购存货多，库存积压，产品下降，库存呆滞 | 资金占用 |
| 8 | 采购价格波动大，猫腻多（回扣现象） | 内控缺失→利润流失 |
| 9 | 门店跨区经营，多门店，资金管控风险 | 管理失控 |
| 10 | 大数据应用、数字化技术（收银系统+ERP如何业财一体） | 信息化滞后 |
| 11 | 股权激励问题（总部品牌运营+门店管理，高管激励，分红比例） | 治理结构 |

### 谈单解决思路

```
1. 流水太大税务风险 → 合规路径：核定征收/小规模分流/合理费用列支（3种方法）
2. 管理制度混乱 → 建立规范化管理体系（制度+流程+监督）
3. 对账不准 → 通过系统归类，与收银系统匹配核对，业财融合
4. 三高一低 → 做产品标准化，各项指标占比分析，结构调整
5. 缺进项票 → 规范采购渠道，寻找有资质供应商（供应商管理制度）
6. 未买社保 → 劳务派遣/灵活用工/商业保险（风险转移和应对）
7. 库存积压 → 用数据说话，确定采购频次
8. 采购猫腻 → 建立规范采购体系和内控制度（询价+监督+轮岗）
9. 资金管控 → 建立资金池（集团资金管理模式）
10. 数字化 → 业财一体化，信息化管理（数字化运营方案）
```

### 开口话术

> 「王总，餐饮行业现在金税四期最关注的就是**个人卡收款**。你门店50%以上的流水走老板私卡，税务局后台看你的纳税申报和银行流水对不上，系统自动标红。我们做过一个连锁餐饮客户，之前用低价代账，一年被约谈两次。换了合规账后，我们把他的收银系统和ERP打通了，每个月给他出**经营分析报告**——哪家店赚钱、哪个菜毛利高、采购有没有猫腻，一眼看明白。他的原话：'之前是闭着眼睛做，现在是戴着眼镜做。'」

## 02 医药、医疗

### 行业痛点

| # | 痛点 | 本质 |
|---|------|------|
| 1 | "两票制"、"三高一低"：高毛利、高风险、高回扣、低净利润；回扣返点达销售收入10% | 费用列支难题 |
| 2 | 销售人员提成高，未代扣代缴个税，如何列支销售提成 | 个税合规 |
| 3 | 工资平均基数大于社平工资基数，社保改革如何降成本 | 社保压力 |
| 4 | 供应商预付款，客户有账期，应收账款过大，资金压力 | 现金流 |
| 5 | 买发票，其他应付款过大 | 虚开风险 |
| 6 | AB两套账问题 | 账务混乱 |
| 7 | 为投标，关联企业注册成不关联公司，资金拆借通过股东周转，个人流水大 | 架构混乱 |
| 8 | 虚开增值税发票 | 刑事风险 |
| 9 | 库存商品账实不符 | 管理黑洞 |
| 10 | 收入按开票时间确认，不是按权责发生制 | 会计不规范 |

### 谈单解决思路

```
1. 回扣列支 → 成立个独企业（经营范围：广告宣传），回扣开成广告费/业务宣传费
2. 销售提成个税 → 定个税工资标准，超出部分列福利费/个独开票/个人代开
3. 社保降成本 → 以社平工资为上限，超出由另一家管理公司发放
4. 应收账款大 → 应收账款质押融资贷款 + 设置信用政策
5. 其他应付款大 → 做资金流规划，降低其他应付款
6. 两套账 → 销售收入开票达90%，费用合理化列支，做一套账
7. 关联公司资金拆借 → 公司之间直接周转，约定借款利率，支付利息
8. 虚开风险 → 票流、物流、资金流三流一致
9. 库存账实不符 → 盘点找出差异，账面大于实物的做未开票收入申报
10. 收入确认 → 按权责发生制、所有权转移确认
```

### 开口话术

> 「李总，医药行业现在最头疼的——回扣怎么出？两票制下，以前那套走不通了。我们给一个医疗设备代理商做过合规方案，他们销售回扣一年200多万，之前全走个人卡，财务天天提心吊胆。我们帮他搭了一个**个独企业+费用合规化**的架构，回扣变成合法的业务宣传费，有票、有合同、有流水。这200万从'说不清'变成了'合法费用'，不但规避了虚开风险，还抵扣了所得税。花在合规上的钱，其实就是在帮你赚钱。」

## 03 生产、制造

### 行业痛点

| # | 痛点 | 本质 |
|---|------|------|
| 1 | 核算问题：精细化核算、财务分析、管理报表不及时 | 管理决策无依据 |
| 2 | 预算问题：研发延续性、战略与资源匹配度 | 资源配置混乱 |
| 3 | 信息化程度低：ERP实施不完善，跨部门沟通难，数据归集不准 | 管理黑箱 |
| 4 | 资金问题：应收账款多，赊销周期长 | 现金流紧张 |
| 5 | 利润点薄：处于微笑曲线底端，原料供应商相对强势 | 盈利困难 |
| 6 | 个税与社保问题：劳动密集型，缴纳不一致，临时用工 | 用工合规 |
| 7 | 融资难：资本市场偏好问题，投资回报周期长 | 发展受限 |
| 8 | 采购舞弊：采购流程设置简单，吃回扣 | 利润流失 |
| 9 | 库存管理混乱，信息不准确 | 效率低下 |
| 10 | 税务风险：现金业务多，税务政策运用不全面 | 合规风险 |
| 11 | 股权激励与股权分红（高管激励） | 人才流失 |

### 谈单解决思路

```
1. 核算问题 → 建立数据仓库，收集一线数据，建立管理报表库
2. 预算问题 → 建立全面预算管理，费用分摊规则在预算环节制定，落实到考核制度
3. 信息化 → 加强信息化建设，建立管理驾驶舱，加强数字化管理思维
4. 资金问题 → 建立资金预算表、预测表、排款计划
5. 利润薄 → 考虑：经营利润+现金利润+税务利润+大利润四个维度
6. 用工合规 → 业务合作/劳务外包/分拆/转移/消化
7. 融资难 → 加强财务管理+合规性+多种融资方式结合
8. 采购舞弊 → 完善采购流程+询价机制+监督机制+轮岗机制
9. 库存混乱 → 完善库存管理流程+组织结构+物料编码+部门协调
10. 税务风险 → 梳理交易结构+风险点+优惠政策+税筹手段
```

### 开口话术

> 「张总，制造业老板最常见的困惑——**赚的钱去哪了？** 车间加班加点，月底一算净利润还不如隔壁开饭店的。这不是你不行，是财务没告诉你钱漏在哪了。我们帮一个苏州的机械加工厂做过合规账服务，三个月把他的成本分析做出来——原材料采购价格比市场价高了15%，为什么？采购吃回扣。车间报废率8%，行业平均3%，差距在哪？这些数据，他之前的代账会计从来没跟他说过。合规账每月给你一份**经营分析报告**，你看到数字就知道该管哪。」

## 04 房地产、建筑

### 行业痛点

| # | 痛点 | 本质 |
|---|------|------|
| 1 | 对外借款数额巨大（尤其民间借贷），利息高、无发票、无法税前扣除 | 融资成本高 |
| 2 | 个人卡过账频繁，大量非正常交易通过个人卡消化 | 隐匿收入 |
| 3 | 税收成本高，节税方法全靠买发票，成本高、风险大 | 虚开风险 |
| 4 | 劳务用工问题面临严峻挑战 | 用工合规 |
| 5 | 政策影响很大（限购/限贷/限价），对运营产生严重影响 | 政策风险 |
| 6 | 股权设计混乱，自然人持股，一家出事多家受牵连 | 风险传染 |
| 7 | 资金压力大，资金链管理基本老板自己在管 | 管理失控 |
| 8 | 基本上是糊涂账，财务工作对老板起不到作用 | 财务无效 |
| 9 | 库存压力大，大量不良资产积压 | 资产效率低 |
| 10 | 公私不分，老板的钱和公司的钱分不开 | 股东风险 |

### 谈单解决思路

```
1. 民间借贷无票 → 发行可转债/做私募股权，获取合法利息票据
2. 个人卡过账 → 设立小规模纳税人公司，屏蔽非正常交易
3. 买发票 → 转移定价方案，用关联方交易转移利润
4. 劳务用工 → 劳务派遣
5. 政策影响 → 销售分劈（收入拆分）
6. 股权混乱 → 顶层设计
7. 资金压力 → 资金管理体系导入
8. 糊涂账 → 账体系导入
9. 不良资产 → 流程及风险控制体系导入
10. 公私不分 → X公司设立
```

### 开口话术

> 「陈总，做建筑的老板十个有九个自己管钱，为什么？因为**信不过财务**。但又没办法——民间借贷走公账不正规，个人卡又怕查。这不是你的错，是你的公司架构和财务体系没搭好。我们给苏州一个建筑公司做过合规账，月流水600多万，之前老板个人卡走400万。我们帮他搭了**X公司架构+资金管理体系**，三个月把他个人卡里的资金全理顺了。原来每个月光利息支出白交20多万，现在通过可转债方式，利息合法入账还能抵税。他说的：'早三年找你们，我多赚一套房。'」

## 05 软件、互联网、科技

### 行业痛点

| # | 痛点 | 本质 |
|---|------|------|
| 1 | 业务流程简单（研发+技术人员80%以上） | 资产结构单一 |
| 2 | 用工成本/技术研发成本高（技术工程师薪酬高） | 成本高 |
| 3 | 进项成本少（基本没有进项发票） | 增值税负高 |
| 4 | 销项有票（客户签合同需开发票付款） | 收入透明 |
| 5 | 增值税比较高（进销差额大） | 税负不匹配 |
| 6 | 人员工资高、个税高、社保缴纳多（2-5万/月） | 个税社保压力 |
| 7 | 股权设计、股权激励需求（优秀技术/管理人才） | 人才绑定 |
| 8 | 企业容易做大，资本市场关注度高 | 上市准备 |
| 9 | 融资估值，市盈率倍数如何做大（应对财务尽职调查） | 资本运作 |
| 10 | 股权激励、股权分红问题（股权转让溢价如何避税） | 退出税负 |

### 谈单解决思路

```
1. 工资高个税高 → 拿钱的6种方法（业务拆分/费用报销/个独/分红等）
2. 软件产品增值税 → 即征即退（增收16%，退13%，实际交税3%）
3. 研发费用 → 加计扣除（所得税前列支，加计扣除100%）
4. 高新企业 → 企业所得税15%
5. 小微企业 → 企业所得税优惠（小微政策）
6. 硬件设备分拆 → 硬件16%+软件3%（退后）+服务6%（多税率拆分）
7. 股权架构 → X公司、Y公司、高管持股平台搭建
8. 融资溢价避税 → 洼地注册+Y公司套现
9. 上市规范 → 3年规范账准备+报送材料
10. 资本运作 → 专业财务团队培训，支撑发展
```

### 开口话术

> 「刘总，做软件的公司最亏的地方——**人贵、票少、税重**。技术骨干一个月5万，个税社保扣完到手3万出头。公司想留人，钱都交税了。我们帮过一个苏州SaaS公司，研发30多个人，月工资支出过百万。我们帮他做了**业务分拆+高新认定+软件即征即退**的组合方案——研发费用加计扣除省了所得税，软件产品即征即退把增值税从16%降到3%，高管拿钱的6种方法让员工实际到手多了。一年省下来的税，够再招5个工程师。这叫：**合法合规地省钱，而不是偷偷摸摸地逃税。**」

## 06 国际贸易

### 行业痛点

| # | 痛点 | 本质 |
|---|------|------|
| 1 | 出口退税问题，报关、核销问题（商品HS海关编码） | 退税合规 |
| 2 | 个体户采购，缺进项增值税发票（虚开骗取出口退税罪） | 虚开/骗税风险 |
| 3 | 传统外贸：报关单、商业发票、通关单、外汇核销 | 流程繁琐 |
| 4 | 买单报关的监管（非正常退税业务，核销无法对应） | 合规隐患 |
| 5 | 全球CRS跨国金融数据监管（外汇转账，个人资产海外监管100万美金） | 跨境监管 |
| 6 | 国内利润转移到国外问题（报关价做低，国外售价高） | 转移定价 |
| 7 | 外汇监管，地下钱庄交易（香港公司→老板个人卡） | 外汇合规 |
| 8 | 离岸公司应用（香港、英属维尔京、开曼群岛） | 架构设计 |
| 9 | 工资高、社保、个税问题 | 用工合规 |
| 10 | 跨境电商平台（亚马逊/易贝/速卖通/海外仓） | 平台税务 |
| 11 | 国际支付工具（TT电汇、信用证、Paypal、西联汇款） | 资金通道 |
| 12 | 国际VIE架构，X公司股权架构：如何节税避税 | 跨境税筹 |

### 谈单解决思路

```
1. 退税合规 → HS海关编码选择：退税率高的，监管类型少的
2. 虚开骗税 → 改变交易性质：个体户采购→集中采购（从源头规避虚开认定）
3. 买单报关 → 规范报关单证（证明出口真实性）
4. 退税问题 → 注意退税率差异（16%/13%/10%视产品而定）
5. CRS监管 → 海外资产个人卡100万美金监管，购买保险基金避税
6. 利润转移 → 注意个税20%风险
7. 地下钱庄 → 规范供应链，避免香港公司→个人卡
8. 个税社保 → 费用成本化+业务拆分+合理拿钱方法
9. 跨境电商 → 平台代扣税（如亚马逊帮商家扣税）
10. 支付工具 → 多种支付工具组合运用，避开1-2万美金外汇额度限制
11. VIE架构 → X公司适合国内企业避个税，海外无效
```

### 开口话术

> 「周总，做外贸最怕的事——出口退税款还没到，税务局先到了。我们不比那些大会计师事务所，但**实操经验**扎实。给一个苏州做纺织出口的客户做过合规账——原来的报关方式、采购模式都有隐患，HS编码选的不对，退税率少拿了3个点，一年少退几十万。我们帮他重新梳理了**采购链+报关链+退税链**，管了一年多，退税一分没少，稽查零问题。做外贸的不缺订单，缺的是**一个懂出口退税、懂外汇监管的财务搭档。**」

## 07 物流、供应链

### 行业痛点

| # | 痛点 | 本质 |
|---|------|------|
| 1 | 两套账问题：开票记入收入，不开票记入个人卡 | 隐匿收入 |
| 2 | 增值税虚开发票问题严重 | 刑事风险 |
| 3 | 个人卡流水过大，大量现金交易，逃避个税 | 逃税风险 |
| 4 | 劳务用工问题面临严峻挑战 | 用工合规 |
| 5 | 无票支出项目较多 | 缺进项 |
| 6 | 股权设计混乱，自然人持股，一家出事多家牵连 | 风险传染 |

### 谈单解决思路

```
1. 两套账 → 两账合一势在必行（唯一出路）

```

### 开口话术

## 通用场景话术

### 破冰话术（第一次接触）

> 「我是盈信财税的江敏，做了25年财税。我们不做那种收一堆发票就完事的低价代账，做的是**合规账**——帮企业把账做规范、把税缴明白、把风险看清楚。简单说：让你的财务数据，变成你做决策的底气。
>
> 对了，我们公司是**TSC五级涉税专业服务机构**（438.11分），这是税务局对涉税服务行业的最高信用评级。全国代理记账公司能评上TSC五级的不到0.5%。为什么说这个？因为税务对我们服务的客户有天然信任感——我们合规，意味着我们服务的客户也合规，被稽查的概率自然就低。」

### 核心价值话术（当客户问"有什么不一样"时）

> 「传统代账是做'昨天的账'——你把发票给他，他做进去报了就完事。合规账是管'明天的账'——我提前告诉你：这票能不能收、这钱能不能走、这笔交易有没有风险。你现在的代账会计，有没有每个月给你出**财务分析报告**？有没有告诉你哪些地方能省税？如果你的会计什么都没说——说明他只在做账，没在帮你管钱。」

### 异议处理话术

**客户嫌贵（基础版）：**
> 「我理解。你觉得贵，是因为你在跟500块一个月的代账公司比。但你想一个问题——万一被稽查了，罚款是你的钱，不是代账会计的。合规账一年也就多花几千块，但一个税务罚单可能就是十几万。**你是在花几千块，保十几万。**」

**客户嫌贵（杀手锏——怼低价同行）：**
> 「王总，想一个问题：您的服务费其实是买断了我们一定的时间，您支付的这个价格就对应着我的工作时间。肯定会有同行说：'我们是跟别人不同，帮你免费开票的'——但是您要知道：开发票这件事情是没有技术性的，只是耗时间而已。他把分配给你的时间全部用在开发票上面了，那你说他分配给你用来认真做账的时间会有多少？做账这件事情才是真的体现专业性，而且还是看不见的无形服务。上帝对每一个人唯一公平的就是给了24小时，每个人用来工作的时间就那么多，大家都是靠出卖时间来赚钱的。这个承诺免费开票的同行，他的时间也是这样的——做了这个就不能做那个。」

**客户拿同行低价来比：**
> 「同样的价格你要比质量，在相同的专业资质情况下，你才去对比价格，这才是科学理智的。如果你只看价格的高低，我们还是建议你多比较比较。**先比专业，再比价格。** 我们的高级会计师资质摆在这——你先看看同行有没有？再看我们公司是**TSC五级涉税专业服务机构**（438.11分，全省前列）——你再去问问同行是什么评级？没有高级会计师也不是TSC五级？那就不用比价格了。」

**客户说"我再考虑考虑"：**
> 「没问题。但我建议你考虑的时候想三件事：第一，你现在的账，经得起查吗？第二，你现在的会计，有没有主动告诉你公司哪里有问题？第三，如果税务局来了，你确定你睡得着觉？想明白了，给我个信。」

**客户说"我会计挺好的"：**
> 「那很好啊。不过我想请你**对比一下**：你给我看你会计最近三个月给你的报表，我再免费给你出一份合规账的分析报告。你看完对比，觉得我们不行，我请你吃饭。觉得我们有用，我们再聊合作——怎么都不亏。」

## 10 公转私专项话术

### 核心逻辑

客户问"公司赚了钱怎么合理拿出来"，本质是**股东分红个税20%**和**公私不分风险**的问题。
解决方案参考 `06-公转私50法.md`，这里给出最常见的3个场景话术。

### 场景一：老板想从公司拿钱

> 「钱在公户里是你的，但拿不出来不敢动，一拿就担心税务局找你。放心，公转私有**50种合法路径**——不是走歪门邪道，是每个方法都有法律依据。最常见的好用方案：你给公司提供了什么？有房产可以收租金，有车可以拿油费补贴，有专利技术可以做无形资产摊销。我们给客户定过最经典的方案——**业务分拆+费用合规化**，老板从公司拿钱不但不用交20%个税，公司还能抵25%企业所得税。不是没方法，是你之前的会计没告诉你。」

### 场景二：年底分红个税太高

> 「王总，年底分红个税20%，300万利润分红到手240万，60万交税。你觉得肉疼很正常。但我们给客户做了个**业务架构调整**方案——把老板的一部分业务装进一家小规模公司，企业所得税只要5%（小微企业优惠），分红缓交或用薪酬体系替代，综合税负从40%降到12%。一年省下来的税够给员工发年终奖了。想听详细方案的话，我给你做个测算。」

### 场景三：供应商要私账付款

> 「供应商要求走个人卡，你怕走了不合适，不走没货。这种情况我们做过的方案：**委托支付+合规票据处理**。你公对公付款给一家贸易公司，贸易公司代采后给你开票——三流一致，税务局怎么查都不怕。还有一个路子——**小规模采购公司**，让供应商跟你的小规模公司交易，账走公户解决。公转私不是问题，没有合法依据的私账才是问题。」

## 11 股东不交社保专项话术

### 核心逻辑

老板不给自己交社保 = 省钱vs合规的博弈。
解决方案参考 `07-股东不交社保方案.md`，核心是4种方案的选择。

### 场景一：老板说"我给自己交社保不划算"

> 「你说得对，按高工资交社保确实不划算。但**不交不等于没办法**。我们有4种方案帮你选：

> **方案①**：按最低基数交——合法合规，成本最低，月缴一千多
> **方案②**：走劳务报酬（董事费）——不用交社保，有发票即可
> **方案③**：分红代替工资——顺带解决个税问题
> **方案④**：做业务分拆——老板从对公户拿钱的合规路径

> 你不交社保省下的钱，一年也就万把块。但万一被人举报或者税务查到了，**滞纳金+罚款**够你交好几年社保。这不是让你多花钱，是让你选一个不踩雷的省钱方式。」

### 场景二：公司有员工没交社保被预警了

> 「你之前那个员工投诉没交社保，说明税务已经盯上了。**个税系统和社保系统已经打通了**——你申报个税的人数和社保缴纳人数对不上，系统自动标红。现在补救还来得及：该补的补、该调的调。我们给客户做过**员工结构优化**方案——核心员工交社保，临时工走灵活用工平台，股东走董事费。既合规、又省钱、还能干活。」

## 13 财税顶层思维话术

### 核心逻辑

帮助老板建立财税认知，从"出事了找会计"升级到"提前设计财税架构"。
参考 `09-财税顶层思维.md`。

> 「很多老板找我的时候，第一句话都是：'江姐，我们公司被查了，怎么办？'我说：**你现在花钱请我解决问题，不如一年前花钱让我帮你避免问题。**

> 做企业先搞定这几层：

> **第1层——股权架构**：自然人持股还是法人持股？出事了是烧到个人还是只到公司？差一个X公司架构，天差地别。

> **第2层——资金路径**：公户、私户、微信、支付宝，四五个口子进钱，你管的过来吗？需要做资金归集，把所有钱的口子收进一个体系。

> **第3层——交易结构**：你卖给客户的产品，形式上怎么卖、谁来卖、签什么合同——每一处都有税务成本。交易结构调整好了，税负降一半不稀奇。
>
> 我24年看过太多企业——业务做得很好，账一塌糊涂。不是老板不聪明，是没人告诉他财税要'从设计开始'。**合规成本越低的时候，就是你改造财务体系最好的时候。**」

## 15 续费话术（年度续约）

> 适用场景：服务到期前45天至到期日，分铺垫→面谈→文字跟进三步
> 核心目标：把过去一年的隐形价值显性化，让客户觉得"续费是划算的投资"

### 核心逻辑

```
不是"再交一年钱" → 而是"继续买这份安心"

你的对手不是其他代账公司，
是客户心里那个声音："没出事 = 不需要"

所以要帮客户看见：
```
