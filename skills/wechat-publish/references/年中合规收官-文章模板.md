# 年中合规收官类文章模板

> 适用场景：每年6月中下旬，工商年报+汇算清缴双收官节点
> 首次使用：2026-06-23，用户认可

## 结构模板

```
标题：年中报税收官：你的企业合规，我们心里有数
作者：盈信税务
摘要：120字以内，点明年中节点+服务价值+专业背书
封面：商务办公/财务报表风格

──

## ① [开篇钩子] 两场大考同时交卷
- 节点事实：工商年报6月30日截止 + 企业所得税汇算清缴5月31日截止
- 企业真实感受：数据多、时间紧、怕出错
- 引出价值：我们帮你兜底

## ② [服务体现] 背后有人兜底
- 闫经理团队10年经验
- 具体做了什么事（核对数据、检查异常、提醒缺失）
- 不要抽象说"专业服务"，用流程说话

## ③ [客户关系] 为什么要找你核对数据？
- ✅ 正名话术：'我们发消息提醒核对，这是严谨的工作方式'
- ❌ 不要写：'可能会打扰您'
- 解释原因：留痕防遗漏、做一遍交叉复核

## ④ [实力证明] 数字说话
- 高级会计师领衔（24年经验 / 非挂靠独立通过评审）
- 1000+企业客户、90%转介绍率
- TSC5级代理记账机构

## ⑤ [行动号召] 下半年提醒
- 税务规划提前做
- 政策红利（小微企业优惠、研发加计扣除等）
- 有事群里说，快速响应

──

品牌落款：
盈信企业管理（苏州）有限公司
高级会计师领衔 | 24年财税经验
```

## 调性要点

1. **用流程说话，不喊口号** — 比如"我们做了ABC三件事"而不是"我们很专业"
2. **客户配合要正名** — 催数据不是打扰，是严谨工作方式
3. **数字增强可信度** — 5号前、60天催缴期、7年对比
4. **不卖惨不道歉** — 不需要"求理解"，专业本身就是说服力
