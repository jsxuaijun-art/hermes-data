# English Learning & Translation (absorbed from `english-learning` skill)

This reference documents English learning and translation support, particularly for Chinese users in the finance/tax industry. Absorbed from the standalone `english-learning` skill (archived).

## When to use

- User asks for English-Chinese translation of professional documents
- User needs finance/tax English terminology help
- User wants grammar explanations or sentence parsing
- User needs English writing polishing for emails/reports/contracts

## Translation modes

### Standard translation
```
Input: 翻译 >>> [text to translate]

```

### English learning modes

| Mode | Prefix | Purpose |

| Sentence parsing | `拆句 >>> [sentence]` | Structural analysis |

### Example
```
Input: 财税翻译 >>> 一般纳税人可以开具增值税专用发票
Output: General taxpayers can issue special VAT invoices.
Key note: 增值税专用发票 → "special VAT invoice"
```

## Study advice for finance/tax professionals

1. Read professional English materials first (customs notices, tax bureau docs, IFRS standards)
2. Focus on high-frequency terms: tax/accounting/foreign trade core vocabulary
3. Start with reading: English articles → understand → translate to Chinese → back to English
4. 15 minutes daily > 2 hours weekly

## Useful tools

- Collins Dictionary: https://www.collinsdictionary.com/ (English definitions + examples)
- Ludwig.guru: https://ludwig.guru/ (check natural collocations)
- YouGlish: real-world pronunciation by accent
