|------|------|

| 记账/会计服务类 | 常州安永亚泰会计服务有限公司（安俊林，序号306） |

**核心事实：** 819人中第362号为苏州盈信，前后分别是企事业单位、国企、高校财务负责人。以纯代理记账公司身份独立通过（不挂靠）——全省唯一。

**7年对比：** 从1/819（0.12%）到3/1216（0.25%），代理记账行业通过率虽翻倍但占比仍极低。

## 二、数据提取与分析方法（可复用）

### 工具链
```
WSL2 + Python3 + pypdf + python-docx
```

### 标准脚本模板
```python
from pypdf import PdfReader
reader = PdfReader("/path/to/list.pdf")
text = ""
for page in reader.pages:
    text += page.extract_text()
lines = [l.strip() for l in text.split("\n") if l.strip()]
# 然后按关键词搜索、按序号排序等
```

### 关键坑
- PDF可能只是最终名单，不包含单位性质字段 → 需人工按关键词分类：代账/记账、事务所、国企/事业单位、高校/医院等
- 企业名称含"财务有限公司"（如集团财务公司）≠代理记账公司，需注意区分
- 代理记账公司名称常含：财务代理、财务管理、会计服务、财务咨询、记账等关键词，但不一定直接写"代理记账"
- 扬中市会计服务中心是事业单位非企业，不要误算代理记账公司
- 2018名单标题写"362人"，文件名也写362，但实际有819条记录——说明该PDF可能是某批次而非当年全部

## 六、内容生产提示

当需要生成营销内容时，可以从以下角度提取素材：
1. 代理记账行业的高会通过率（0.12%→0.25%）——制造"稀缺性"锚点
2. 2018年江姐前后左右都是大厂/国企/外企——制造"反差感"
3. 七年时间维度——制造"坚持/深耕"叙事
4. 2025年3家对1216家——制造"行业对比"抓手
5. 所有数据来自官方公开名单（可溯源验证）——制造"可信度"
6. 触发关键词：用户问"高级会计师""高级会计师名单""高会""评审""代理记账公司能不能评高会"等均触发本素材
