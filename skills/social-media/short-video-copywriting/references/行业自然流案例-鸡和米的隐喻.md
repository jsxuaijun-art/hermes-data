# 行业自然流案例 — "鸡和米"的隐喻

> 来源：2026.6.19 江姐提供的 29 秒抖音短视频
> 类型：**行业自然流**（比喻式商业认知）

---

## 完整转写

你去追鸡，鸡就会跑。要是你手里有把米，鸡就会跟着你。但你千万不要沾沾自喜，因为鸡喜欢的从来都不是你，而是你手里的那把米。如果你的手里没有米，那它的眼里就没有你。你要是把它喂饱了，它就会离你而去，还会到处下蛋。所以啊，鸡只能给看，不能给吃。只要你手里有足够多的米，是真是假都有你。

---

## 隐喻映射表

| 原文 | 商业隐喻 | 通用场景 |
|------|---------|---------|
| 追鸡 → 鸡跑 | 追着客户推销 → 客户反感逃跑 | 销售跟进、流量获取 |
| 手中有米 → 鸡跟着你 | 有价值输出 → 客户自然靠近 | 内容营销、价值主张 |
| 鸡喜欢的是米，不是你 | 客户在乎的是你能提供的价值 | 产品定位、差异化 |
| 沾沾自喜 | 不要因为客户跟着你就得意忘形 | 客户留存心态 |
| 喂饱了 → 离你而去、下蛋 | 过度交付/一次性满足 → 客户流失但会传播 | 交付节奏、转介绍 |
| 鸡只能给看，不能给吃 | 保持价值展示 + 适度稀缺 | 获客漏斗、诱饵设计 |
| 足够多米 → 真假都有你 | 价值足够大时，真实性差异被淡化 | 个人品牌、信任建设 |

---

## 结构拆解

**三幕式结构：**

| 段落 | 时间 | 功能 |
|:----|:----|:-----|
| 第一幕 — 冲突制造 | 0-5s | "追鸡跑" vs "有米跟着" → 制造认知冲突 |
| 第二幕 — 反转警示 | 5-15s | "不是喜欢你，是喜欢你的米" → 打破常见认知 |
| 第三幕 — 升华结论 | 15-29s | "给看不能给吃" + "有米真假都有你" → 收尾爆点 |

**节奏控制：**
- 0-5s：快节奏抛双场景对比（黄金三秒钩子）
- 5-15s：慢下来讲道理，制造顿悟感
- 15-29s：加速收尾，三个短句递进上升

## 为何这是一个优秀的行业自然流视频

1. **低门槛载体** — 用"农村养鸡"日常生活画面，覆盖全年龄段
2. **高认知共鸣** — 隐喻直指"客户关系本质"，各行各业都能代入
3. **黄金三秒钩子** — "你去追鸡，鸡就会跑" → 跟常识相反的行为观察，制造好奇
4. **三幕式结构完整** — 冲突→反转→升华，标准爆款节奏
5. **评论区可引发讨论** — "这不就是说的某某行业吗""太真实了" → 自然发酵
6. **可延伸财税场景** — 可后续跟"公司注册客户也是这个理""代账客户留不住？看看你的米是啥"

## 如果转化为财税行业自然流文案

**转化思路**：保持"鸡和米"的比喻结构，把场景换成财税行业

**示例开头（黄金三秒钩子）：**
> "你去追客户注册公司，客户就会跑。要是你手里有张高级会计师证，客户就会跟着你。"

**可融入的财税元素：**
- 米 = 专业资质（高级会计师/TSC5级）
- 鸡 = 中小微企业老板
- 追 = 主动推销/电销陌拜
- 喂饱了 = 做完注册就没人理
- 下蛋 = 转介绍/口碑传播
- 有米 = 持证+年检+可查

**核心警示（结尾爆点）：**
> "所以啊，专业资质只能给客户看，不能一次性给完。只要你手里有足够多的真本事，是真是假都有你。"
