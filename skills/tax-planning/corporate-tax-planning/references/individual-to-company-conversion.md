# 个体工商户转有限责任公司 · 税务筹划参考框架

## 适用场景

客户是个体工商户（小规模纳税人，查账征收），年营收即将超过500万，担心变成一般纳税人后税负增加，询问是否应该转为有限公司。

## 核心逻辑链

```
营收即将超过500万 → 必须变为一般纳税人（无论个体户还是公司）
增值税税率由业务类型决定（贸易13%/服务6%/建筑9%），与主体类型无关
个体户 vs 公司的真正差异 = 经营所得个税(5%-35%) vs 企业所得税(5%-25%) + 分红个税(20%)
```

**关键信息：** 增值税是"平移"的——转不转公司，一般纳税人资格和税率都一样。方案的真正价值在所得税端。

## 信息收集清单（必须逐一确认）

| 信息项 | 必问话术 |
|--------|---------|
| 业务类型 | "您做的是贸易/生产加工，还是服务业？" → 决定一般纳税人税率 |
| 毛利率 | "大概毛利率多少？进项发票能拿到多少？" → 决定增值税实际负担 |
| 年净利润 | "一年下来，扣除所有成本费用后大概剩多少利润？" → 决定个税vs企税差异 |
| 工资发放 | "您现在给自己发工资吗？还是年底直接拿利润？" → 社保和个税筹划 |
| 资金取用习惯 | "日常开销从公司账上走还是从个人账户走？" → 决定"利润不分配"是否可行 |
| 员工情况 | "有没有雇人？社保怎么交的？" → 转公司后社保成本变化 |
| 固定资产 | "有没有车、设备在公司名下？" → 资产过户成本 |
| 客户/供应商 | "客户要专票还是普票？供应商能开专票吗？" → 上下游影响 |

## 关键判断维度

### 维度1：增值税实际负担评估

一般纳税人增值税 = 销项税 - 进项税。**进项充足度决定实际负担。**

| 毛利率 | 进项状况 | 13%增值税的实际负担率 | 核心判断 |
|--------|---------|---------------------|---------|
| <20% | 进项充足 | 约2%-3%（净收入） | **增值税不是问题** |
| 20%-40% | 中等 | 约4%-7% | 可接受 |
| >50% | 进项很少 | 约10%-11.5% | **增值税是真实负担** |

> **关键洞察：** 贸易/生产类企业毛利率<30%时，进项发票覆盖大部分销项税，实际增值税负担很低。客户"怕13%"往往是误解——真实的缴税额是"销项减进项"后的差额，不是全额。

### 维度2：所得税差异评估

以下基于2025年标准税率：

```
个体户：经营所得个税 — 5%~35%超额累进
  年利润30万 → 约4.5万（≈15%）
  年利润50万 → 约10.5万（≈21%）
  年利润100万 → 约28.45万（≈28.45%）
  年利润200万 → 约71.05万（≈35.5%）

有限公司：小微企业企业所得税 — 实际税率5%（年利润≤300万）
  年利润100万 → 5万企税（留公司不分红）
  年利润100万 → 5万+95万×20%≈24万（全分红）
  年利润300万 → 15万企税（留公司不分红）
```

### 维度3：分红决策树

```
利润是否必要拿到个人账户消费？
  ├── 不需要（可留公司运转）→ 只交5%企税 → 强烈推荐转
  │   日常用度通过工资+报销合理支取
  │   大额消费通过公司名义买车/设备
  │
  └── 必须拿回家（无其他资金来源）
      ├── 年利润<100万 → 转公司后全分红≈24% vs 个体户≈28%
      │   差额不大，但仍有约4个点节省
      │
      └── 年利润>100万 → 转公司后全分红≈36%-40% vs 个体户≈28%-35%
          优势缩小，需配合其他筹划
```

## 方案策略核心

### ⭐ "利润不分配"策略（核心卖点）

这是整个方案的关键。必须跟客户讲清楚：

```
日常用度：老板通过工资+报销从公司合理支取（有票可抵）
大额消费：以公司名义买车、买设备（计入公司资产）
年底利润：留存在公司不分配 → 只交5%企税，不触发20%分红个税
未来用钱：可以通过公司借钱给股东（有息借款，利息交税但本金不必）
```

### 话术模板（给客户）

```
老板，增值税这块您不用担心——个体户和有限公司超过500万都要变一般纳税人，
税率跟业务类型走，跟主体类型没关系。

您真正要关注的是所得税：
个体工商户的利润最高要交35%个税（经营所得累进），
而有限公司作为小微企业，利润300万以内只要交5%企业所得税。

如果您把利润留在公司账户不分配——日常开销走工资和报销，
大件用公司名义买——综合税负从35%直接降到5%。
这是最直接的节税空间。
```

### 话术模板2（面对"那分红还是得交20%啊"）

```
对，分红有个20%的个税。但关键是——您一定要现在把这20%拿出来吗？
钱放在公司账户上还是您的钱，公司是您的。
日常开销通过工资和报销就能出来，大额支出用公司名义买。
只有在您确实要把利润拿回家不再用于经营的时候，那20%才触发。
和个体户35%的累进税率比，不管怎么算，都更划算。
```

## 精算模板

> 以下公式供内部使用，代入客户实际数据即可生成数字

```
假设：年营收S万 | 毛利率G% | 年费用F万（不含进货成本）
→ 年净利润 = S × G% - F

个体户（查账征收·经营所得个税）：
  应纳税所得额 = 净利润
  适用表：经营所得个税 5-35%超额累进

有限公司（小微企业）：
  应纳税所得额 = 净利润
  企税 = min(净利润, 100万) × 5% + max(0, 净利润-100万) × 10%
  不分配 → 总税负 = 企税
  全分红 → 总税负 = 企税 + (净利润-企税) × 20%
```

### 速算表（供快速给客户参考）

| 年利润 | 个体户个税 | 公司企税(不分红) | 公司企税+分红 | 最优选择 |
|-------|-----------|-----------------|--------------|---------|
| 30万 | ≈4.5万 | ≈1.5万 | ≈7.2万 | 转公司且不分红 |
| 50万 | ≈10.5万 | ≈2.5万 | ≈12万 | 转公司且不分红 |
| 100万 | ≈28.45万 | ≈5万 | ≈24万 | 转公司且不分红 |
| 200万 | ≈71.05万 | ≈10万(前100)+10万(后100×10%) | ≈68万 | 个体户略优(但差距小) |
| 300万 | 超过个税上限 | 第I类利润15+200×20%≈55万 | 合计≈69万 | 需个案分析 |

> ⚠️ 注：上述速算表基于标准税率，各省可能有地方优惠政策，需按客户实际确认。

## 新增成本评估

### 社保成本（纯新增支出，必须告知客户）

| 缴费方式 | 个体户老板 | 有限公司老板 |
|---------|-----------|-------------|
| 养老+医疗 | 灵活就业≈20%+10% | 职工社保≈24%(单位)+10.5%(个人) |
| 基数 | 可选当地社平60%~300% | 按实际工资(≥最低基数) |
| 年成本(最低) | ≈1.2-1.8万/年 | ≈2.5-3.5万/年 |

> **影响：** 转公司后社保成本可能增加1-2万/年，但相对于节税金额（尤其是利润>50万时）可以忽略。

### 其他潜在成本

- 个体户注销：可能触发查账补税（未申报收入的风险点）
- 资产过户：车辆、设备从个体户转到公司名下的税费
- 银行账户：重新开对公账户的行政成本
- 刻章/章程/注册地址：约2000-5000元一次性成本

## 决策树（快速判断方案方向）

```
客户说"我要转公司"
  ↓
① 业务类型是？
   └── 贸易/生产（13%）→ 查毛利率
        │
        ├── 毛利率<30%（进项充足）
        │   → 增值税不是问题，主攻个税端说服
        │   → 推荐：转公司 + 利润不分配
        │
        ├── 毛利率30%-50%（进项中等）
        │   → 增值税部分真实，但个税节约更显著
        │   → 推荐：转公司 + 优化进项管理
        │
        └── 毛利率>50%（进项很少）
            → 增值税是真实负担，转公司不能解决
            → 建议：先改善进项管理 / 考虑简易计税 / 拆分业务主体
            → 仅当利润>50万时才推荐转（所得税节约覆盖增值税差异）
  │
  ② 利润规模？
   ├── <50万 → 个税与企税差异约5-8万/年，转不转意义有限
   ├── 50-200万 → 最佳转化区间，省税效果明显
   └── >300万 → 企税不再享受小微优惠（25%），优势缩小
  │
  ③ 转公司时机？
   ├── 营收<500万（小规模）→ 随时可转，小规模期间无增值税影响
   ├── 营收接近500万 → 建议在超线前完成转企，避免"成为一般纳税人后才转"的税务衔接麻烦
   └── 营收已超500万 → 也可以转，但需做税务清算+一般纳税人衔接
```

## 资金回流策略（结构化表格）

这是方案落地后客户最关心的实际问题——钱转成公司了，怎么花？

| 方式 | 操作方法 | 个税影响 | 社保影响 | 适用场景 | 注意事项 |
|------|---------|---------|---------|---------|---------|
| **工资** | 给自己发1~2万/月 | 3%~10%累进 | 按实际发 | 日常家用 | 最低基数城居保 vs 按工资交职工社保（成本差约1-2万/年） |
| **报销** | 手机费、交通、差旅、应酬等 | 0% ✅ | 无 | 日常运营开支 | 必须有票、与经营相关、留存消费凭证备查 |
| **公司资产** | 公司名义买车、设备、电脑 | 折旧抵企税 | 无 | 大额消费 | 车辆使用公私分明，避免被认定为"个人消费" |
| **借款** | 公司借钱给股东（签协议） | 利息收入交税，本金不交 | 无 | 临时大额用款 | 年底需归还或转为分红（否则视同分红征20%） |
| **分红** | 税后利润正式分配 | 20% ⚠️ | 无 | 利润确定性不再用于经营 | 最后手段——能用前4种就别分红 |

### 核心原则

```
第一层：工资+报销（日常，每月走）
第二层：公司名义资产（大额，一次性）
第三层：有息借款（临时周转，年终结清）
第四层：分红（最后手段，能不触发就不触发）
```

## 操作时间线（标准流程）

从个体户到有限公司的完整办理路径（假设客户配合度正常）：

```
第1周 — 评估与准备
  ├── 确认个体户工商状态正常（无异常/吊销/冻结）
  ├── 确认税务申报齐全（无逾期未申报）
  ├── 准备股东身份信息、注册资本方案、经营范围
  └── 至市场监管局窗口咨询"个转企"绿色通道（部分地区有）

第2周 — 税务清算（最关键一步）
  ├── 对个体户做税务注销申请
  ├── {{⚠️ 查账征收个体户注销时税务局可能清查近3年账务}}
  ├── 补缴历史欠税、滞纳金、罚款（如有）
  └── 取得《清税证明》

第3-4周 — 个体户注销
  ├── 工商注销公示（简易注销20天/一般注销45天）
  ├── 注销营业执照
  ├── 注销开户许可证、社保账户、公章缴销
  └── 📌 个体户执照缴销后至新公司执照下发前，业务不可断档

第4-5周 — 新公司注册
  ├── 企业名称自主申报
  ├── 工商设立登记（"个转企"通道优先审批）
  ├── 领取营业执照（约3-5个工作日）
  ├── 刻章（公章+财务章+发票章+法人章）
  ├── 开设银行基本户（5-10个工作日）
  └── 📌 注意：公司名称可以与原个体户不同，但建议相似以维持客户识别

第5周 — 税务登记与衔接
  ├── 税务局做税务登记（新办企业流程）
  ├── 一般纳税人资格登记（原个体户已超500万则自动认定）
  ├── 开通电子税务局、数电票
  └── ⚠️ 个体户期间已开票未回款项目：让客户付款至公司账户，附情况说明

第6周 — 资产与人员平移
  ├── 车辆过户（个体户→公司名，需缴二手车交易税≈0.5-1%）
  ├── 设备/存货/无形资产转移（作价入股或销售）
  ├── 客户合同重签（原个体户签订合同→公司名义重新签订）
  ├── 供应商/代理商主体变更通知
  ├── 员工劳动合同重签（与公司签约）
  ├── 社保账户变更（个体户→公司，重新开户）
  └── 📌 商标/知识产权建议转让至公司名下（避免权属争议）

第7周起 — 正式运营
  ├── 公司账簿启用（电子账或软件）
  ├── 发票系统切换确认
  ├── 首次纳税申报（注意申报口径变化）
  └── 正常经营
```

### 常见卡点

| 卡点 | 严重程度 | 应对 |
|------|---------|------|
| 个体户历史账务不清 | 🔴 可能无法顺利注销 | 聘请会计补账后再申请注销 |
| 个体户有未了结诉讼/处罚 | 🔴 注销被驳回 | 先处理完法律事项 |
| 个体户名称不能直接继承 | 🟡 要重新核名 | 新名类似即可，非必须完全一致 |
| 银行开户排队 | 🟡 需等待 | 提前预约，部分银行可加急 |
| 供应商/客户合同因主体变更不愿重签 | 🟢 沟通问题 | 出具情况说明 + 新公司法人签字确认承继原合同义务 |

## 风险分析表（6大风险与应对）

| # | 风险点 | 风险等级 | 触发条件 | 后果 | 应对措施 |
|---|-------|---------|---------|------|---------|
| 1 | 个体户注销查账补税 | 🔴 高 | 个体户历史账务不齐全，曾有隐匿收入 | 补缴税款+滞纳金0.05%/日+0.5-5倍罚款 | 注销前自查补税，主动补税可争取从轻处罚 |
| 2 | 转公司后社保成本上升 | 🟡 中 | 老板从灵活就业参保转为职工社保 | 年增约1.5-2万（按最低基数） | 评估是否从老板一人工资开始，后续逐步规范 |
| 3 | 分红20%成本不可忽视 | 🟡 中 | 利润较大且老板有用钱需求 | 实际综合税负从5%升至24%+ | 优先工资+报销+借款，最后才考虑分红 |
| 4 | 客户/供应商合同主体断裂 | 🟡 中 | 未及时重签合同或未通知对方 | 业务暂停、回款困难 | 提前两个月发函通知，同步签订合同承继协议 |
| 5 | 个体户已开票未回款 | 🟡 中 | 发票已以个体户名义开具，但款项尚未收到 | 客户付款时开票主体与收款主体不一致 | 附情况说明，要求客户付款至新公司账户并注明代收 |
| 6 | 车辆/资产过户税费 | 🟢 低 | 个体户名下有车或设备需过户 | 二手车交易税0.5-1%+代办费 | 评估是否值得过户；如车辆残值低可直接新购 |

## 方案交付检查清单

- [ ] 增值税部分：是否澄清"转不转公司，13%都一样"？
- [ ] 所得税部分：是否给出个体户vs公司的具体数字对比？
- [ ] 利润不分配策略：是否讲清楚"日常工资+报销，大额用公司名义买"？
- [ ] 社保成本：是否明确告知新增的社保支出？
- [ ] 注销风险：是否提醒个体户注销可能触发查账补税？
- [ ] 时机建议：是否给出最佳转企时间窗口？
- [ ] 一次性成本：是否告知代理费、刻章、银行开户等成本？
