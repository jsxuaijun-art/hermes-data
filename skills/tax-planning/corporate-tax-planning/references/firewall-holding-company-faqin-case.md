# 防火墙控股架构 · 夫妻公司案例（2026.6.2）

## 场景描述

自然人A（夫）和B（妻）名下共有4家公司：
- A持有：苏州S桥国际旅行社有限公司（100%）
- B持有：苏州KB虾商务科技有限公司（100%）、苏州ls达汽车服务有限公司（80%）、苏州EU环境技术有限公司（100%）
- ls达汽车服务有外部股东C和D各10%

目标：新设一家控股公司，由控股公司持有各运营子公司股权，实现防火墙隔离效果。

## 核心架构

```
                      ┌──────────────────┐
                      │  A 44% + B 44%    │
                      │  + 第三方 12%     │  ← 一致行动人协议(A+B)
                      │  (控股公司)        │
                      └────────┬─────────┘
                               │
            ┌──────────────────┼──────────────────────┐
            │                  │                      │
       ┌────▼────┐       ┌────▼────┐            ┌────▼────┐
       │ S桥国际 │       │ KB虾   │            │ ls达   │
       │ 100%    │       │ 100%   │            │ 80%    │
       └─────────┘       └─────────┘            └─────────┘
                                            C 10% / D 10%(维持)
                                         ┌────────────┐
                                         │ EU环境     │
                                         │ 100%       │
                                         └────────────┘
```

## 关键设计要点

### 1. 第三方持股比例 ≥12%

| 持股方案 | 防火墙效果 | 控制权 |
|---------|-----------|--------|
| A 44% + B 44% + 第三方 12% | ✅ 强（打破实质一人公司认定） | 一致行动人协议（A+B合计88%） |
| A 50% + B 50%（无第三方） | ⚠️ 中（夫妻公司被穿透风险） | A+B合计控制 |

**Why 12% rather than <10%?** 10%以下法院可能认定为"象征性持股"，仍然穿透认定为实质一人公司。12%以上可主张"非一人公司"。

### 2. 外部股东（ls达）处理策略

| 外部股东关系 | 推荐策略 | 法律工具 |
|------------|---------|---------|
| 紧密（朋友/合伙人） | 策略B：收购100%（含C和D的20%） | 协商转让 |
| 一般/关系不好 | 策略A：收购B的80%，C和D各10%保留 | 控股公司持股80%>50%实现控股 |

**法律关键**：《公司法》(2023)第84条——B向控股公司转让80%股权属于"对外转让股权"，必须书面通知C和D，C和D有优先购买权。30日内未答复视为同意。

### 3. 股权转让税务路径

| 路径 | 个税 | 条件 |
|-----|------|------|
| 平价转让 | 无（增值=0） | 注册资本原值转让，净资产未大幅增值 |
| 非货币性资产投资 | 20%但可5年分期 | 财税〔2015〕41号 |
| 溢价转让 | 差额×20% | 净资产>注册资本 |

- 契税：股权转让不征（财税〔2021〕29号）
- 印花税：转让金额万分之五
- 后续分红：子公司→控股公司免企业所得税（企业所得税法第26条）；控股公司→A/B个人按20%缴纳

### 判例

| 来源 | 裁判要旨 | 对本案的指导 |
|------|---------|-------------|
| 最高院（2019）民终1364号 | 夫妻公司＝实质一人公司，举证责任倒置 | 夫妻股东需自证财产独立，否则连带→引入第三方打破认定 |

## 执行路线图

```
① 确定第三方人选 → 谈妥持股12%条件
② 新设控股公司（注册资本100-500万，5年实缴）
③ 经营范围含"企业管理咨询""股权投资"（不含"投资管理"等需金融牌照范围）
④ 签署一致行动人协议（A+B联合控制）
⑤ 评估各公司净资产 → 确定转让价格
⑥ 通知ls达外部股东C/D → 获取放弃优先购买权声明
    ⚠️ 外部股东反对→只能走策略A（收购B的80%）
⑦ 依次办理股权转让工商变更
⑧ 各子公司修改章程（股东变更为控股公司）
⑨ 建立财产独立管理制度（独立账户/账簿/决策记录）
```

## 注意事项

1. **防火墙不等于绝对隔离**——公司法第23条是防火墙的例外：如果A/B用控股公司逃债，法院会刺破面纱要求连带清偿
2. **财产独立是防火墙的基础**——即使持股架构完善，账户混同/账簿混同/业务混同都会让防火墙失效
3. **必须先做信息核实再做方案**——各公司净资产、外部股东意向、征收方式、营收数据必须逐一确认后才能做精确税务测算
4. **客户沟通话术**：核心卖点是"花最小的代价（引入第三方12%）让防火墙真正生效"，而不是"控股公司能帮你完全隔离"
