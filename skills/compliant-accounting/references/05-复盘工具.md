# 合规账复盘工具包

> 用途：内部复盘、客户回访、续费评估、服务质量管理
> 覆盖：谈单复盘 → 交付月复盘 → 季度客户回访 → 年度续费评估

---

## 一、谈单复盘（每次客户面谈后24小时内）

### 复盘模板

```
客户名称：__________  面谈日期：__________  参与人员：__________

1. 痛点挖掘
   □ 是否挖出客户核心痛点？
   □ 客户对哪些风险指标反应最强烈？
   □ 还有哪些潜在风险没来得及说？

2. 团队配合
   □ 规划师与代账会计配合是否到位？
   □ 现场是否及时报价、逼单？
   □ 下次配合有哪些需要调整？

3. 价格博弈
   □ 客户付费能力判断（高/中/低）
   □ 现场报价后客户反应（感兴趣/犹豫/拒绝）
   □ 确定的底价是否给到位？

4. 合作意向
   □ 客户真实合作意图还是套方案？
   □ 如果真要合作：重点讲服务价值 vs 讲价格优惠
   □ 如果套方案：如何引导到签约环节？

5. 下一步计划
   □ 二次跟进时间：____月____日
   □ 跟进负责人：__________
   □ 需要准备的补充材料：__________
```

### 谈单分类跟进策略

| 客户类型 | 跟进策略 | 话术重点 |
|----------|----------|----------|
| **价格敏感型**（嫌贵未签） | 次日负责人亲自电话让价 | "我也不赚您钱，给您一个本钱XXX元" |
| **需商量型**（要和股东商议） | 次日跟进商议结果 | "您和股东商议怎么样了？我这月就想给您开始服务" |
| **非决策人型**（不是老板） | 预约与老板见面时间 | — |
| **犹豫对比型**（对比同行） | 对比盈信差异化优势 | 高级会计师+16年经验+苏州上海双城 |
| **敲定执行型**（要具体方案） | 专家老师出具体合规方案 | 「7日内出方案讲解」 |

| **总分** | **100%** | | |

### 流失预警评估表

| 预警信号 | 风险等级 | 应对措施 |
|----------|----------|----------|
| 超过2次回访未回应 | 🔴 高 | 负责人直接电话联系 |
| 客户投诉响应不及时 | 🔴 高 | 会计+主管上门道歉，承诺改进 |
| 客户对报价有意见 | 🟡 中 | 评估是否调整服务方案 |
| 客户业务量明显减少 | 🟡 中 | 了解经营情况，评估是否需要调整服务 |
| 客户被同行联系 | 🟡 中 | 强化服务差异化展示 |
| 3个月无主动沟通 | 🟢 低 | 安排常规回访，保持存在感 |

### 续费跟进SOP

```
续费到期前3个月：
  └ 季度回访，了解满意度，发现隐患及时处理

续费到期前2个月：
  └ 出具年度服务价值报告
  └ 与客户沟通来年服务方案

续费到期前1个月：
  └ 提交续约方案和报价
  └ 如有存量代账折扣，计算并体现在报价中

续费到期前15天：
  └ 跟进签约进度
  └ 处理客户异议

续费到期前7天：
  └ 如未签约，启动流失预警
  └ 负责人介入挽留

续费到期日：
  └ 如签约 → 服务无缝衔接
  └ 如未签约 → 启动服务退出流程（资料交接单、后续安排）
```

## 五、KPI考核参考（内部团队管理）

| 考核指标 | 计算方式 | 目标值 | 权重 |
|----------|----------|--------|------|
| 做账及时率 | 月5号前完成 × 100% | ≥ 95% | 20% |
| 申报准确率 | 无差错月 / 总月数 × 100% | 100% | 25% |
| 风险发现数 | 每季度发现风险点数量 | ≥ 3个/季度/客户 | 15% |
| 客户满意度 | 回访评分均值 | ≥ 4.5分 | 20% |
| 续约率 | 续约客户 / 到期客户 × 100% | ≥ 85% | 20% |
