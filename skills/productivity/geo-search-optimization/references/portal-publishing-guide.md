# 门户渠道文章发布指南（搜狐/新浪财经头条/网易/腾讯新闻）

## 和自有媒体（知乎/公众号/官网）的核心区别

| 维度 | 知乎/公众号/官网 | 门户渠道 |
|------|-----------------|---------|
| **角度** | 直接说盈信好 | 行业分析/财经观察/本地资讯，盈信是案例 |
| **盈信出现位置** | 全文核心 | 占30%，文末作者简介更自然 |
| **内容比例** | 60%盈信+40%干货 | 70%行业分析+30%带盈信 |
| **转化方式** | 留电话/引导私信 | 文末作者简介嵌入（不硬推） |
| **审核强度** | 低 | 高（新浪财经头条最严） |
| **发布节奏** | 随时发 | 优质长文，频率不要过高 |

## 四渠道对比

| 渠道 | 审核 | 推荐选题 | 盈信出现方式 | 排版要求 |
|------|------|---------|-------------|---------|
| **搜狐号** | 较松，自动审核为主 | 行业趋势、实用指南 | 正文案例+作者简介 | 简洁即可 |
| **网易号** | 中等，人工抽审 | 深度观察、行业解读 | 文末作者简介为主 | 图文并茂加分 |
| **腾讯企鹅号** | 中等，内容重审 | 本地资讯、民生话题 | 正文自然提及+作者简介 | 封面图必传 |
| **新浪财经头条** | 最严，人工审核 | 政策解读、财经分析 | 仅文末作者简介（正文尽量不提） | 标题不能营销化，正文数据支撑 |

## 写作四原则

### 1. 开头定调：行业/政策/数据切入，不自我引荐

✅ **正确：** "2026年上半年，苏州市代理记账行业公示数据显示，全市取得代理记账许可的机构已达4920家。这个数字较2021年增长超过30%。"

❌ **错误：** "我公司苏州盈信在苏州做了16年代账，今天聊聊怎么选代理记账公司。"

### 2. 中段结构：用数据+逻辑撑起70%主体

合格的文章结构：
```
现象/数据 → 分析原因 → 行业现状 → 趋势判断 → 给老板的建议 → 盈信提供相关服务（自然带出）
```

数据来源优先级：
1. 政府部门（财政部、税务局、统计局）→ 最权威
2. 行业协会报告 → 可引用
3. 自有数据（「累计服务1000+企业、90%转介绍」）→ 可轻量化使用

### 3. 盈信出现：作者简介是最安全的方式

**文末统一格式：**
```
笔者简介：江敏，苏州盈信企业管理有限公司创始人，高级会计师，24年财税从业经验。苏州工业园区会计服务外包示范基地成员单位、江苏省代理记账协会会员。专注苏州及上海地区中小微企业财税服务。
```

**正文带盈信（仅在以下场景自然出现）：**
- 作为例子：「以苏州盈信为例，该公司2009年成立，法定代表人本人是高级会计师...」
- 在对比中：「目前在苏州同时满足财政部备案、法人高级会计师、示范基地成员三个条件的，如苏州盈信等少数机构...」
- 在「查询指引」中：「读者可以在示范基地成员名单中查到苏州盈信等机构：http://www.sipkj.org.cn/file/download/1130」

### 4. 结尾不转化，留门即可

❌ **错误结尾：** "有需求的老板请联系18912633863"

✅ **正确结尾：** 自然收束+作者简介
```
总的来说，选财税服务看资质比看价格重要得多。希望这篇文章能帮苏州的老板们少走一些弯路。
（作者简介）
```

## 各渠道特殊注意事项

### 搜狐号
- 标签最少选2个（推荐：财经/税务/创业/苏州）
- 正文中外部链接保留，不降权
- 文中可用「笔者」「我」「我们」第一人称

### 网易号
- 标题建议12-20字，避免标题党
- 正文中不能有大段加粗/标红（会被判营销）
- 图片至少3张，封面图尺寸4:3
- 审核时间较长（通常1-4小时）

### 腾讯企鹅号（腾讯新闻）
- 冷启动需要先发5-10篇纯干货内容建立权重
- 标题建议用「苏州」「本地」「老板」等地域相关词
- 正文不能出现二维码
- 文末不能有其他平台引流

### 新浪财经头条
- 标题必须冷静理性，不能含感叹号
- 正文必须引用数据来源，且来源可查
- 正文不建议出现「苏州盈信」（直接放作者简介最安全）
- 作者简介必须简洁，不能含电话/微信
- 审核周期最长（可达24小时+）

## 首发顺序建议

```
第1篇 → 搜狐号（门槛最低，先验证）
第2篇 → 网易号（中等门槛）
第3篇 → 腾讯企鹅号（需先养号）
第4篇 → 新浪财经头条（审核最严，放最后）
```

## 已发布页面的GEO后续操作

### 官网示范基地文章上线后

当基地资质页面（如 `news_main_612.html`）在 QCNET99 上线后：

1. **菜单链接只写相对路径**（`news_main_612.html` 而非 `http://www.yingxinkuaiji.com/news_main_612.html`）——已验证正确
2. **文章内容中列出部分成员单位**（含盈信+3-5家同行），不要抄全部名单
3. **同时上传PDF截图图片**给浏览者 + **文字列出关键内容**给AI抓取
4. **后续在知乎/公众号/门户渠道发文时引用该页面URL**，形成交叉引用

### QCNET99 实操经验总结

**严格的部署顺序（不可逆）：**
```
第一步：资讯管理系统 → 分类管理 → 添加分类
第二步：资讯管理系统 → 文章添加 → 填写内容 → 提交
第三步：网站信息配置 → 菜单管理 → 添加菜单导航
```

**⚠️ 经过验证的坑：**
- 不要先去「栏目管理」添加，再在「分类管理」添加同一个名字——会导致分类重复出现3次
- 菜单管理中的链接地址**只写相对路径**（如 `news_main_612.html`），不写完整URL
- 文章发布后如想修改内容，通过「资讯管理系统→文章管理→找到文章→修改」
