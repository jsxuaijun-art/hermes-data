# 首次试运行案例：南通体育用品有限公司

## 场景描述（2026.5.13）

客户：南通一家体育用品有限公司，股东三人
门店：苏州某商业综合体×3家个体户
模式：有限公司统一采购阿迪达斯/耐克产品 → 分售给个体户 → 个体户零散零售

## 用户诉求

"全部来一遍"——行业研究 + 税务筹划 + 合规检查

## 执行进度（截至当前对话）

已完成：模块一（信息收集，已确认6项，待确认6项）
待进行：模块二（行业研究）→ 模块三（法规采集）→ 模块四（筹划）→ 模块五（合规）

## 待确认的关键信息

1. 有限公司全称+成立时间
2. 个体户征收方式（查账/核定）
3. 各主体营收范围
4. 增值税处理（有限公司→个体户开票情况）
5. 房租与员工归属主体
6. 已享受的税收优惠

## 后续方向

完成信息收集后按顺序执行：
行业研究 → 法规采集（重点：体育用品零售+关联交易+个体户政策）→ 筹划方案（有限公司-个体户架构优化）→ 合规检查
