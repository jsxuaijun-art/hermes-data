# 推荐 IMA 笔记本结构（财税服务公司）

> 适用于代理记账、税务合规、高端会计服务类企业。
> 用户：苏州盈信企业管理有限公司，客户为苏州/上海中小微企业老板。

---

## 一、笔记本列表（9个）

```
📁 全部笔记（根目录）
├── 📁 客户方案          ← 客户项目归档
├── 📁 财税政策库         ← 政策原文+解读
├── 📁 内部SOP           ← 团队操作标准
├── 📁 营销素材           ← 内容资产仓库
├── 💡 灵感记录           ← 灵感捕捉（单篇笔记持续追加）
├── 📁 模板库            ← 可复用的交付模板
├── 📁 个人品牌           ← 高级会计师相关
├── 📁 学习笔记           ← 持续专业教育
└── 📁 日常记录           ← 会议纪要、待办、临时备忘
```

## 二、各笔记本明细

### 1. 📁 客户方案

**用途**：每个客户一个独立笔记，归档架构方案、税务筹划、合规分析等交付物。

| 笔记标题 | 内容说明 |
|----------|----------|
| `南通咖萌_母子公司+分公司架构方案` | 架构重组方案+验算+待办 |
| `爽煌_个体户注销+差额补税方案` | 注销前补税估算+补救成本 |
| `{客户名}_{项目类型}` | 按此命名规范持续扩展 |

**规范**：
- 标题格式：`{客户简称}_{项目类型}`（如 `XX公司_股权转让税务分析`）
- 内容包含：方案要点、数据验算、待办事项、讨论记录
- 重要方案本地 .docx 同步保存到：`桌面\客户方案\{客户名}\`

### 2. 📁 财税政策库

**用途**：归类整理财税政策原文+解读，方便快速检索。

| 笔记标题 | 内容说明 |
|----------|----------|
| `小微企业税收优惠汇总（2026版）` | 小型微利企业、增值税减免等 |
| `增值税专题_留抵退税+加计抵减` | 政策要点+适用条件 |
| `企业所得税专题_扣除标准+优惠备案` | 限额扣除、加计扣除、优惠备案 |
| `个人所得税+社保专题` | 个税汇算、社保基数、年终奖 |
| `小规模纳税人专题` | 免税额度、开具发票规范 |

**规范**：
- 按税种/业务类型拆分，不混在一篇
- 附政策文号+生效日期+官方链接
- 政策更新时在末尾追加，标注更新时间

### 3. 📁 内部SOP

**用途**：团队操作标准文档，新人培训+老员工查漏。

| 笔记标题 | 内容说明 |
|----------|----------|
| `新客户建群标准流程` | 建群话术+身份锚点+配合事项 |
| `代理记账月度操作清单` | 每月1日~15日逐项清单 |
| `申报日历+截止日期提醒` | 各税种申报截止日 |
| `会计交接/离职交接清单` | 资料移交+系统权限清理 |
| `开票操作规范` | 数电票、税控盘操作 |

**规范**：
- 步骤清晰，可逐条打钩
- 含截图/模板链接
- 定期 review 更新

### 4. 📁 营销素材

**用途**：短视频文案、话术、客户案例、内容选题。

| 笔记标题 | 内容说明 |
|----------|----------|
| `短视频文案_行业自然流` | 讲故事带业务，润物细无声 |
| `短视频文案_纯广告` | 精准筛选，有需求者直接私信 |
| `短视频文案_趣味破播` | 轻松有趣，泛人群传播 |
| `短视频文案_范话题` | 与财税无关的泛话题爆款 |
| `高级会计师身份锚点话术` | 评审经历+行业揭秘+反差制造 |
| `客户见证_可用案例` | 隐去敏感信息的客户成功案例 |
| `选题清单_待拍摄` | 已策划未拍摄的选题池 |

**规范**：
- 按文案类型分笔记，不混放
- 每条文案标注「状态：已拍/待拍/废弃」
- 含评论区话术+转化链路设计

### 5. 💡 灵感记录（特殊）

**用途**：单篇笔记，持续追加。所有灵感、想法、突发的商业构想全部汇总至此。

**标题**：`💡 灵感记录`

**内容格式**（每条追加）：
```
---

[2026-05-15 16:45] #财税 #产品
想法摘要：代理记账行业的AI自动生成凭证工具
详细内容：（用户原始想法全文）
后续动作：找技术聊聊成本
讨论记录：
- 2026-05-16 补充：需要考虑现有的会计人员接受度
```

**规范**：
- 一篇笔记管所有灵感，不建多篇
- 每条灵感带：时间戳 + 标签 + 摘要 + 详细内容
- 可选：后续动作 + 讨论记录
- 标签分类：#财税 #产品 #营销 #技术 #管理 #客户 #个人品牌

### 6. 📁 模板库

**用途**：可复用的交付模板，直接复制修改即可复用。

| 笔记标题 | 内容说明 |
|----------|----------|
| `税务合规审查报告模板` | 通用框架，填空式输出 |
| `企业架构分析报告模板` | 母子公司/分公司税务对比模板 |
| `年度经营分析报告模板` | 收入/成本/税负分析框架 |
| `报价方案模板` | 代理记账/高端服务报价结构 |
| `客户接洽信息收集清单` | 首次接触需收集的资料清单 |

**规范**：
- 模板含占位符 `{{客户名}}`、`{{金额}}` 等
- 每份模板附使用说明

### 7. 📁 个人品牌

**用途**：高级会计师相关的个人品牌资产。

| 笔记标题 | 内容说明 |
|----------|----------|
| `高级会计师评审经历` | 评审过程、关键节点、亮点提炼 |
| `行业揭秘素材（评审内幕）` | 代理记账行业 vs 国企大厂的评审反差 |
| `行业分享/演讲内容` | 演讲稿、PPT大纲、会后复盘 |
| `媒体采访/合作记录` | 采访问题+回答稿 |
| `个人简介多版本` | 适合不同场景的自我介绍（1min/3min/书面） |

**规范**：
- 个人品牌内容注意统一调性
- 避免在不同平台出现矛盾的自我介绍

### 8. 📁 学习笔记

**用途**：持续专业教育（CPE）、政策学习、行业知识积累。

| 笔记标题 | 内容说明 |
|----------|----------|
| `2026年新政策学习笔记` | 年度内新发布的重要政策摘要 |
| `行业会议/培训笔记` | 参会记录、要点、可落地的启发 |
| `实务难题解决方案` | 遇到的疑难杂症+解决路径 |
| `读书笔记_财税管理类` | 阅读的专业书籍精华摘录 |

### 9. 📁 日常记录

**用途**：会议纪要、待办事项、临时备忘，不归入上述大类的内容。

| 笔记标题 | 内容说明 |
|----------|----------|
| `2026年5月周会纪要` | 按月或按周组织 |
| `待办事项` | 跨项目的待办清单 |
| `临时备忘` | 不成熟但需记录的信息 |

## 三、存储规范

### 双存规则（重要文档）

| 文档类型 | IMA 位置 | 本地位置 |
|----------|----------|----------|
| 客户税务方案 | `客户方案` 笔记本 → 对应客户笔记 | `桌面\客户方案\{客户名}\{文件名}.docx` |
| 短视频文案 | `营销素材` 笔记本 → 对应类型笔记 | `桌面\营销素材\{类型}\{文件名}.docx` |
| 内部SOP | `内部SOP` 笔记本 → 对应笔记 | `桌面\内部SOP\{文件名}.docx` |
| 灵感/想法 | `💡 灵感记录` 笔记追加 | 不存本地 |

### 命名规范

- **IMA 笔记标题**：`{客户名}_{项目类型}`（如 `南通咖萌_架构方案`）
- **本地文件名**：`{日期}_{客户名}_{项目类型}.docx`（如 `20260515_南通咖萌_架构方案.docx`）
- **本地目录**：每日方案次日整理归档

---

## 四、维护规则

- **笔记本结构一经确认不做频繁变动**——变动会打散笔记归属导致检索困难
- **定期归档**：每季度 review 一次，废弃笔记标注 `[已归档]` 前缀
- **标签体系**（IMA 正文中手动标注）：
  - `#客户` 客户相关
  - `#政策` 政策解读
  - `#SOP` 内部流程
  - `#营销` 内容素材
  - `#学习` 专业提升
  - `#灵感` 想法记录
  - `#模板` 可复用框架
