# 江姐财税全案智能体 — COZE System Prompt

## 用途

这份 prompt 用于在扣子(COZE)平台创建一个完整的税务筹划与合规方案设计智能体。

## 文件结构

```
~/.hermes/skills/coze-tax-agent-prompt/
├── SKILL.md                          # 本文件 — 使用说明
├── references/
│   ├── system-prompt.txt             # 完整System Prompt（1.4万字，可直接贴入COZE）
│   ├── prompt-v1-coze.txt            # 智能体一（COZE原版）复盘记录
│   └── prompt-v2-self.txt            # 智能体二（自写版）复盘记录
```

## 使用方式

### 场景A：首次创建COZE智能体
1. 加载本技能：`skill_view('coze-tax-agent-prompt')`
2. 读取 system-prompt.txt 内容
3. 在COZE中新建智能体 → 将内容粘贴到System Prompt区域
4. 参照prompt文档配置对应的工具（搜索、文件上传等）

### 场景B：需要更新优化
1. 加载本技能
2. 告知江姐的修改需求
3. 我会基于原版进行增删改，输出新版system-prompt.txt
4. 同步覆写COZE中的prompt

### 场景C：需要拆分/定制子智能体
1. 加载本技能
2. 告知需要哪个模块（如：单独做一个"合规检查智能体"）
3. 我会从全案中抽出对应模块，精简后输出

### 场景D：在Hermes引擎中直接运行（江姐2026.5.13确认可用）

本技能完整System Prompt已设计为可直接由Hermes Agent按流程执行。加载技能后，通知用户"已加载，开始跑流程"即可直接运行。

**运行流程（按system-prompt.txt九大模块顺序）：**

1. **模块一**：信息收集 — 分3阶段提问，支持跳过/部分回答
2. **模块二**：行业研究 — 搜索行业研究报告，输出Markdown报告
3. **模块三**：法规采集 — 搜索政府网站，带文件号引用
4. **模块四**：税务筹划 — 输出2-3个备选方案，含风险提示
5. **模块五**：合规检查 — 输出风险等级+整改路径
6. **模块六**：财报分析 — 支持上传分析+老板视角解读
7. **模块七**：对话迭代 — 最多3轮方案修改
8. **模块八**：企业生命周期导航 — 按初创/成长/成熟/收缩/退出分流
9. **模块九**：双城政策适配 — 苏州/上海政策差异自动识别

**⚠️ 关键实践规则（2026.5.13 江姐定）：**

- 表格必须用 workbuddy-output 技能的 box_maker.py 生成，禁止Markdown表格
- 法规引用必须列明文件号（如：财税〔2023〕XX号）
- 税务筹划方案每项必须附带风险提示，不得只讲好处
- 所有方案末尾加「⚡ 以上方案基于您提供的信息生成，实际执行前建议进一步确认」
- 用老板语言，不堆砌会计术语
