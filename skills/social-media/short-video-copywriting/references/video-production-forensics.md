# Video Production Forensics — 实际分析案例

## 背景

2026.6.19 江姐发来一个 29秒 MP4 视频文件（抖音改版算法解读类），要求「拆解制作风格、工具、手法，以便复刻」。

## 完整分析链路

### Step 1: 元数据提取

```bash
ffprobe -v quiet -print_format json -show_format -show_streams <视频路径>
```

**关键发现：**

| 字段 | 值 | 含义 |
|------|-----|------|
| `encoder` | Lavf58.76.100 | 被 FFmpeg 重编码（不保留原始制作工具信息） |
| `comment` | vid:v0300fg10000d8n61ufog65v7uubtaig | **微信视频号 ID**（`v0300` 前缀 = 腾讯视频编码规则） |
| `codec_name` | h264, High@L5.0 | 标准在线视频，非 ProRes/RAW |
| `width×height` | 1920×1080 | 全高清 |
| `bit_rate` | 518kbps (video) + 62kbps (audio) | **极低**（29秒仅2.1MB），明显经平台二次压缩 |
| `avg_frame_rate` | 8190/289 ≈ 28.33fps | 非标准帧率，非原始拍摄 |
| `vendor_id` | [0][0][0][0] | 无原始设备信息（被抹去） |
| 无苹果/安卓 metadata | — | 不是直接导出的成品，是下载品 |

**推断：** 非原始导出文件，是从微信视频号/抖音下载后再经 FFmpeg 重编码。

### Step 2: 画面属性分析

取帧并分析像素统计数据：

```bash
# 场景切换检测
ffmpeg -i <videopath> -filter:v "select='gt(scene,0.1)',showinfo" -f null - 2>&1 | grep "pts_time:"
```

**像素数据解读：**

- Y（亮度）均值 85-93/255 → 暗色背景
- U/V（色度）均值 128/128 → 中性灰（无彩色信息）
- U/V 标准差 0.5-2.0 → **极低**，画面接近黑白/完全去色
- 场景切换点仅出现在：0.1s, 2.0s, 5.0s, 12.2s, 22.1-22.6s → **5-6次切换**

**推断：** 暗色底 + 白字文字知识卡片风格（去色/黑白画面）。不是真人出镜，不是实拍。

### Step 3: 关键帧分析

```bash
ffprobe -v quiet -select_streams v -show_entries frame=pict_type <videopath>
# 结果：5 I-frames, 227 P-frames, 587 B-frames
```

**GOP 解读：**
- 29秒视频只有 5 个 I 帧（关键帧）→ 约每 5.8 秒一个新画面
- 绝大多数是 B/P 帧 → 画面极度静态，只在文字切段时有微小变化

**推断：** 5次画面切换 = 5-6个知识卡片段落。每个卡片停留约5-6秒。

### Step 4: 工具链还原

| 证据 | 结论 |
|------|------|
| 格式 h264 + AAC + 极低码率 + 无原始工具 metadata | 非专业软件制作后直接导出 |
| 暗底去色文字卡片风格 | 剪映「知识卡片」模板 / Canva 文字动画模板 |
| 语音大概率 TTS（29秒约600-800字，语速偏快） | 剪映自带朗读 / 魔音工坊 |
| 配乐需求（知识科普类通常有轻BGM） | 剪映音乐库 / 抖音收藏 |

**最终制作链路推断：**

```
文案编写（600-800字，分5段）
  → 剪映/Canva 套文字卡模板（暗底+大标题+关键词高亮）
  → TTS 配音（语速约250-300字/分）
  → 加背景BGM（轻鼓点/知识感）
  → 导出 1080p → 发微信视频号/抖音
  → 平台压缩 → 从平台下载 → FFmpeg 重编码
```

### Step 5: 可复制性评估

✅ **高度适合财税科普类内容** — 纯文字卡片形式：
- 无需出镜、无需拍摄设备
- 文案即一切（适合江姐的强文案能力）
- 剪映/Canva 都有现成模板
- 制作成本极低（10-15分钟一条）

## 注意事项

1. **区分「内容分析」与「制作分析」** — 用户给的视频需先确认分析目的
2. **FFmpeg 重编码后原始工具信息丢失** — 工具推断基于画面特征和经验，非100%确定
3. **色彩分析技巧** — 无 vision API 情况下，YUV 像素统计是判断画面类型的可靠替代方案
4. **comment 字段** — 微信视频号视频的 `vid:v0300fg...` 是腾讯视频ID格式，可作为来源判定依据
