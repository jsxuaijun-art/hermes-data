# 客户沟通话术（Client Communication）

## 适用场景
- 好消息通知（评上A级纳税人、退税到账、政策利好等）
- 日常关系维护（拉近距离、展示专业度）
- 服务到期/变更提醒
- 节日/生日/特殊日子问候
- 新政策解读推送

## 核心原则

### 黄金结构（三段式）
1. **好消息前置** — 先说客户关心的结果，营造积极情绪
2. **价值解释** — 用客户听得懂的话说明"这对他意味着什么"
3. **自然收尾** — 归功于合作 + 为后续互动留口子

### ⚠️ 话术红线（已整合用户偏好）
- 先表扬客户公司底子好 → 再说咱们配合得好 → 最后落到持续合作
- 不能显得"邀功"，要让客户感觉这是"双赢"
- 不贬低同行，不踩低价（用"我们坚持专业路线"替代"他们低价没质量"）
- **默认风格：中和版** — 介于简短版和加强版之间，既不过于精简也不过于冗长。除非用户指定，否则优先输出"先给结论→再解释价值→最后收尾"的三段中和版

### 发送时机
- 好事趁热，当天发
- 配图（截图/证书）增加冲击力
- 客户回复后顺带一句"最近有没有新需求/政策要关注"，延续话题

### 补充场景：解释复杂服务为什么贵（2026.6.17 9810实战沉淀）

当客户觉得报价比普通代账贵太多时，用"复杂度对比 + 工作量倍数 + 三档分层"的结构：

```
「9810 海外仓和普通公司完全不是一个量级——

普通公司记账：票到了→做账→报税，一个流程就完事。
9810 海外仓：货出去了 → 还没卖 → 同一批货分 N 个月卖完 → 
报关一次，退税分批 N 次申报 → 外汇回款混在一起要匹配 
→ 库存月月核对 → 超期要预警……

工作量是我们的普通代账客户的 3-5 倍。

所以分三档：
  • 基础版（不退税+核定征收）：8,000-12,000元/年
  • 标准版（退税+台账管理）：12,000-18,000元/年  
  • 高端版（全流程+专人对接）：18,000-30,000元/年

不是想收贵，是这活确实费时费力。按普通代账 3,000 接，
我做亏本，你也别指望我用心盯预警——到头来你损失更大。」
```

**核心：** 不说"因为复杂所以贵"，而是"因为花3-5倍时间所以贵"——用工作量倍数代替抽象描述。

### 补充场景：三档报价沟通模式（可复用框架，不限于9810）

```
当需要向客户解释不同价位时：
  基础版：把"最便宜"换成"满足基本需求"
  标准版：标注"大部分客户选这个"  
  高端版：说"有问题随时能找到人"
  
话术结构：
  "分三档：第一档XX元，够用；第二档XX元，大部分人选的；
  第三档XX元，一对一专人服务。你先看看哪个合适。"
```
