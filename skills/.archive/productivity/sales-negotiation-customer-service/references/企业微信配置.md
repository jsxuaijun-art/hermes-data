### 上传文件
需要 multipart/form-data 上传。使用Python urllib:
```python
import json, urllib.request

def upload_file(token, filepath):
    boundary = "----WebKitFormBoundary7MA4YWxkTrZu0gW"
    with open(filepath, "rb") as f:
        data = f.read()
    body = (f"--{boundary}\r\n"
            f'Content-Disposition: form-data; name="media"; filename="{os.path.basename(filepath)}"\r\n'
            f"Content-Type: application/octet-stream\r\n\r\n").encode() + data + f"\r\n--{boundary}--\r\n".encode()
    
    req = urllib.request.Request(
        f"https://qyapi.weixin.qq.com/cgi-bin/media/upload?access_token={token}&type=file",
        data=body,
        headers={"Content-Type": f"multipart/form-data; boundary={boundary}"}
    )
    return json.loads(urllib.request.urlopen(req).read())
```

或直接 `curl -F media=@file.pptx "URL"` 但本机环境可能被安全策略拦截。

### 发送消息给用户
```python
def send_msg(token, touser, msgtype, content):
    payload = {"touser": touser, "msgtype": msgtype, "agentid": 1000036}
    if msgtype == "text":
        payload["text"] = {"content": content}
    elif msgtype == "file":
        payload["file"] = {"media_id": content}
    
    req = urllib.request.Request(
        f"https://qyapi.weixin.qq.com/cgi-bin/message/send?access_token={token}",
        data=json.dumps(payload, ensure_ascii=False).encode("utf-8"),
        headers={"Content-Type": "application/json"}
    )
    return json.loads(urllib.request.urlopen(req).read())
```

## 常见错误

| errcode | 含义 | 解决 |
|---------|------|------|
| 60020 | IP不在白名单 | 企业微信后台 → 应用管理 → 企业可信IP |
| 40014 | token过期 | 重新获取（有效期2小时） |
| 40003 | 无效的userid | 检查用户ID拼写 |
| 41001 | 缺少access_token | 检查token是否为空 |
| 42001 | access_token超时 | 重新获取 |
