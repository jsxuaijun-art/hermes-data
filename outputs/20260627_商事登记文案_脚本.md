## 二、三条深化拍摄脚本

### 📍 脚本 #1：《注册公司，千万别图便宜》

**时长：** 60-75秒 | **风格：** 直面镜头口播 + 中景  
**BGM：** 轻节奏感（Lo-fi beat或轻鼓点）

**口播文案：**

```
【0-5s】韩红走冯小刚的面儿，看的是她值不值得那个位置。
你注册公司选代办，看的也是——值不值得。

【5-15s】有些老板来问我第一句话就是：
"江姐，哪家便宜？"
我说句实话——公司注册这个事，便宜没好货，好货不便宜。

【15-30s】便宜的中介，300块帮你注册。
流程是走完了，但——经营范围写错了，
税务类别选漏了，后面变更注销全是坑。
等你想改的时候，发现已经多花了十倍的钱。

【30-45s】我做了16年，见过太多这样的案例了。
所以我跟客户说：注册公司的钱不能省。
因为你省下的每一分钱，后面都要加倍还回去。

【45-55s】你在苏州创业，是为了赚钱，
不是为了省那几百块注册费。

【55-70s】选对代办，一次搞定，后面顺风顺水。
16年来，我们经手的公司注册上千家，
每一家我都盯着办完。
需要注册、变更、注销的，找江姐就对了。

【70-75s】字幕：找江姐，注册/变更/注销一条龙
```

**分镜：** 00s中景→15s近景→30s中景手势→45s近景→55s中景微笑→70s字幕

### 📍 脚本 #2：《你开公司的第一步就走错了》

**时长：** 70-85秒 | **风格：** 叙事+观点  
**BGM：** 温暖系钢琴/吉他

**口播文案：**

```
【0-5s】韩红走面儿，走对了是《天路》，
走错了就是《抓特务》。

【5-10s】你注册公司也是一样的道理——
第一步走对了，后面全顺；
第一步走错了，后患无穷。

【10-25s】很多老板注册公司的第一步就错了。
经营范围写得太窄，后面想扩大业务还得变更；
注册资本写得太高，认缴期到了没钱缴；
股东结构没想清楚，改的时候要全体签字。

【25-40s】所以我给大家做注册前，都会先做个"公司体检"：
你的业务模式是什么？
预计年营收多少？
有没有合伙人？
要不要开票？

【40-55s】这些分析清楚了再注册，一次到位。
而不是注册完了发现问题，再花冤枉钱去改。

【55-70s】所以说，注册公司不是填个表那么简单。
你选什么类型、写什么经营范围、注册资本多少，
每一件事都影响你后面几年的税负和成本。

【70-85s】你在苏州、上海开公司，想注册个体户还是有限公司？
拿不准的，找江姐聊聊，
免费给你做公司体检，避开80%的坑。
```

**分镜：** 00s中景→10s微微皱眉→25s手指列举→40s摊手轻松→55s点头→70s微笑CTA

### 📍 脚本 #3：《公司注册后这些事你做了吗？》

**时长：** 75-90秒 | **风格：** 揭秘型，打破认知  
**BGM：** 轻快节奏，带悬疑感

**口播文案：**

```
【0-5s】韩红走完面儿就走了。
你呢？公司注册完，你以为就完事了？

【5-15s】很多老板拿到营业执照就以为万事大吉了。
我告诉你——大错特错。
注册完只是开始，后面还有一堆事等着你。

【15-30s】刻章备案，办了吗？
银行开户，开了吗？
税务登记，登了吗？
社保公积金，办了吗？
发票申领，申了吗？
行业许可证，有吗？
——每一项都有时间节点，错过了就是罚款。

【30-45s】我们遇到过太多次了。
老板拿了执照，开开心心做生意。
一个月后接到电话——"您逾期未报税，罚款500起"。
老板懵了："我还没营业呢报什么税？"
这就是最典型的坑——不经营也要零申报。

【45-60s】每一家新公司我都盯着做"全流程清单"，
签合同的时候就把所有后续事项排好，
什么时间做什么事，清清楚楚。

【60-75s】拿了执照只是你生意的第一步，
后面的路还长着呢。
需要一条龙服务的，找江姐。
从注册到运营，全帮你搞定。

【75-90s】字幕：找江姐，商事登记+后续运营一条龙
```

**分镜：** 00s悬念表情→15s切近景→30s手指数数→45s摇头无奈笑→60s自信语气→75s字幕
