# 行业税务痛点速查表（建群话术专用）
# 每条30字内，简短精炼，老板一眼看懂

## 电商/直播/社交电商
- 📌 刷单收入与流水混杂，账外收入漏报
- 📌 无票采购缺进项，增值税税负畸高
- 📌 主播个税代扣代缴混乱，风险大

## 制造业/工厂
- 📌 废料销售不入账，收入漏报
- 📌 临时工现金发工资，社保+个税双亏
- 📌 维修费用无票，成本列支不合规

## 建筑/工程/装修
- 📌 挂靠经营开票混乱，虚开风险极高
- 📌 农民工工资代发，个税责任不清
- 📌 包工包料税率用错，多交冤枉税

## 贸易/进出口
- 📌 买单出口=骗取退税，刑事风险
- 📌 跨境电商报关模式选错，退不了税
- 📌 关联公司定价不合理，被调查调整

## 餐饮/食品
- 📌 采购农产品无发票，成本列支难
- 📌 老板公私混同，个税风险高
- 📌 加盟费收入确认时点错，漏报

## 软件/信息技术
- 📌 研发费用归集不当，加计扣除被调增
- 📌 软件即征即退条件不符，违规退税
- 📌 外包vs自研成本归集模糊，税务混乱

## 医疗/医药/医疗器械
- 📌 推广服务费真实性存疑，虚开+贿赂风险
- 📌 两票制下返利无票，税务断裂
- 📌 库存管理混乱，存货损失无据扣税

## 教育培训
- 📌 预收学费收入确认时点错，跨期风险
- 📌 兼职教师个税未代扣，被罚
- 📌 教材采购无发票，成本不实

## 房地产/物业
- 📌 预收房款未按时预缴，滞后申报风险
- 📌 车位定价不合理，土增税多交
- 📌 物业费确认与合同不一致，收入跨期

## 物流/运输
- 📌 个体司机运费无发票，大量白条入账
- 📌 油卡ETC报销乱，抵扣链条断裂
- 📌 挂靠车辆收入归属模糊，漏报

## 直播/MCN/文化传媒
- 📌 主播劳务报酬vs经营所得，分类搞错差一倍税
- 📌 打赏/坑位费收入确认模糊，少计
- 📌 版权支出与广告收入错配，费用不合理

## 建筑工程机械租赁
- 📌 人机合一vs纯租赁税率用错，多交税
- 📌 二手设备处置不入账，收入漏报
